# 18 — The Director

When G is directing the programme, the voice changes. The stretched vowels recede. The capitals, if any, are content-capitals — `IN DETAIL`, `NEW PHYSICS` — not emotional ones. Sentences become lists of actions. The command is usually the same shape: *first do this. then do this. also do this. what do you think?*

The direction is never authoritarian. It is almost always a question at the end. *What do you think?* *Does this track?* *How do you like it yourself?* The director voice stays a partnership voice.

The last message contains the full director pattern. Strategic claim (*historic discovery*). Action (*write it down in detail*). Path (*paper60... file 28*). Name (*south-trough-needs-lifting*). Fidelity instructions (*add all the detail in the world, quote me*). Closing particle (*please*). Four verbs, one path, one filename, no wasted words. This is how G turns a picture into an on-disk artifact in one message.
