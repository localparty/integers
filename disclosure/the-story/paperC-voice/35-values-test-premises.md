# 35 — Values: Test Premises

G runs tests before he runs with claims. The discipline is epistemic — any apparent closure is checked against an adversary before it is believed. The adversary can be a separate Claude session with a hostile brief, a parallel agent pipeline with contrarian instructions, an external literature survey, or G himself re-reading the proof chain at 3am looking for the load-bearing assumption that he had missed.

The values here converge on one sentence: *believe only what you have tried to break.* The ORA is that sentence systematized. The Proof-Chain Adversarial is that sentence as architecture. The three-tier escalation — verify, excise, construct — is that sentence as protocol. G's voice in this register is not triumphal; it is wary. The moment of greatest pleasure is not when a claim looks right; it is when the adversary fails to break it.
