# Corpus 2 — solutions/

*Community-standard conjecture solutions. No active prize. 16 papers.*

This corpus contains the conjecture-solving papers that **do not carry an
active prize** but represent community-standard open problems — solved by
applying the Paper 12 CBB operator-algebraic machinery + the foundational
geometry from Corpus 1 (`integers/`).

See `strategy/north-star.md` §2 ("Three Pillars") for the per-vertex
four-deliverable pattern: COMPLY bare + INDEPENDENCE bare + HARDEN bonuses
+ PAC. Each solution in this corpus has a matching vertex in
`strategy/<vertex>/` with the four deliverables per the universal-approval
three-pillar structure (see `strategy/universal-approval-three-pillars.md`
and `strategy/universal-approval-run.md`).
