PART 5: Epstein zeta E_2(s; Q) for the S^1/Z_2 KK spectrum
  Q(n,m) = 2n^2 + 2nm + 2m^2  (sunset topology on S^1)
------------------------------------------------------------------------

The Epstein zeta function is:
  E_2(s; Q) = sum'_{(n,m) in Z^2} [Q(n,m)]^{-s}
where sum' excludes (n,m) = (0,0).

By Theorem K.1 (Universal Epstein Vanishing):
  E_2(-j; Q) = 0 for all integers j >= 1
Proof: E_2(s;Q) = pi^s * Lambda(s) / Gamma(s),
and 1/Gamma(-j) = 0 for j = 1,2,3,...

Direct numerical evaluation at s = 2 (convergent region, cross-check):
  E_2(2; Q) (direct, 200 terms) = 1.92776310187537

Demonstrating the vanishing mechanism for L=1 (Riemann zeta):
  zeta_R(s) = pi^s * Lambda(s) / Gamma(s)   [Lambda(s) = pi^{-s} Gamma(s) zeta_R(s)]
  At s = -j: 1/Gamma(-j) = 0 => zeta_R(-j) = 0

  j=1: zeta_R(-1) = -0.083333333,  1/Gamma(-1) = 0.0
  j=2: zeta_R(-2) = 0.0,  1/Gamma(-2) = 0.0
  j=3: zeta_R(-3) = 0.0083333333,  1/Gamma(-3) = 0.0
  j=4: zeta_R(-4) = 0.0,  1/Gamma(-4) = 0.0

The vanishing is mechanical: 1/Gamma(s) = 0 at all non-positive integers.
This is a NUMBER-THEORETIC identity — not a regularization choice.

PART 6: S^1/Z_2 orbifold: Z_2-even vs Z_2-odd mode contributions
------------------------------------------------------------------------

On S^1/Z_2, modes split into:
  Z_2-even: phi(n) with n = 0, 1, 2, 3, ...  (cosine modes)
  Z_2-odd:  phi(n) with n = 1, 2, 3, ...  (sine modes, no zero mode)

The full S^1 sum: sum_{n=-inf}^{inf} f(n) = f(0) + 2 * sum_{n=1}^{inf} f(n)
The Z_2-even sum: f(0) + sum_{n=1}^{inf} f(n)  [half the oscillating part]
The Z_2-odd sum:  sum_{n=1}^{inf} f(n)           [no zero mode]

For the constant summand f(n) = 1:
  Full S^1:   1 + 2*zeta_R(0) = 1 + 2*(-1/2) = 0  [S_0 = 0]
  Z_2-even:   1 + zeta_R(0) = 1 + (-1/2) = 1/2  [S_0^even = 1/2]
  Z_2-odd:    zeta_R(0) = -1/2  [S_0^odd = -1/2]

  Cancellation: S_0^even + S_0^odd = 1/2 + (-1/2) = 0 = S_0
  But also: S_0^even - S_0^odd = 1/2 - (-1/2) = 1 ≠ 0

Numerical values:

only Z_2-even modes contribute to the zero-mode sector.
The Goroff-Sagnotti operator is Z_2-even (it is built from 4D curvature).
The leading KK sum for Z_2-even modes: S_0^even = 1/2 ≠ 0.

CRITICAL POINT: For orbifold boundary conditions, S_0 ≠ 0 for each parity sector.
The full vanishing S_0 = 0 uses the COMPLETE S^1 sum (both parities).
On S^1/Z_2, one must combine Z_2-even AND Z_2-odd internal loop modes.

PART 7: Is s_0 a negative even integer (trivial zero of zeta_R)?
------------------------------------------------------------------------

The S_0 cancellation uses zeta_R(0) = -1/2, not a trivial zero.
The subleading Epstein sums use E_L(-j; Q) = 0 for j >= 1.

The MECHANISM of E_L(-j; Q) = 0 is IDENTICAL to the trivial zero mechanism:
  - Trivial zeros: zeta_R(-2k) = 0 because 1/Gamma(-2k) = 0
  - Epstein zeros: E_L(-j; Q) = 0 because 1/Gamma(-j) = 0
Both are consequences of the same gamma-function pole structure.

However, there is an important distinction:
  Trivial zeros of zeta_R: only at NEGATIVE EVEN integers (-2, -4, -6, ...)
  Universal Epstein vanishing: at ALL NEGATIVE INTEGERS (-1, -2, -3, ...)

The Goroff-Sagnotti subleading sum lands at E_2(-1; Q) (j=1, odd),
which is NOT a trivial zero of zeta_R — but it IS a universal Epstein zero.

Scheme independence question:
  Trivial zeros of zeta_R ARE scheme-independent (number-theoretic identities).
  Universal Epstein zeros: also scheme-independent, by the same gamma-pole argument.
  The gamma-pole 1/Gamma(-j) = 0 is a mathematical fact, not a regularization choice.

Claim: The Epstein vanishing at negative integers is SCHEME-INDEPENDENT in the
same sense that trivial zeros of zeta_R are scheme-independent.
Both follow from 1/Gamma(negative integer) = 0,
which is a mathematical identity holding in any regularization scheme that
represents the zeta function as (pi^s / Gamma(s)) * Lambda(s).
