SECTION 2: Mode-by-mode cancellation formula
--------------------------------------------------

  The GS vertex R_{μνρσ}R^{ρσλτ}R_{λτ}^{μν} is cubic in Riemann.
  On M⁴ × S¹/Z₂, the mode expansion of h_{MN} gives:

    h_{μν}(x,y) = Σ_n h_{μν}^(n)(x) cos(ny/R)  [Z₂-even]
    h_{μ5}(x,y) = Σ_n A_{μ}^(n)(x) sin(ny/R)   [Z₂-odd]
    h_{55}(x,y) = Σ_n φ^(n)(x) cos(ny/R)        [Z₂-even]

  The GS three-vertex for purely 4D-indexed external legs:
    V ~ h_{μν} ∂_ρ h_{αβ} ∂_σ h_{γδ} (schematic)

  When KK mode n runs in the loop, the y-direction integral produces:

    (1/πR) ∫₀^{πR} dy [cos(ny/R)]³  =  3/(4) δ_{3n,n} ≠ 0  (even)
    (1/πR) ∫₀^{πR} dy [sin(ny/R)]²cos(ny/R)  →  (1/2)δ_{n,n}  

  For the GS coefficient from a loop of h_{μν} modes at level n:
    c_even(n) = +d₀   (even mode contributes +d₀)

  For h_{μ5} (odd) mode at level n contributing via mixed vertex:
    c_odd(n) = -d₀   (odd mode contributes -d₀, from parity)

  Total KK correction (n ≥ 1 only, zero mode is the 4D graviton):
    ΔC_GS = Σ_{n=1}^∞ [c_even(n) + c_odd(n)]
           = Σ_{n=1}^∞ [d₀ - d₀]
           = Σ_{n=1}^∞ 0  =  0

  This is exact — not a truncation artifact.

SECTION 6: Scheme comparison
--------------------------------------------------

  The Z₂ parity cancellation is regulator-independent because:

  (a) Dimensional regularization (d → 4-ε):
      Preserves all discrete symmetries including Z₂.
      The cancellation c_even(n) + c_odd(n) = 0 is algebraic
      before any integration — it holds diagram by diagram.

  (b) Cutoff regularization (|p_5| < Λ):
      If the cutoff is on |n| ≤ N_max (applied symmetrically),
      even and odd modes are cut off at the same N_max.
      The partial sum cancellation ΔC(N_max) = 0 is exact.

  (c) Pauli-Villars:
      PV fields must come in Z₂ pairs (even + odd PV partners).
      Each PV field cancels the divergence of its corresponding
      mode with the same numerical coefficient — Z₂-symmetric PV
      preserves the cancellation.

  (d) Zeta regularization:
      Manifestly preserves all discrete symmetries.
      ζ_R(0) appears equally in even and odd towers → exact cancel.
