# 01. The 2D CP^2 Sigma Model on R x S^1_L with Z_3 Twist

## 1.1 The target space

The target space is CP^2 = SU(3)/U(2), the space of lines through the
origin in C^3. Concretely:

  CP^2 = { z in C^3 : |z|^2 = 1 } / U(1)

where U(1) acts by z -> e^{i alpha} z. The Fubini-Study metric on CP^2
is the unique SU(3)-invariant Kahler metric with sectional curvatures
in [1/4, 1] (normalized so max curvature = 1, Fubini-Study radius = 1).

**Key topological data:**
- pi_1(CP^2) = 0
- pi_2(CP^2) = Z (generated by CP^1 ↪ CP^2)
- H^2(CP^2, Z) = Z (generates the instanton sectors)
- dim_R(CP^2) = 4

The model has N-1 = 2 complex bosonic degrees of freedom per spacetime
point.


## 1.2 The 2D sigma model on R^2

The CP^2 sigma model on flat R^2 is defined by the action:

  S[phi] = (1/g^2) integral_{R^2} g_{ab}(phi) d_mu phi^a d^mu phi^b  d^2x

where phi: R^2 -> CP^2 is a smooth map, g_{ab} is the Fubini-Study
metric, and g^2 > 0 is the coupling constant.

In terms of the homogeneous coordinates z_i(x) (i = 1,2,3) with
|z|^2 = 1:

  S[z] = (1/g^2) integral_{R^2} ( |D_mu z|^2 ) d^2x

where D_mu z = d_mu z - (z-bar . d_mu z) z is the covariant derivative
with respect to the induced U(1) connection A_mu = -i z-bar . d_mu z.

**Asymptotic freedom** [PROVED]. The one-loop beta function is:

  beta(g^2) = -(N/2pi) g^4 + O(g^6)    with N = 3

so beta(g^2) = -(3/2pi) g^4 for CP^2. The theory is asymptotically free.
The dynamical scale is:

  Lambda_2D = mu exp( -2pi / (3 g^2(mu)) )

This is scheme-independent at one loop.


## 1.3 Compactification on S^1_L

Replace R^2 = R_t x R_x with R_t x S^1_L, where S^1_L is a spatial
circle of circumference L:

  S^1_L = R / (L Z)

The spacetime is a cylinder of circumference L. The field is:

  phi: R_t x S^1_L -> CP^2

with some boundary condition around S^1_L (specified below).


## 1.4 The Z_3-twisted boundary condition

CP^2 has an isometry group SU(3)/Z_3 = PSU(3). The center Z_3 of SU(3)
acts trivially on CP^2 (since z -> omega z with omega^3 = 1 is the same
point in CP^2). However, SU(3) contains elements of order 3 that act
NONTRIVIALLY.

**The twist element.** Choose Omega in SU(3) with:

  Omega = diag(1, omega, omega^2),    omega = e^{2pi i/3}

This satisfies Omega^3 = I and has three distinct eigenvalues. Its
action on CP^2 has exactly three fixed points:

  [1:0:0],  [0:1:0],  [0:0:1]

**The twisted boundary condition.** We impose:

  phi(t, x + L) = Omega . phi(t, x)

More precisely, in homogeneous coordinates:

  z(t, x + L) = Omega . z(t, x) . e^{i alpha(t,x)}

for some U(1) gauge transformation alpha. The key point: the twist
Omega is NOT in the center, so this is a GENUINE (non-periodic) twist.

**Why Z_3 specifically.** The Z_3 twist is the maximal abelian twist for
SU(3). It breaks the SU(3) global symmetry to its maximal torus:

  SU(3) -> U(1) x U(1)

This is the 2D analogue of the 4D compactification R^3 x S^1 with
holonomy diag(1, omega, omega^2), which is the center-symmetric vacuum
used in Unsal's deformed Yang-Mills.


## 1.5 Why this twist matters: symmetry breaking pattern

**Without twist** (periodic b.c.): the vacuum has full SU(3) symmetry.
At small L, the theory decompactifies and there is no semiclassical
control.

**With Z_3 twist:** the symmetry breaks to U(1) x U(1) for ALL L.
At small L, the low-energy theory is a 1D quantum mechanics of two
U(1) phases (the holonomies), and semiclassical methods apply.

This is the Eguchi-Kawai / Unsal mechanism: the twist prevents the
theory from "seeing" that L is small, maintaining the structure needed
for semiclassical analysis.


## 1.6 The effective theory at small L

At small L with the Z_3 twist, the field phi can be decomposed as:

  phi(t,x) = phi_0(t) + fluctuations

where phi_0(t) is the constant mode around S^1, living near one of the
three fixed points.

**The moduli space.** The constant maps phi: S^1 -> CP^2 satisfying the
twisted b.c. are exactly the fixed points of Omega:

  M_0 = {[1:0:0], [0:1:0], [0:0:1]}

These are three isolated points — a DISCRETE set. There is no continuous
moduli space. This is crucial: it means the zero modes are GAPPED.

**The Kaluza-Klein tower.** Expanding around the fixed point [1:0:0],
the fluctuations delta z_2, delta z_3 satisfy:

  delta z_j(t, x + L) = omega^{j-1} delta z_j(t, x)

so the Fourier modes are:

  delta z_j(t, x) = sum_n  a_{j,n}(t) exp(2pi i (n + (j-1)/3) x / L)

The lowest KK mass is:

  m_KK = 2pi / (3L)

(from n=0, j=2 or j=3). All fluctuations have mass >= m_KK > 0.


## 1.7 The mass gap as a function of L

**Definition.** The mass gap m(L) of the Z_3-twisted CP^2 model on
R x S^1_L is:

  m(L) = inf { E > 0 : E is in the spectrum of H_L } - E_0(L)

where H_L is the Hamiltonian of the sigma model on S^1_L with
Z_3-twisted b.c., and E_0(L) is the ground state energy.

Equivalently, m(L) is the inverse of the largest correlation length:

  <O(t) O(0)> ~ exp(-m(L) |t|)  as |t| -> infinity

for any local operator O with <O> = 0.

**The central question:** Is m(L) > 0 for ALL L > 0?


## 1.8 What is known about m(L)

**Small L regime** [PROVED, semiclassically]:
  m(L) ~ (1/L) exp(-4pi/(3 g^2(1/L)))  for L << 1/Lambda

The mass gap is generated by fractional instantons (see Section 02).

**Large L regime** [PROVED, numerically]:
  m(L) -> m(infinity) = c_3 Lambda_2D > 0  as L -> infinity

where c_3 is a numerically determined constant from lattice simulations
of the uncompactified 2D CP^2 model. The value c_3 ~ O(1).

**N = infinity** [PROVED, exactly]:
  m(L) = Lambda  for all L

No phase transition, no L-dependence. Witten's 1979 exact solution of
the CP^{N-1} model at large N gives m = Lambda_2D identically.

**N = 2 (CP^1)** [PROVED]:
  m(L) > 0 for all L

Adiabatic continuity proved by Dunne-Unsal (2012), using the fact that
CP^1 = S^2 has a Z_2 twist that reduces to the O(3) model, where
exact integrability (Bethe ansatz) gives full control.

**N = 3 (CP^2)** [OPEN]:
  m(L) > 0 for all L ???

This is our target. Could there exist L* with m(L*) = 0?


## 1.9 The scenario we must exclude

The dangerous scenario: as L increases from 0, m(L) decreases,
eventually hits zero at some critical L = L*, signaling a phase
transition.

Physical intuition says this does NOT happen (there is no natural phase
transition in the Z_3-twisted model). But intuition is not proof.

The Z_3 twist preserves the 't Hooft anomaly of the original model
(the mixed anomaly between Z_3 center symmetry and CP symmetry). The
anomaly matching condition constrains the IR behavior: the vacuum must
be nontrivial (either gapped with Z_3 breaking, or gapless). Since the
Z_3 twist explicitly breaks SU(3) -> U(1)^2, the anomaly is saturated
by the mass gap — UNLESS the gap closes and reopens with different
anomaly matching.

For a continuous phase transition (m(L*) = 0), the CFT at L* would need
to match the anomaly. This constrains but does not exclude the scenario.
