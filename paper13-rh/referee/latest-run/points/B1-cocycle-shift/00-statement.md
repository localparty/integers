# B1: The Cocycle Shift Formula [HEAVY]

**Location:** Section 5, Appendix F

**Claim:** If zeta has a non-trivial zero at s = 1/2 + delta + i*gamma with delta != 0, the Brauer cocycle at bridge (p, N, k) shifts by the exact amount Delta_c(delta) = (1 - p^{-2*delta})/(p - p^{-2*delta}). This vanishes iff delta = 0.
