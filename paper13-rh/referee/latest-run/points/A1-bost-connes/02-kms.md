# A1.02: KMS_1 Uniqueness

BC 1995 Theorem 25 states: the KMS_1 state on A_BC is unique and given by omega_1(mu_n mu_m*) = delta_{nm}/n. The paper cites this correctly. The phase structure (unique state for beta <= 1, SSB for beta > 1) is standard.

**Subtlety:** At beta = 1, the partition function zeta(1) diverges, so the Gibbs formula does not apply directly. The uniqueness is proved by showing the KMS condition plus Hecke relations determine all matrix elements. This is correctly noted in Appendix A.3.

**Verdict: CORRECT.** Properly cited and within scope.
