# C2.03: The Domain

The domain of T_BC is span{e_n}, which is dense in H_R by definition (Axiom 1 defines H_R as having {e_n} as a complete ONB).

**Assessment:** This is tautological. The Hilbert space is defined to have this basis. The domain is dense by construction.

**The independent review (Concern 5) makes the correct observation:** If you have a densely defined symmetric operator with a complete set of real eigenvectors, essential self-adjointness is automatic. Nelson's theorem is invoked for "insurance" but is unnecessary. The direct argument via deficiency indices suffices.

**Verdict: CORRECT but trivial.** The domain is dense by construction.
