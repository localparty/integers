# C2.02: Nelson's Theorem

Nelson's theorem (Reed-Simon X.39): If T is a symmetric operator on a separable Hilbert space and the analytic vectors for T are dense, then T is essentially self-adjoint.

**Hypotheses check:**
1. H_R separable: Yes (countable ONB {e_n}).
2. T_BC symmetric on span{e_n}: Yes, because gamma_n are real (from Steps 1-6).
3. Analytic vectors dense: Yes, span{e_n} consists of analytic vectors and is dense in H_R.

**Assessment:** All hypotheses are satisfied. Nelson's theorem applies.

**Verdict: CORRECT.** Nelson's theorem is correctly invoked.
