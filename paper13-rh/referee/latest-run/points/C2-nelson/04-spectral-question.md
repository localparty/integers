# C2.04: The Critical Spectral Question

**Does spec(T-bar_BC) = {gamma_n}? Or could T-bar_BC have additional spectrum?**

If H_R is defined with {e_n} as a complete ONB and T_BC e_n = gamma_n e_n, then the self-adjoint closure T-bar_BC has spectrum exactly {gamma_n} (the closure of the set of eigenvalues in this case). This is a trivial consequence of the spectral theorem for self-adjoint operators with a complete orthonormal set of eigenvectors.

**But this is entirely tautological.** The question of whether T-bar_BC has additional spectrum only arises if T_BC is constructed on a LARGER Hilbert space (such as the full GNS space H_1 of omega_1) where additional spectral data could appear.

The paper defines H_R by Axiom 1 as a specific Hilbert space with specific basis. On THIS space, there is no room for additional spectrum. The real question is whether H_R, as defined by Axiom 1, is the correct Hilbert space -- that is, whether the BC system at KMS_1 actually produces a Hilbert space with these properties.

**Verdict: TAUTOLOGICAL within the axiom system.** The spectral completeness is built into the definition of H_R.
