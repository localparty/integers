## Point E2(c): Class Number > 1

**The question:**
Is the extension to h_K > 1 tractable or is there a fundamental obstruction?

**Finding:**
Section 15.4 describes the h_K > 1 case as "structurally tractable" because:
1. The bridge primes and cocycles are field-independent.
2. The obstruction is in the KMS analysis: multiple KMS states from the class group.
3. Each KMS state could be analyzed separately.

This assessment is reasonable. For h_K > 1, the BC system has |h_K| extremal KMS_1 states, one per ideal class. The bridge argument would need to be applied to each state separately, and one would need to verify that each state captures the zeros of the corresponding Hecke L-function.

The obstruction is technical (multi-state analysis) rather than fundamental (no known counter-indication). However, "expected to work but not carried out" is an honest statement, not a proof.

**Verdict for this sub-question:**
SOUND -- The limitation is correctly identified and honestly described as expected to extend.
