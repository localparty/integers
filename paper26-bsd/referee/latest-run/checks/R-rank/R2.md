# Check R2: Scope precisely stated

**Group:** R
**Source:** Paper 26 §1.4 statement + §9.3 extension + §15
**Pass criterion:** Scope characterized precisely (which fields,
which curves, which ranks).

**Verdict:** PASS
**Rigor label:** [LEMMA]

**Justification:** Paper 26 explicitly states the scope:
"CM elliptic curves over ℚ with CM by an imaginary quadratic
field K of class number 1, with analytic rank 0 or 1." The nine
class-number-1 fields are enumerated explicitly (§1.4, §9.3):
Q(√-d), d ∈ {1, 2, 3, 7, 11, 19, 43, 67, 163}. The paper is
honest about the scope being a proper subset of Wiles' "all
elliptic curves over ℚ" — see Section 15 and §15.3 on non-CM
curves.
