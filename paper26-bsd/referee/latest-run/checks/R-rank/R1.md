# Check R1: Rank equality for curves in scope

**Group:** R (Rank equality, Wiles main statement)
**Source:** Paper 26 Theorem 13.1 + Part IV §11-12
**Pass criterion:** rank(E(ℚ)) = ord_{s=1} L(E,s) explicitly proved
for CM curves over class-number-1 fields, analytic rank 0 and 1.

**Verdict:** PARTIAL
**Rigor label:** [KEY LEMMA — OPEN]

**Justification:** The rank equality is correctly claimed and the
downstream steps (Kolyvagin at rank 0, Gross-Zagier + Kolyvagin at
rank 1) are classical and correctly applied. **But** the chain
depends on three upstream [KEY LEMMA — OPEN] items:
(i) Meyer distributional → point spectrum upgrade for ζ_K;
(ii) twisted spectral inclusion for L(s, ψ); and
(iii) cohomology-class integrality premise.
Without these, the GRH-for-CM-L-functions step is not established,
and the rank equality is conditional on open lemmas.

**Additionally**, per Remark 12.6, the rank-1 case is vacuous
within scope: GRH implies L(1, ψ) ≠ 0, so no CM curve in scope
has analytic rank 1 over ℚ. The substantive content is entirely
at rank 0.
