# Check LC7: |E_tors| correctly determined

**Group:** LC
**Source:** Paper 26 §13.3 table, §14.1
**Pass criterion:** Torsion subgroup correct; cross-check LMFDB.

**Verdict:** PASS
**Rigor label:** [LEMMA]

**Justification:** For y²=x³−x, the rational 2-torsion points are
(0, 0), (1, 0), (-1, 0) (the three roots of x³−x = 0), plus the
identity O. So E(ℚ)_tors = (ℤ/2ℤ)² ≅ ℤ/2 × ℤ/2, of order 4.
|E_tors|² = 16. This matches LMFDB 32.a3.
