## 0. Critical Correction to Cycle 1

**Cycle 1 claimed 8 exotic Hodge classes in H^{2,2}(A).  This was WRONG.**

The Cycle 1 computation used two conditions for a 4-subset
S = {a, b, c, d} of (Z/13Z)* to define a Hodge class:
1. Weight zero: a + b + c + d = 0 mod 13
2. Type (2,2): |S cap Phi| = 2, |S cap Phi_bar| = 2

It found 23 such subsets, 15 of which decompose into conjugate pairs
{e, 13-e}, and 8 which do not.  The 8 were labelled "exotic Hodge classes."

**The error:** these two conditions are necessary but NOT sufficient.
The full condition for a Hodge class on A is **MT-invariance**: for each
conjugate pair (j, 13-j), the count of j in S must equal the count of 13-j.
Formally, define f_j(S) = #{s in S : s = j} - #{s in S : s = 13-j}.
MT-invariance requires f_j(S) = 0 for all j = 1, ..., 6.

**Verification (computed):**
- All 15 product classes (conjugate-pair decomposable) satisfy f_j = 0. CHECK.
- All 8 "exotic" classes have f_j != 0 for some j. They FAIL MT-invariance.
- E1 = {1, 4, 10, 11}: f = (1, -1, -1, 1, 0, 0). NOT Hodge.

**On A (simple, primitive CM type), there are exactly 15 Hodge classes
in H^{2,2}(A), all products of divisor classes.  Zero exotic.**

## 1. The Parity Theorem

**Theorem.**  For A with CM by Q(zeta_13) and primitive CM type
Phi = {1, 2, 3, 4, 6, 8}:

> Every MT(A)-invariant type-(p,p) class in Lambda^{2p}(H^1(A^n, Q)),
> for ALL p >= 1 and ALL n >= 1, decomposes into p conjugate pairs
> {a_i, 13-a_i}.

**Proof.**
MT-invariance requires f_j(S) = 0 for each conjugate pair (j, 13-j).
This means each pair contributes an equal number k_j of j's and (13-j)'s
(i.e., 2k_j elements total).  Type (p,p) requires p elements in Phi and
p in Phi_bar.  Since each conjugate pair {j, 13-j} has exactly one element
in Phi and one in Phi_bar, each pair contributes k_j to each side.
Then sum_j k_j = p, and the 2p elements decompose as k_1 copies of
{j_1, 13-j_1}, ..., k_6 copies of {j_6, 13-j_6}.
This is a product of p conjugate pairs.  QED.

**Corollary.** The Hodge ring of A^n (all n) is generated by divisor
classes and correspondence classes from H^{1,1}.  There are NO exotic
Hodge classes on any power A^n.

This confirms Dodson (1984) and Moonen-Zarhin (1999): for simple CM
abelian varieties with primitive CM type and maximal Hodge group, the
Hodge ring is generated in degree 1.

## 3. Cross-Bridge Products

For A1 with CM by Q(zeta_13), A2 with CM by Q(zeta_19):

MT(A1 x A2) = MT(A1) x MT(A2) (independent, since K1 != K2 with
no common CM subfield).

On A1 x A2, Hodge classes require:
- Characters from A1 satisfy MT(A1)-invariance (parity per 13-pair)
- Characters from A2 satisfy MT(A2)-invariance (parity per 19-pair)
- Odd splits (e.g., 3 from A1, 1 from A2) violate parity: impossible

**Result:** All Hodge classes on A1 x A2 are products.  No exotic classes.

## 4. The Angle Attempts (Pre-Correction)

Before discovering the Cycle 1 error, three construction angles were attempted:

### Angle 1: Orbit Correspondence Cycles

**Attempt:** Construct codimension-2 cycles on A as images of orbit projectors
P_{O_i} or as abelian subvarieties.

**Obstruction:** A is SIMPLE (primitive CM type implies simplicity by Shimura).
A simple abelian variety has no proper abelian subvarieties.
The only codimension-2 algebraic subvarieties are complete intersections of
divisors, which give product classes.

**Also:** the orbit projectors P_{O_i} lie in End(A) tensor Q = K, so they
define abelian subvarieties only in an isogeny category over Q-bar.
Over Q-bar, A is isogenous to B_1 x B_2 x B_3 (dim 2 each), but this
isogeny decomposition does not produce exotic classes on A itself.

### Angle 2: Hecke Correspondence Cycles

**Attempt:** Use the Hecke correspondences T_{P_i} at primes P_i above 5
to construct inter-orbit algebraic cycles.

**Obstruction:** The Hecke operators are elements of End(AxA) = M_2(K), which
acts on H^1(AxA) by 2x2 matrices WITHIN each character eigenspace.
This preserves character multisets: Lambda^4(F^*) for F in M_2(K) maps
a class with character multiset {a,b,c,d} to another class with the
SAME multiset.  Cannot transform product classes into exotic classes.

### Angle 3: Jones Sub-Factor Correspondence

Not reached: the Parity Theorem shows there are no exotic classes to target.

## 5. Where Exotic Classes Genuinely Appear

Exotic Hodge classes require the Hodge group to be SMALLER than maximal.
This happens when:

1. **A = E^g** (power of a CM elliptic curve): End(A) = M_g(Q(sqrt(-d))).
   The Weil classes in H^{g,g}(A) are exotic.

2. **A = B1 x B2** with same CM field K but different CM types Phi1 != Phi2:
   MT(A) is a proper subtorus, giving additional invariants.

3. **Degenerate CM type** (induced from a subfield): A is not simple.

**The bridge family (all entries have prime N) always gives primitive CM types,
hence simple varieties with maximal Hodge groups.  No exotic classes arise.**

## 7. Recommendation for Cycle 3

**PIVOT TARGET.** The bridge family at prime N gives trivially-verified Hodge.
Two options for new content:

**(A) Non-prime conductors.**  Extend the bridge family to composite N
(e.g., N = 21 = 3 x 7).  The CM field Q(zeta_21) has non-trivial subfields,
giving potentially non-primitive CM types and genuine exotic classes.
The bridge's orbit structure at composite N could organise or construct
these exotic classes.  Feasibility: 5/10.

**(B) Weil classes on E^g.**  For E a CM elliptic curve at a bridge entry
(e.g., E with CM by Q(sqrt(-7)) at (2,7)), the variety E^g has exotic
Weil classes for g >= 2.  These are well-understood (Weil 1977, Schoen 1998)
and the bridge might provide constructive realisations.  Feasibility: 6/10.

**Recommended: (B) first** -- the Weil classes are concrete and the
bridge at (2,7) gives the simplest CM elliptic curve.
