# Corpus 1 — integers/

*Foundational derivation papers. 13 papers post-SPLIT-1.*

This corpus contains the papers that **derive the geometry**: one postulate
(the compact e-circle), the Kaluza-Klein 5D geometry M⁴ × S¹, and every
consequence that follows (spin-statistics, cosmology, black hole information,
Standard Model gauge group, flux tubes, thermal history, moduli + GUT
unification, scheme-independence of UV finiteness, CBB system, all-orders UV
finiteness, SM gauge group from three-qubit entanglement, geometry of the
circle, projections of 5D geometry).

See `strategy/north-star.md` for the programme-level context; this corpus is
the foundational layer on which the solutions corpi (Corpus 2 + Corpus 3)
rest.
