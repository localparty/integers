
========================================================================
PART 1  Mercedes diagram KK conservation → derivation of Q_3
========================================================================

The 3-loop Mercedes vacuum diagram in pure 5D KK gravity has:
  - Two trivalent vertices V₁, V₂
  - Four internal propagators connecting V₁ and V₂
  - Three independent KK mode numbers n₁, n₂, n₃
  - The 4th propagator carries KK momentum −(n₁+n₂+n₃) by KK conservation
    at both vertices

At each vertex, KK (U(1)) charge conservation: Σ_incoming n_i = 0

The mass-squared of each internal line is m_i² = n_i² / R².
Summing over all four internal propagators:

  Σ m² = [ n₁² + n₂² + n₃² + (n₁+n₂+n₃)² ] / R²

Expanding (n₁+n₂+n₃)² = n₁²+n₂²+n₃² + 2(n₁n₂+n₁n₃+n₂n₃):

  Σ m² · R² = 2*n1**2 + 2*n1*n2 + 2*n1*n3 + 2*n2**2 + 2*n2*n3 + 2*n3**2

  Σ m² · R² = 2 · [n1**2 + n1*n2 + n1*n3 + n2**2 + n2*n3 + n3**2]

Defining Q_3(n₁,n₂,n₃) = n₁² + n₂² + n₃² + n₁n₂ + n₁n₃ + n₂n₃ ,
we have Σ m² · R² = 2 · Q_3(n₁,n₂,n₃).

The Gram matrix of Q_3 (so that Q_3(n) = n^T A_3 n):

  A_3 = [[1, 1/2, 1/2], [1/2, 1, 1/2], [1/2, 1/2, 1]]

  det(A_3) = 1/2

  Reconstructed Q_3(n) = n^T A_3 n = n1**2 + n1*n2 + n1*n3 + n2**2 + n2*n3 + n3**2

  Expected Q_3        = n1**2 + n1*n2 + n1*n3 + n2**2 + n2*n3 + n3**2

  Match: True

This agrees with Appendix K §K.2.2, equations for the Mercedes topology.

PART 3  Analytic continuation of E_3(s; Q_3) — completed Mellin
========================================================================

We use the completed zeta function:

    Λ(s; Q) = π^(-s) Γ(s) E_L(s; Q)

By Epstein (1903), Terras (1985, Vol I, Ch. 1 §4):

    Λ(s; Q) = F_1(s) + det(A)^(-1/2) F_2(L/2 - s)
              + det(A)^(-1/2)/(s - L/2)  −  1/s 

where for L = 3,

    F_1(s) = ∫₁^∞ t^(s-1) [θ_Q(t) − 1] dt
    F_2(s) = ∫₁^∞ t^(s-1) [θ_{Q^{-1}}(t) − 1] dt
    θ_Q(t) = Σ_{n ∈ ℤ^L} exp(−π t Q(n))

Both F_1 and F_2 are ENTIRE (θ − 1 decays as exp(−π λ_min t)).
The only poles of Λ are at s = 0 and s = L/2 = 3/2.

RECONSTRUCTING E_L(s; Q) = π^s Λ(s; Q) / Γ(s).

KEY OBSERVATION (Theorem K.1 mechanism):
   At s = −j (j ≥ 1),  Γ(s) has a pole,  Λ(s) is finite,
   hence  E_L(−j; Q) = π^{-j} · Λ(−j; Q) · [1/Γ(−j)] = 0.

det(A_3)     = 0.5      (expected 1/2)

A_3^{-1} (exact):
   [[3/2, -1/2, -1/2], [-1/2, 3/2, -1/2], [-1/2, -1/2, 3/2]]

det(A_3^-1)  = 2.0   (expected 2)

Q_3^{-1}(n) = 3*n1**2/2 - n1*n2 - n1*n3 + 3*n2**2/2 - n2*n3 + 3*n3**2/2

min-eigenvalue of A_3     = 0.5
min-eigenvalue of A_3^{-1} = 0.5

  (direct summation valid for Re(s) > 3/2)
  E_3(3) direct  = (14.45370363017354034454762 + 0.0j)
  E_3(3) Mellin  = (14.45392104374447186282084 + 0.0j)
  relative error = 1.50421e-5

PART 6  Cross-check: Mercedes physical form 2·Q_3 also vanishes
========================================================================

The Mercedes diagram's physical KK mass-sum is Σ m² R² = 2·Q_3(n).
The Epstein zeta evaluated on the physical form:

    E_3(s; 2·Q_3)(n) = Σ'_n [2·Q_3(n)]^{-s} = 2^{-s} · E_3(s; Q_3)

At s = −j:  E_3(−j; 2·Q_3) = 2^j · E_3(−j; Q_3) = 2^j · 0 = 0.

PART 7  The Mercedes R⁴ counterterm coefficient vanishes at every order
========================================================================

The 3-loop Mercedes contribution to the R⁴ counterterm has the form:

    c(R⁴)|_Mercedes = Σ_{n₁,n₂,n₃ ∈ ℤ} f(n₁,n₂,n₃; R)

By the UV expansion, f admits a polynomial-in-(n²/R²) development:

    f(n; R) = d_0 + d_2 · [2·Q_3(n)/R²] + d_4 · [2·Q_3(n)/R²]² + …

where d_{2k} are 4D-momentum integral coefficients (Seeley-DeWitt data).
The triple KK sum at each order is:

    Σ_n [2·Q_3(n)]^k = 2^k · E_3(−k; Q_3)

  k = 0:  E_3(0; Q_3)          — finite, NOT zero, but pre-multiplied by
                                 d_0 · S₀³ = d_0 · 0 = 0  (leading cancel)
  k = 1:  E_3(−1; Q_3) = 0    (Thm K.1, verified above)
  k = 2:  E_3(−2; Q_3) = 0    (Thm K.1, verified above)
  k = 3:  E_3(−3; Q_3) = 0    (Thm K.1, verified above)
  k = 4:  E_3(−4; Q_3) = 0    (Thm K.1, verified above)
  k = 5:  E_3(−5; Q_3) = 0    (Thm K.1, verified above)
  k ≥ 6:  all zero by Thm K.1

E_3(0; Q_3)  = (-1.0 + 0.0j)   (via limit s → 0+)
   (Predicted by K.4.2: E_L(0; Q) = −1 for any positive-definite Q.)

The constant-summand triple sum [Σ_n 1]³ = S₀³ = 0 is a
DIFFERENT object from E_3(0; Q_3) = −1; the former is the leading
mass-independent UV coefficient, the latter is the s = 0 Epstein value.
Both appear in the Mercedes expansion and both contribute 0 after
multiplication by the appropriate 4D momentum integrand — S₀³ kills
d₀ and E_3(−j; Q_3) = 0 kills every subleading d_j for j ≥ 1.

S₀  = 1 + 2·ζ(0) = 0.0
S₀³ = 0.0

THEREFORE: c(R⁴)|_Mercedes = 0 identically at every order in 1/R².

This directly confirms Theorem K.1 at L = 3 for the Mercedes topology.
