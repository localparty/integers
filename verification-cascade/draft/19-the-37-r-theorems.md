## What an R-Theorem is

An R-Theorem is a theorem of physics — a known, established result in quantum mechanics, quantum field theory, general relativity, or cosmology — that has been TRANSPOSED into the Bost-Connes operator-algebraic language and proved to hold within the framework. The "R" stands for "Riemann" — each transposition connects a physics fact to the Riemann zero spectrum via the BC bridge.

The R-Theorems are not new physics. They are OLD physics in a NEW language. The theorem of physics is unchanged. What changes is the PROOF — the proof uses BC operator-algebraic machinery instead of the original physics machinery. And because the BC machinery is connected to the Riemann zeros via the KMS_1 spectral data, each R-Theorem becomes a new pathway between physics and number theory.

As of April 2026, the framework has 37 named R-Theorems across six families. Each is a transposition pathway — a route that an ORA agent can use to verify, excise, or construct alternative proofs.

## The 37 R-Theorems as 37 verification pathways

Each R-Theorem is a PATHWAY — a route from physics to arithmetic (or vice versa) that an ORA agent can traverse. In the verification cascade, these pathways serve three functions:

**1. Cross-domain verification.** A spectral claim can be checked against its R-Theorem image. If a step claims a spectral gap, the R-Theorem dictionary shows what that gap looks like in the GR domain (R.GR-3: positive energy), in the QFT domain (R.QFT-6: asymptotic freedom), and in the QM domain (R.QM-1: uncertainty). Each image is an independent check.

**2. Excision routes.** If a proof step breaks in one domain, the R-Theorem network suggests proofs in OTHER domains. The Penrose singularity path to RH (R.QFT-7) is an alternative to the CCM spectral path. The Atiyah-Singer index path (R.QFT-1) is yet another alternative. Each R-Theorem is a potential excision route.

**3. LOCK construction.** Multiple R-Theorems proving the same conclusion from genuinely different physics constitute a LOCK. Three physics proofs of RH (Stone/positivity, Penrose/causal, Atiyah-Singer/topology) each use different R-Theorems, different pattern categories, different failure modes. The LOCK confidence is 10/10 precisely because the three routes are genuinely independent — an independence the R-Theorem dictionary makes verifiable.
