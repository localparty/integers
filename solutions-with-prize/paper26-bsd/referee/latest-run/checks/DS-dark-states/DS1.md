# Check DS1: |w^k| = N(𝔭)^{-k/2} < 1

**Group:** DS
**Source:** Paper 26 §6.1, Proposition 6.1
**Pass criterion:** Trivial bound verified.

**Verdict:** PASS
**Rigor label:** [THEOREM]

**Justification:** Min norm is 2 (for 𝔭 = (1+i)), k ≥ 2, so
N^{-k/2} ≤ 2^{-1} = 0.5 < 1. Trivial.
