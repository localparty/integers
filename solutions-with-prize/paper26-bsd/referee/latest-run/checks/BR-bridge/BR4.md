# Check BR4: Cocycle match Hasse = FK determinant = 1/k

**Group:** BR
**Source:** Paper 26 §4.6, Theorem 4.6
**Pass criterion:** Field-independence of the cocycle match.

**Verdict:** PASS
**Rigor label:** [THEOREM]

**Justification:** Theorem 4.6 (field-independence) is correctly
stated and proved. The Hasse invariant of a cyclic algebra is
1/k by construction, independent of the base field. The
Fuglede-Kadison determinant is a property of the subfactor
(Jones 1983), independent of number-field data. The match is
structural, not field-dependent.
