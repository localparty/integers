# Check TR4: Is Baker actually needed vs Gelfond-Schneider?

**Group:** TR
**Source:** Paper 26 §8.5
**Pass criterion:** Rigorous need for Baker's full strength.

**Verdict:** PASS (editorial observation)
**Rigor label:** N/A

**Justification:** The paper's own §8.5 admits: "The upgrade from
Gelfond–Schneider to Baker is a convenience (cleaner statement
over number fields), not a necessity. The proof would go through
with Gelfond–Schneider alone."

This is honest. Baker is used for rhetorical symmetry with the
RH case, not for mathematical necessity. The proof could use
Gelfond-Schneider or even elementary unique factorization for
the irrationality step. No rigor concern.
