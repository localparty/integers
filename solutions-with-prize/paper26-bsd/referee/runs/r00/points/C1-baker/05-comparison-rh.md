## Point C1(e): Comparison to the RH Proof

**The question:**
In the RH proof, Gelfond-Schneider was used. Baker is strictly stronger. Is the upgrade necessary?

**Finding:**
The preprint itself answers this in Section 8.5: "Baker's theorem is strictly stronger than Gelfond-Schneider, but for this application the additional strength is not needed: two bridge primes with distinct norms already suffice. The upgrade from Gelfond-Schneider to Baker is a convenience (cleaner statement over number fields), not a necessity."

This is correct. The argument requires only two bridge primes with multiplicatively independent norms, which is the Gelfond-Schneider case (n = 2). Baker's theorem (arbitrary n) is not needed for the core argument.

The presentation is honest about this: Baker is used for aesthetic reasons (cleaner generalization) rather than mathematical necessity. Gelfond-Schneider would suffice.

**Verdict for this sub-question:**
SOUND -- Baker is a convenient but unnecessary upgrade. Gelfond-Schneider suffices for the core argument.
