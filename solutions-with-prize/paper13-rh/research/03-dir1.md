# research/03-dir1 -- Does KMS_1 Uniqueness Buy Compact Resolvent?

*Date: 2026-04-10. Spectral Realisation Cycle 3, Direction 1.*

## 1. The question

At beta = 1, the KMS state omega_1 on A_BC is UNIQUE (Bost-Connes
Theorem 25). Does this uniqueness constrain the spectral density of
T_BC enough to force compact resolvent (purely discrete spectrum)?

## 2. The GNS representation argument

In the GNS representation (H_1, pi_1, Omega_1) built from omega_1:

- T_BC acts on H_1 as the infinitesimal generator of the modular
  automorphism sigma_t
- The spectral measure of T_BC w.r.t. Omega_1 is UNIQUE because
  omega_1 is unique
- The spectral decomposition T_BC = integral lambda dE(lambda) has
  a UNIQUE spectral measure mu = <Omega_1, E(.) Omega_1>

Question: does a unique spectral measure force discrete spectrum?

## 3. Literature search: KMS uniqueness and spectrum

### Bratteli-Robinson (Operator Algebras and Quantum Statistical
Mechanics, Vol 2)

**Theorem 5.3.30 (Takesaki):** If M is a factor and omega is a
faithful normal KMS state, the modular operator Delta = e^{-beta H}
has spectrum determined by the Connes spectrum S(M).

For type I_infty factors: S(M) = {0,1}, modular operator has
discrete spectrum. COMPACT RESOLVENT would follow.

For type III_1 factors: S(M) = R_+, modular operator has spectrum
= R_+. This gives CONTINUOUS spectrum for log(Delta) = -beta*H.

**The BC system at beta = 1 is type III_1** (Bost-Connes 1995,
Connes-Marcolli 2006). The factor pi_1(A_BC)" is type III_1.

### Verdict on the Bratteli-Robinson search

There is NO theorem of the form "unique KMS state implies discrete
spectrum." In fact the opposite tendency holds:

- Type I factors: many KMS states at high beta, discrete spectrum
- Type III_1 factors: unique KMS state at beta = 1, CONTINUOUS
  Connes spectrum

The uniqueness of omega_1 at beta = 1 is a CONSEQUENCE of the
type III_1 structure, not a constraint that forces discreteness.

## 4. The spectral measure uniqueness argument

The spectral measure mu of T_BC w.r.t. Omega_1 is unique. Does a
unique spectral measure for a symmetric operator imply discrete
spectrum?

**Answer: NO.** The spectral measure of ANY self-adjoint operator
with respect to a FIXED cyclic vector is unique (spectral theorem).
This has nothing to do with the uniqueness of the KMS state.
Every self-adjoint operator on a separable Hilbert space has a
unique spectral measure w.r.t. each vector. This is the spectral
theorem, not a special property of KMS_1.

The argument conflates two different "uniquenesses":
- Uniqueness of omega_1 as a KMS state (non-trivial, number-theoretic)
- Uniqueness of the spectral measure for a given vector (trivial,
  always true by the spectral theorem)

## 5. A subtler version: does uniqueness constrain the spectral TYPE?

Suppose there are two self-adjoint extensions of T_BC (call them
T_1 with discrete spectrum and T_2 with continuous spectrum). Both
give rise to KMS states. If the KMS state is unique, then T_1 and
T_2 must give the SAME KMS state. Does this force T_1 = T_2?

**Answer: Not directly.** The KMS state depends on the algebra AND
the dynamics, not just the Hamiltonian. Two different Hamiltonians
can give the same KMS state if the algebra sees them identically.
Moreover, T_BC is essentially self-adjoint (Nelson's theorem applies),
so there is only ONE self-adjoint extension. The uniqueness of the
extension is already guaranteed; it doesn't constrain the spectral
type.

## 6. Premise check

**Does this argument distinguish spec = {gamma_n} from
spec = {gamma_n} union {extra}?**

NO. The uniqueness of omega_1 is:
(a) A property of the ALGEBRA (A_BC, sigma_t), not of the spectrum
(b) Compatible with both discrete and continuous spectrum
(c) Already used in the construction of H_1 (it's the input, not
    the output)

The argument is VACUOUS for the spectral realisation question.

## 7. What KMS_1 uniqueness DOES give

Despite not forcing compact resolvent, omega_1 uniqueness provides:

1. **A canonical GNS representation** -- there is ONE Hilbert space
   H_1, not a family. This is infrastructure, not a spectral
   constraint.

2. **Extremality** -- omega_1 is an extremal (factorial) KMS state,
   meaning pi_1(A_BC)" is a factor. Combined with the ITPFI
   structure, this gives type III_1.

3. **The Connes-Takesaki flow of weights** -- the flow of weights on
   the type III_1 factor is ergodic, which is equivalent to KMS_1
   uniqueness. This is a reformulation, not a new constraint.

## 8. Verdict

**DEAD END.** KMS_1 uniqueness does not buy compact resolvent.
The type III_1 structure that produces the uniqueness is actually
IN TENSION with discrete spectrum (see Direction 4 for the detailed
analysis). This direction is closed.
