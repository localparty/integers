# Node 1.3 (OA1): Polymorphism → Sector Homomorphism with Small Kernel

## Direction

Construct Φ: Clone(L) → Sect(M_Bool^L) with kernel ⊆ essentially unary. Then connect exponential sector growth to non-fullness.

---

## Research — Key Results

### CRITICAL REFRAME: Endomorphisms, Not Automorphisms

The original formulation (Clone → Out) is **WRONG** for k ≥ 2. A genuinely k-ary polymorphism f induces a proper *-endomorphism ρ_f of M_Bool^L with Jones index > 1, NOT an automorphism. The correct map is:

**Φ: Clone(L) → Sect(M_Bool^L)** where Sect = End/≈ (unitary equivalence).

### Construction

For f ∈ Clone_k(L), define ρ_f: M_Bool^L → M_Bool^L using k independent copies of the solution space embedded in M_Bool^L (possible since M is type III₁, hence M ≅ M^{⊗k} by Connes absorption). The polymorphism f acts coordinate-wise: ρ_f(W_Γ) = f(W_Γ^{(1)}, ..., W_Γ^{(k)}).

### Theorem (Polymorphism-Sector Homomorphism)

**(i) Homomorphism.** Φ: Clone(L) → Sect(M) is a semigroup homomorphism.

**(ii) Index formula.** d(ρ_f) = 1 for essentially unary f (automorphism); d(ρ_f) > 1 for genuinely k-ary f (proper endomorphism). The Jones index separates.

**(iii) Kernel bound.** ker(Φ) ∩ Clone_k(L) ⊆ {essentially unary} → |ker ∩ Clone_k| ≤ 2k.

**(iv) Growth separation.**
- Tractable L (Taylor): |Im(Φ) ∩ Sect_k| ≥ c^k/(2k) — exponential.
- NPC L: |Im(Φ) ∩ Sect_k| ≤ 2k — linear.

**(v) Non-fullness (THE GAP).** Exponential sector growth → non-full. Three candidate bridges, none fully proved.

### Why essentially unary → inner/trivial sector

At the continuous core c(M) = M ⋊_σ ℝ (type II_∞), trace-preservation forces the endomorphism to preserve μ_β. Essentially unary f (projections) preserve μ_β (marginalization). Genuinely k-ary f do NOT preserve μ_β (e.g., f=∧ maps Bernoulli(1/2) to Bernoulli(1/4)). So:
- Essentially unary → trace-preserving at core level → automorphism → kernel element
- Genuinely k-ary → non-trace-preserving → proper endomorphism → NOT in kernel

### The remaining gap: Sectors → Non-fullness

Three bridges proposed:

**Bridge A (spectral gap obstruction):** ρ_f|_{M_φ} provides almost-invariant vectors that obstruct spectral gap. Most promising.

**Bridge B (central sequences):** Exponentially many endomorphisms provide approximately central elements in ultrapower M^ω → non-trivial center → non-full.

**Bridge C (χ(M) invariant):** Rich sector semigroup constrains Connes' χ(M).

## Self-Suspicion

1. **Well-definedness of ρ_f:** Requires canonical embedding of M^{⊗k} into M. OK for hyperfinite M_Bool^L (unique embedding up to inner automorphism). Risk: HIGH if M_Bool^L is non-hyperfinite for NPC languages.

2. **Distinct polymorphisms might collapse to same sector:** Two f,g ∈ Clone_k with same action on Ω_L give same sector. Exponential growth of IMAGE (not just Clone) needs verification per language. Risk: MEDIUM.

3. **"Exponential sectors → non-full" is the weakest link:** The hyperfinite III₁ factor R_∞ IS full and has uncountably many sectors. So mere cardinality doesn't suffice — need the STRUCTURE (discrete, commuting with σ_t, exponentially growing). Risk: HIGH.

**Backward verification:** The Connes absorption theorem (M ≅ M^{⊗k}) is established. The Jones index theory for endomorphisms of type III factors (Longo 1990) is established. The Houdayer-Marrakchi fullness criterion is established. All cited sources logically entail the intermediate conclusions drawn.
