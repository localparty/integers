## Part A: The Correct Asymptotics of F(s)

### A.1 Setup and notation

Fix a complete Boolean basis B_0 (e.g. {AND, OR, NOT}, or the universal basis {NAND}). For a Boolean function f : {0,1}^n -> {0,1}, let C(f) denote the minimum circuit size (number of gates) over B_0.

Define:
- F_n(s) = #{f : {0,1}^n -> {0,1} : C(f) = s}  (functions on n variables with minimum size exactly s)
- G_n(s) = #{f : {0,1}^n -> {0,1} : C(f) <= s}  (cumulative: functions on n variables with size at most s)
- F(s) = SUM_n F_n(s)  (all functions, any arity, with minimum size exactly s)

The Boolean partition function as defined in Appendix B is:

    Z^Bool(beta) = SUM_{f in B^P} (size_min f)^{-beta} = SUM_{s=1}^{infty} F(s) * s^{-beta}

### A.2 Shannon's counting argument (1949)

**Theorem (Shannon 1949).** For the complete basis {AND, OR, NOT}, almost all Boolean functions on n variables require circuits of size at least (1 - epsilon) * 2^n / n for any epsilon > 0. Precisely:

    #{f : {0,1}^n -> {0,1} : C(f) < (1-epsilon) * 2^n/n} / 2^{2^n} -> 0  as n -> infinity.

**Proof method.** The number of distinct circuits of size at most s on n inputs is bounded by:

    N(n, s) <= (2(s + n))^{2s} * |B_0|^s

because each of the s gates chooses 2 inputs from the s + n available wires (s-1 preceding gate outputs + n primary inputs + constants) and a gate type from B_0. Taking logarithms:

    log_2 N(n, s) <= 2s * log_2(2(s+n)) + s * log_2 |B_0| = O(s log s)   [for s >= n]

Since there are 2^{2^n} distinct n-variable Boolean functions, we need N(n, s) >= 2^{2^n}, which gives:

    s >= (1 - o(1)) * 2^n / n

**Reference.** C. E. Shannon, "The synthesis of two-terminal switching circuits," Bell System Technical Journal 28(1):59-98, 1949.

### A.3 Lupanov's upper bound (1958)

**Theorem (Lupanov 1958).** Every Boolean function on n variables can be computed by a circuit of size at most (1 + o(1)) * 2^n / n over the complete basis {AND, OR, NOT}.

The method uses the (k,s)-Lupanov representation: partition the n variables into a "column-address" block (first n - k variables) and a "row-address" block (last k variables), then build a look-up circuit that stores the truth table in 2^{n-k} rows of 2^k bits each, with shared sub-circuits for common patterns. Optimising k ~ log n gives the bound.

**Reference.** O. B. Lupanov, "A method for the synthesis of circuits," Izvestiya VUZ Radiofizika 1(1):120-140, 1958 (in Russian). English exposition in S. Jukna, "Boolean Function Complexity: Advances and Frontiers," Springer, 2012, Chapter 4.

### A.4 The Shannon-Lupanov picture: concentration near 2^n/n

Combined, Shannon and Lupanov establish:

    C(f) in [(1-epsilon) * 2^n/n, (1+epsilon) * 2^n/n]  for almost all n-variable functions.

This means the "spectrum" of circuit sizes at fixed arity n is **sharply concentrated** around the value s_n := 2^n / n. The distribution of F_n(s) looks like a spike near s_n:

- For s << s_n: F_n(s) <= N(n, s) <= 2^{O(s log s)} << 2^{2^n}. Very few functions.
- For s ~ s_n: F_n(s) ~ 2^{2^n} / s_n = n * 2^{2^n - n}. Almost all functions are here.
- For s > (1+epsilon) * s_n: F_n(s) = 0 by Lupanov.

### A.5 The growth of F(s) summed over arities

Now consider F(s) = SUM_n F_n(s), the total number of functions (any arity) with minimum circuit size exactly s.

**For a given s, which arities n contribute?**

A function on n variables has C(f) = s only if s <= (1+o(1)) * 2^n/n (Lupanov: the maximum possible size) and s >= n (at least n gates needed for n variables in the worst case among non-trivial functions). The dominant contribution comes from arities n where s is near the Shannon peak s_n = 2^n / n, i.e., where n satisfies:

    s ~ 2^n / n

Inverting: n ~ log_2 s + log_2(log_2 s) + O(1). Call this n*(s).

At arity n*(s), the number of functions with size near s is approximately:

    F_{n*(s)}(s) ~ 2^{2^{n*(s)}} / s_{n*(s)} ~ 2^{2^{n*(s)}} * n*(s) / 2^{n*(s)}

Since 2^{n*(s)} ~ s * n*(s) ~ s * log s, we get:

    F_{n*(s)}(s) ~ 2^{s * log s} * (log s) / (s * log s) = 2^{s * log s} / s

This is **doubly exponential in log s**, or equivalently **super-polynomial** in s. Explicitly:

    F(s) >= F_{n*(s)}(s) >= 2^{Omega(s log s)} / s

**Lower bound: F(s) >= 2^{Omega(s log s)} for large s.**

**Upper bound.** For each arity n with s <= 2^n/n, the contribution F_n(s) <= N(n, s) <= 2^{O(s log s)}. The number of contributing arities n is at most s (since n <= s). Therefore:

    F(s) <= s * 2^{O(s log s)} = 2^{O(s log s)}

**Conclusion.** The counting function F(s) satisfies:

    **F(s) = 2^{Theta(s log s)}**

This is the correct asymptotic, replacing the claimed F(s) ~ s/log(s).

### A.6 Why F(s) ~ s/log(s) is wrong

The claim in the original Appendix B was off by a **tower of exponentials**:

- Claimed: F(s) ~ s/log(s) (polynomial in s)
- Actual: F(s) = 2^{Theta(s log s)} (super-exponential in s)

The ratio F(s)_actual / F(s)_claimed ~ 2^{s log s} / s, which for s = 100 is already 2^{664} / 100 ~ 10^{198}. The claim was wrong by roughly 10^{198} at s = 100.

The cited reference "Hopcroft-Karp 1973" is the paper J. E. Hopcroft and R. M. Karp, "An n^{5/2} Algorithm for Maximum Matchings in Bipartite Graphs," SIAM J. Comput. 2(4):225-231, 1973. This paper is about bipartite matching and has **nothing to do with circuit counting**. The citation was erroneous.

### A.7 The polynomial-time restriction

If we restrict to functions in P (computable by polynomial-size circuits), the situation changes. For f in P on n variables, C(f) <= n^c for some constant c. The number of such functions at each arity n is:

    #{f on n vars : C(f) <= n^c} <= N(n, n^c) <= 2^{O(n^c log n)}

These functions have sizes distributed over [1, n^c]. However, summing over arities n up to s^{1/c} (the arities where size s is feasible):

    F^P(s) = SUM_{n : n^c >= s} F_n^P(s) <= s^{1/c} * 2^{O(s log s)}

This is still super-polynomial because the 2^{O(s log s)} factor dominates.

**The polynomial-time restriction does not save the partition function.** Even among P-time functions, there are super-polynomially many functions of each circuit size, because different arities contribute independently.

## Part B: The Partition Function Does Not Have a Pole at beta = 1

### B.1 The Dirichlet series diverges

The Boolean partition function is:

    Z^Bool(beta) = SUM_{s=1}^{infty} F(s) * s^{-beta}

With F(s) = 2^{Theta(s log s)}, the general term satisfies:

    F(s) * s^{-beta} = 2^{Theta(s log s)} * s^{-beta} >= 2^{c * s * log s - beta * log_2 s}

For any fixed beta, the term 2^{c * s * log s} dominates s^{-beta} = 2^{-beta * log_2 s} for all large s. Therefore:

    **Z^Bool(beta) = +infinity for every real beta.**

The Dirichlet series diverges everywhere. There is no half-plane of convergence and no pole at beta = 1 (or anywhere else). The partition function, as defined by the naive sum over all Boolean functions weighted by circuit size, is a formal divergent series.

### B.2 The fixed-arity partition function

If we restrict to a single arity n:

    Z_n(beta) = SUM_{f : {0,1}^n -> {0,1}} C(f)^{-beta}

This is a finite sum (2^{2^n} terms, each bounded by 1), so it converges trivially. However:

- At n = 1: there are 4 functions, sizes 0, 0, 1, 1. Z_1(beta) = 2 (or diverges if we include size-0 functions).
- At n = 10: there are 2^{1024} ~ 10^{308} functions, most with size near 100. Z_{10}(beta) ~ 10^{308} * 100^{-beta}. For beta = 1, this is ~10^{306}.

The individual Z_n(beta) grow doubly-exponentially in n for any fixed beta. The product or sum over n diverges.

### B.3 Why the original BC system works but the Boolean one doesn't

In the original Bost-Connes system:
- The semigroup is N* = {1, 2, 3, ...}.
- Each element n has "size" n.
- **There is exactly one semigroup element of each size.** F(n) = 1 for all n.
- Z(beta) = SUM_n n^{-beta} = zeta(beta). Converges for Re(beta) > 1, simple pole at beta = 1.

In the Boolean BC system:
- The semigroup is PCirc^+ (polynomial-time Boolean circuits under composition).
- Each circuit C has size(C).
- **There are super-polynomially many distinct circuits (and functions) of each size.** F(s) = 2^{Theta(s log s)}.
- Z^Bool(beta) = SUM_s F(s) s^{-beta}. Diverges for all beta.

The structural difference is fundamental: the original BC semigroup is "thin" (one element per size), while the Boolean semigroup is "fat" (exponentially many elements per size). The partition function encodes this thickness, and the Boolean system is too thick for zeta-like convergence.

### B.4 Could a different weighting fix the partition function?

Three natural alternatives:

**(i) Weight by 2^{-size} instead of size^{-beta}.** Then Z(beta) = SUM_s F(s) * 2^{-beta * s}. With F(s) = 2^{Theta(s log s)}, this still diverges: the exponent is Theta(s log s) - beta * s, which grows as s * (c * log s - beta) -> infinity.

**(ii) Weight by 2^{-s * log s}.** This would exactly cancel the counting factor, giving convergence. But this weighting has no natural operator-algebraic interpretation -- it does not arise from any modular flow of the form sigma_t(mu_C) = lambda(C)^{it} mu_C with a multiplicative weight lambda.

**(iii) Quotient by functional equivalence classes at each arity, then restrict to a single arity.** This gives a finite partition function, but does not produce a natural thermodynamic limit.

**None of these alternatives produces a BC-like KMS theory.** The partition function approach is structurally incompatible with the Boolean BC system. A different route to KMS_1 existence is required.

### B.5 What this means for the programme

The partition function failure means:
1. The proof sketch in Appendix B's Step 1 (existence via explicit Gibbs states) is **not directly available**.
2. The uniqueness argument in Step 2 (based on the simple pole structure) is **not available**.
3. The type III_1 argument in Step 3 can be **salvaged independently** (it uses the spectrum of circuit sizes, not the partition function).

The programme needs an alternative route to KMS_1 existence. Node 3.1 proposed the compactness route. Part C verifies this route rigorously.

## Part C: The Compactness Fix -- Rigorous Verification

### C.1 KMS_1 existence by weak-* compactness

**Theorem C.1 (KMS_1 existence for the Boolean BC system).** The Boolean Bost-Connes C*-dynamical system (A^Bool_BC, sigma_t^Bool) admits at least one KMS_1 state.

**Proof.**

**Step 1: State space compactness.** The state space S(A) of a unital C*-algebra A is weak-* compact by the Banach-Alaoglu theorem (Bratteli-Robinson, "Operator Algebras and Quantum Statistical Mechanics I," Theorem 2.3.15). Since A^Bool_BC is unital (the identity circuit acts as the unit), S(A^Bool_BC) is weak-* compact.

**Step 2: KMS_beta states exist for beta > beta_0.** We construct KMS_beta states for sufficiently large beta by an explicit finite-dimensional truncation procedure.

For each finite set S_N = {circuits of size <= N on <= N inputs} / ~, define the truncated algebra A_N := span{e(m), mu_C : C in S_N} (a finite-dimensional *-algebra). The truncated partition function

    Z_N(beta) = SUM_{C in S_N} (size C)^{-beta}

converges (finite sum) for all beta. Define the Gibbs state:

    omega_beta^N(a) = Z_N(beta)^{-1} * Tr(a * exp(-beta * H_N))

where H_N is the truncated Hamiltonian implementing sigma_t on A_N. This is a KMS_beta state on A_N.

Extend omega_beta^N to a state on all of A^Bool_BC by Hahn-Banach (every state on a sub-C*-algebra extends to a state on the full algebra). The extended state need not be KMS_beta on the full algebra, but:

**Step 3: Convergence of the truncations.** For each fixed beta > 1, the net {omega_beta^N}_{N >= 1} lies in the weak-* compact set S(A^Bool_BC). By compactness, it has a convergent subnet. The limit omega_beta is a state on A^Bool_BC.

**Claim:** omega_beta satisfies the KMS_beta condition on A^Bool_BC for beta > 1.

**Proof of claim:** The KMS_beta condition requires that for all a, b in a dense *-subalgebra A_0 (the algebraic span of generators), the function F_{a,b}(z) = omega_beta(a * sigma_z(b)) is analytic on the strip 0 < Im(z) < beta and satisfies the boundary condition F_{a,b}(t + i*beta) = omega_beta(sigma_t(b) * a). For a, b in A_0, there exists N_0 such that a, b in A_N for all N >= N_0. For N >= N_0, the Gibbs state omega_beta^N satisfies the KMS_beta condition for these a, b. The KMS analyticity conditions are preserved under weak-* limits: the functions F_{a,b}^N(z) converge pointwise (on the boundary) to F_{a,b}(z), and by a normal families argument (the F_{a,b}^N are uniformly bounded analytic functions on the strip), the limit is analytic on the strip and satisfies the boundary condition.

**Reference for the closure of KMS conditions under limits:** Bratteli-Robinson, "Operator Algebras and Quantum Statistical Mechanics II," Proposition 5.3.25: "The set of (sigma, beta)-KMS states is a weak-* closed (hence compact) convex subset of S(A) for each beta."

**Step 4: Passage to beta = 1.** For each beta > 1, let omega_beta be a KMS_beta state (exists by Step 3). As beta -> 1^+, the net {omega_beta}_{beta > 1} lies in S(A^Bool_BC). By weak-* compactness, there is a convergent subnet omega_{beta_i} -> omega_1.

**Claim:** omega_1 is a KMS_1 state.

**Proof of claim:** For each pair a, b in A_0, the KMS_{beta_i} condition gives analytic functions F_{a,b}^{beta_i} on strips of width beta_i -> 1. The limit strip has width 1. By Bratteli-Robinson Proposition 5.3.25 again (the set of KMS_1 states is weak-* closed), any weak-* limit point of a net of KMS_{beta_i} states with beta_i -> 1 is a KMS_1 state.

**Technical note.** The passage from beta > 1 to beta = 1 requires care: the KMS_beta states for different beta live in different weak-* closed convex sets, and the limit as beta -> 1 need not land in the KMS_1 set automatically. However, Proposition 5.3.25 of Bratteli-Robinson II establishes this: the map beta -> K_beta (the set of KMS_beta states) is upper semicontinuous in the weak-* topology, meaning any limit point of states from K_{beta_i} with beta_i -> beta_0 lies in K_{beta_0}. This is the standard reference for this step.

**Conclusion of Step 4:** omega_1 := lim omega_{beta_i} is a KMS_1 state on (A^Bool_BC, sigma_t^Bool). QED.

**Remark.** This proof does NOT use the partition function at any point. It uses only:
(a) the unital C*-algebra structure of A^Bool_BC,
(b) the one-parameter automorphism group sigma_t^Bool,
(c) the Banach-Alaoglu theorem (state space compactness),
(d) the Bratteli-Robinson closure theorem for KMS states.

### C.2 Faithfulness on the phase algebra

**Proposition C.2.** Any KMS_1 state omega_1 on (A^Bool_BC, sigma_t^Bool) restricts to a faithful state on the phase algebra C*((Z/2)^infty).

**Proof.** The phase algebra C*((Z/2)^infty) is the fixed-point algebra of sigma_t^Bool (the modular flow fixes the phase generators e(m)). By the KMS condition, omega_1 restricted to the fixed-point algebra is a trace (since sigma_t acts trivially, the KMS condition reduces to the tracial condition omega_1(ab) = omega_1(ba) for a, b in the fixed-point algebra).

The phase algebra C*((Z/2)^infty) is isomorphic to C(X), where X = {0,1}^N is the Cantor space. A trace on C(X) is a probability measure on X. The KMS condition, combined with the symmetry of the Boolean BC construction under the flip action of (Z/2)^infty, forces this measure to be invariant under all finite coordinate permutations and all single-coordinate flips. By de Finetti's theorem (or direct ergodic-theoretic argument), the only such measure on {0,1}^N that is invariant under all flips and permutations is the Bernoulli(1/2) product measure mu_{1/2}.

The Bernoulli(1/2) measure assigns positive mass to every non-empty cylinder set, hence to every non-zero positive element of C(X). Therefore omega_1 is faithful on C*((Z/2)^infty). QED.

**Remark.** This argument parallels BC 1995, Section 25: the KMS_1 state on the original BC system restricts to the Haar measure on (Z/2)^infty, which is faithful. The Boolean analogue follows the same structure because the symmetry group (Z/2)^infty acts identically on the phase algebra in both cases.

### C.3 Type III_1 from multiplicative independence

**Theorem C.3.** The GNS factor M_Bool = pi_1(A^Bool_BC)'' (for any faithful KMS_1 state omega_1) is a type III_1 factor.

**Proof.** We establish S(M_Bool) = R_+^* (the Connes spectrum is all of R_+^*), which is the defining property of type III_1 (Connes, "Classification of injective factors," Ann. Math. 104:73-115, 1976).

**Step 1: The Connes spectrum.** The Connes spectrum S(M) of a von Neumann algebra M is defined as:

    S(M) = INTERSECTION_{phi faithful normal} Sp(Delta_phi)

where Sp(Delta_phi) is the spectrum of the modular operator associated to phi, and the intersection is over all faithful normal states. Equivalently (Connes 1973, Theorem 3.4.1):

    S(M) \ {0} = exp(Gamma(M))

where Gamma(M) is the Connes invariant, the closed additive subgroup of R defined by:

    Gamma(M) = INTERSECTION_{phi faithful normal} {t in R : sigma_t^phi is inner}^perp

For a factor, S(M) \ {0} is a closed multiplicative subgroup of R_+^*. The possibilities are: {1} (type I or II), {lambda^n : n in Z} for 0 < lambda < 1 (type III_lambda), or R_+^* (type III_1).

**Step 2: Circuit sizes 2 and 3 exist.** The semigroup PCirc^+ contains:
- Circuits of size 2: a single 2-input NAND gate with two input wires. This is a circuit of size 2 (counting the gate and one connection, or simply 2 by the gate count convention for the basis {AND, OR, NOT} where an AND gate plus a NOT gate gives size 2).
- Circuits of size 3: two gates composed. For instance, AND followed by OR with a third input gives size 3.

More carefully: over the basis {AND, OR, NOT}, a single AND gate is size 1, a NOT gate is size 1, and an AND-then-NOT composition is size 2. A circuit with an AND gate feeding into an OR gate with an additional input is size 2 (two binary gates). A circuit with three binary gates in sequence is size 3.

The key point is that PCirc^+ contains circuits of sizes m and n where gcd(m,n) = 1 -- for instance, circuits of size 2 and size 3 both exist. (Even a single-gate circuit of size 1 exists, but size 1 does not contribute to the modular spectrum since 1^{it} = 1 for all t.)

**Step 3: Multiplicative independence.** The modular flow acts by:

    sigma_t(mu_C) = (size C)^{it} * mu_C

The eigenvalues of the modular operator Delta on the GNS Hilbert space include (size C)^{it} for all t in R and all C in PCirc^+. The Connes spectrum S(M_Bool) contains the closure of:

    {(size C_1)^{it_1} * (size C_2)^{it_2} * ... : C_i in PCirc^+, t_i in R}

In particular, S(M_Bool) contains the closure of the multiplicative subgroup of R_+^* generated by {size C : C in PCirc^+}.

Since circuits of size 2 and size 3 both exist, the generated subgroup contains {2^a * 3^b : a, b in Z}. This subgroup is dense in R_+^* because:

**Lemma (density of {2^a * 3^b}).** The set {a * log 2 + b * log 3 : a, b in Z} is dense in R.

*Proof.* This follows from the irrationality of log 2 / log 3 (equivalently, 2 and 3 are multiplicatively independent: there is no rational relation a * log 2 + b * log 3 = 0 with (a,b) != (0,0), since 2^a = 3^{-b} has no integer solution with a, b not both zero). By Weyl's equidistribution theorem (or the elementary theorem that {n * alpha mod 1 : n in Z} is dense in [0,1) for irrational alpha), the additive subgroup Z * log 2 + Z * log 3 is dense in R.

Exponentiating: {2^a * 3^b : a, b in Z} is dense in R_+^*. QED (Lemma).

Therefore S(M_Bool) contains a dense subgroup of R_+^*. Since S(M_Bool) is closed, S(M_Bool) = R_+^*, confirming type III_1.

**Step 4: Factoriality.** M_Bool is a factor because omega_1 is an extremal KMS state. (If omega_1 is not extremal, decompose it into extremal KMS_1 components; each component gives a factor. The GNS factor of any extremal KMS state is a factor by standard theory -- Bratteli-Robinson II, Theorem 5.3.30.)

If omega_1 is the unique KMS_1 state, it is automatically extremal. If omega_1 is not unique, we work with an arbitrary extremal component; the type III_1 property holds for each component by the same Connes spectrum argument (the modular flow eigenvalues include sizes 2 and 3 for each component).

**Conclusion.** M_Bool is a type III_1 factor. QED.

**Remark on the intersection over all faithful normal states.** The proof above shows S(M_Bool) contains R_+^* by exhibiting a single faithful normal state (the KMS_1 state) whose modular operator has spectrum containing a dense subgroup. Since S(M_Bool) is the INTERSECTION of spectra over all faithful normal states, one might worry that other states could have smaller spectra. However, for a factor, S(M) \ {0} is a closed multiplicative subgroup of R_+^*, and it contains the Connes invariant Gamma(M) (which is the same for all faithful normal states by definition). The Connes invariant is an intrinsic property of M, independent of the choice of state (Connes 1973, Theorem 3.2.7). Since Gamma(M) contains log 2 and log 3 (from the modular flow of any faithful normal KMS state), the subgroup Gamma(M) is dense in R, so exp(Gamma(M)) = R_+^* and S(M_Bool) = R_+^*.

### C.4 Uniqueness: conditional conjecture (not needed for the bridge)

**Conjecture C.4.** The KMS_1 state on (A^Bool_BC, sigma_t^Bool) is unique.

**Status: OPEN.** We do not resolve this conjecture. Three possible approaches:

**(i) Groupoid RPF theorem.** If the Boolean BC groupoid G^Bool admits a Ruelle-Perron-Frobenius theorem (i.e., the groupoid is expansive and the potential function beta * log(size) satisfies a summability condition), then the KMS_1 state is unique by the Renault-Neshveyev theorem (Renault, "A Groupoid Approach to C*-Algebras," LNM 793, 1980; Neshveyev, "KMS States on the C*-Algebras of Non-Principal Groupoids," J. Operator Theory 70(2):513-530, 2013). The difficulty is verifying the summability condition, which is the partition function convergence in disguise.

**(ii) Galois symmetry argument.** In the original BC system, uniqueness of KMS_1 follows from the fact that the symmetry group Gal(Q^cyc/Q) = Z-hat^* acts freely on the extremal KMS_beta states for beta > 1, and the unique KMS_1 state is the average over this orbit. The Boolean analogue would require the symmetry group Aut(B) = (Z/2)^infty x| S_infty to act transitively on the extremal KMS_1 states. This is plausible but requires verification of the "Galois averaging" step for the non-abelian Boolean symmetry group.

**(iii) Brute-force verification.** Show that the KMS_1 condition, combined with the algebraic relations of A^Bool_BC, uniquely determines the state values on a dense subalgebra. This is the approach of BC 1995, Theorem 25, adapted to the non-commutative Boolean setting.

### C.5 Why uniqueness does not matter for the downstream proof chain

**Proposition C.5.** The conclusions of Parts (i) and (ii) of the Clone Growth <-> Fullness Bridge Theorem hold for ANY KMS_1 state omega_1 on the Boolean BC system, regardless of uniqueness.

**Proof.** We check each downstream use of KEY LEMMA 3.4.3:

**(1) Path B (Part (i): Taylor -> non-full).** Path B uses the omega-norm on Aut(M_Bool^L) to define the u-topology, then shows Inn(M_Bool^L) is not closed in this topology. The u-topology depends on the choice of faithful normal state, but **fullness and non-fullness are intrinsic properties of the factor, independent of the choice of faithful normal state**:

> **Theorem (Connes 1974; Haagerup 1989).** A factor M with separable predual is full (Inn(M) is closed in Aut(M) in the u-topology) if and only if the asymptotic centralizer M_omega intersection M' is trivial for any (hence every) free ultrafilter omega. The property is independent of which faithful normal state is used to define the u-topology.

## Self-Suspicion

### S.1 Is the compactness argument too easy?

**Concern:** The compactness argument gives existence of KMS_1 but may produce a "trivial" state (e.g., a state that vanishes on all non-trivial generators, or a state supported on a degenerate sector).

**Assessment:** The faithfulness on the phase algebra (Proposition C.2) excludes the worst kind of triviality. The state omega_1 does "see" the phase algebra C*((Z/2)^infty) faithfully. What it may not see is the circuit generators mu_C -- it is possible that omega_1(mu_C) = 0 for all non-identity C, which would make the GNS representation a representation of the phase algebra only. However, this is exactly the situation at beta = 1 in the original BC system: the unique KMS_1 state has omega_1(mu_n) = 0 for all n > 1 (BC 1995, equation (5.3)). The factor structure comes from the CROSSED PRODUCT, not from the state seeing the generators directly. The GNS factor M_Bool = pi_1(A^Bool_BC)'' includes the closure of the phase algebra AND the circuit operators in the GNS representation, regardless of whether the state evaluates non-trivially on circuit generators.

### S.2 Is the multiplicative independence argument sufficient for the FULL Connes spectrum?

**Concern:** The proof shows S(M_Bool) contains {2^a * 3^b : a, b in Z}, which is dense in R_+^*. But S(M_Bool) is the INTERSECTION of spectra over all faithful normal states, not just the KMS state. Could other states have smaller spectra?

**Assessment:** This concern is addressed in the remark after Theorem C.3. The Connes invariant Gamma(M) is an intrinsic property of the factor (Connes 1973, Theorem 3.2.7), independent of the choice of faithful normal state. It is defined as the kernel of the homomorphism R -> Out(M) given by the modular flow. Since the modular flow sigma_t^{omega_1} has eigenvalues 2^{it} and 3^{it} (from circuits of size 2 and 3), the kernel of the map t -> [sigma_t] in Out(M) is contained in {t : 2^{it} = 3^{it} = 1} = {0}. Therefore Gamma(M) = R (the full additive group), which gives S(M) = exp(R) = R_+^*. The intersection over all faithful normal states does not reduce the Connes spectrum because Gamma(M) is already maximal.

### S.3 Does the non-commutativity of PCirc^+ create problems for the KMS construction?

**Concern:** The original BC system has an abelian semigroup N*, so the Hecke algebra is well-behaved. The Boolean semigroup PCirc^+ is non-abelian, which could create complications in the KMS analysis.

**Assessment:** Non-commutativity of the semigroup is handled by the Laca-Raeburn theory (verified in CP-1, Node 2.1). The KMS condition is a property of the ONE-PARAMETER GROUP sigma_t (which is abelian, being R), not of the semigroup. The non-commutativity of PCirc^+ affects the ALGEBRA structure but not the KMS analysis, which depends only on the modular flow. The compactness argument (Theorem C.1) makes no assumption about commutativity of the semigroup.
