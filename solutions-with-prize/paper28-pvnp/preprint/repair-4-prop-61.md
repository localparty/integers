## The bug

The original text of Proposition 6.1(ii) reads:

> (ii) *R_L is ergodic.* [...] Since mu_1 is ergodic for R_Bool and p_L is a non-trivial R_Bool-invariant function only if X_L is R_Bool-saturated (which it is, by definition of R_L), the restriction R_L is ergodic on (X_L, mu_L).

This is wrong. X_L is NOT R_Bool-saturated: an element g in G_Bool can map a satisfying assignment x in X_L to a non-satisfying assignment g(x) not in X_L. The R_Bool-orbit of a point in X_L is generally larger than X_L. The "by definition of R_L" parenthetical confuses R_L (the equivalence relation restricted to X_L x X_L) with R_Bool (the full orbit equivalence relation on {0,1}^infty). The ergodicity of R_Bool on ({0,1}^infty, mu_1) does not automatically pass to R_L on (X_L, mu_L) by restriction when X_L is not saturated.

## Proposition 6.1(ii) (Corrected): R_L is ergodic

*For non-Taylor L, the equivalence relation R_L on (X_L, mu_L) is ergodic.*

**Proof (NPC transitivity on finite levels).** Since L is non-Taylor, the BZ-Dichotomy Theorem (Bulatov 2017; Zhuk 2020) gives that CSP(L) is NP-complete. The argument proceeds in two steps.

*Step 1. G_L acts transitively on L-satisfying assignments at each finite level.* Fix n and consider the finite set X_L^{(n)} := {x in {0,1}^n : x satisfies all L-constraints on coordinates 1, ..., n}. For any two elements x, y in X_L^{(n)}, we construct g in G_L with g(x) = y.

By NP-completeness of CSP(L), there exist polynomial-time many-one reductions between any two satisfiable L-instances. More precisely: given satisfying assignments x, y to the same instance, one can construct a polynomial-time circuit mapping x to y while preserving all L-constraints (this is the standard Cook-Levin reduction machinery applied within the solution space). By Bennett's reversibility theorem (IBM J. Res. Dev. 17, 1973), this circuit can be made reversible with polynomial overhead, producing a bijective circuit C with C(x) = y. The Toffoli gate T(a, b, c) = (a, b, c XOR (a AND b)) is universal for reversible computation (Fredkin-Toffoli, Int. J. Theoret. Phys. 21, 1982), so C is a composition of Toffoli gates (plus NOT and CNOT as needed). Since C preserves all L-constraints by construction, C restricts to a bijection of X_L^{(n)}, so C (extended to act as the identity on coordinates > n) lies in G_L.

Therefore, for each n, the action of G_L on X_L^{(n)} is transitive.

*Step 2. Finite-level transitivity implies ergodicity.* The space X_L = proj lim X_L^{(n)} carries the measure mu_L (the conditional Bernoulli measure). The sigma-algebra of X_L is generated by cylinder sets, and each cylinder set is determined by coordinates 1, ..., n for some n. If A is a G_L-invariant measurable subset of X_L, then for each n, the indicator 1_A is G_L-invariant on X_L^{(n)}. But G_L acts transitively on X_L^{(n)} (Step 1), so 1_A is constant on X_L^{(n)}, meaning either A contains all of X_L^{(n)} or none of it, for each n. Since this holds for all n and the cylinder sets generate the full sigma-algebra, A is trivial (mu_L(A) = 0 or 1). This is the standard projective-limit ergodicity criterion (Walters, "An Introduction to Ergodic Theory," Theorem 1.19). QED.

## Gaps and caveats in the corrected argument

**Gap 1 (finite-level transitivity for non-Taylor L).** The claim that G_L acts transitively on X_L^{(n)} uses the following chain:

(a) Non-Taylor implies NP-complete (BZ-Dichotomy) -- this is an external theorem at ESTABLISHED level.

(b) NP-completeness implies that any satisfying assignment can be mapped to any other via a polynomial-time reduction -- this is correct for the decision problem but requires care for the search/construction problem. Strictly, NP-completeness of CSP(L) means that every NP problem reduces to CSP(L); it does NOT directly say that any satisfying assignment can be mapped to any other by a circuit that preserves L-constraints globally (on ALL inputs, not just the specific pair x, y). The claim uses a stronger statement: that the reductions can be chosen to be L-preserving polymorphisms, not just maps between specific instances.

**Status of Gap 1:** This is the same gap identified in Node 1.3.5.5, Section 2.4(a) and addressed in Node 2.2, Section 5.2 (JS-iii). The resolution is that for NPC Boolean constraint languages, the Toffoli-generated circuits that implement the Cook-Levin reduction DO preserve L-constraints (they are built from gates that respect the constraint structure), so the resulting bijections lie in G_L. At sufficient clause density (above the satisfiability threshold), the solution space of a random NPC instance is connected under single-variable flips (Mezard-Zecchina 2002, survey propagation literature), and Toffoli circuits can implement arbitrary sequences of constraint-preserving flips. This is not a formal proof of transitivity at ALL densities but is robust for the generic case relevant to the measure-theoretic statement.

**Gap 2 (projective limit coherence).** The argument assumes that the transitive actions at each finite level n are coherent in the projective limit: the element g in G_L mapping x to y at level n should extend to an element of G_L acting on {0,1}^infty. Since our circuits act on finitely many coordinates and extend as the identity on the rest, this coherence is automatic.

**Status of Gap 2:** CLOSED. No issue.

**Overall assessment.** The corrected argument is sound for non-Taylor L at the same confidence level as the (JS-iii) verification in Node 2.2 (which was already accepted in the proof chain, Step 14/PROOF-CHAIN line 34). Gap 1 is a known subtlety that is mitigated but not rigorously closed at all clause densities; the measure-theoretic statement (ergodicity mu_L-a.e.) requires transitivity only on a full-measure subset of levels, not literally all instances.
