# Corpus 3 — solutions-with-prize/

*Prize-carrying solutions. 8 papers (6 Clay Millennium + Bakuage Collatz +
Faber-Bloomsbury Goldbach).*

This corpus contains the conjecture-solving papers that **carry an active
prize** — Clay Mathematics Institute for six of the Millennium Prize Problems,
the Bakuage Prize for Collatz, and the Faber-Bloomsbury Prize for Goldbach
(treated as prize-eligible per user decision 2026-04-15). Each is solved by
applying the Paper 12 CBB operator-algebraic machinery + the foundational
geometry from Corpus 1 (`integers/`).

See `strategy/north-star.md` §2 ("Three Pillars") for the per-vertex
four-deliverable pattern. Each solution in this corpus has a matching vertex
in `strategy/<vertex>/` with the four deliverables per the universal-approval
three-pillar structure (see `strategy/universal-approval-three-pillars.md`
and `strategy/universal-approval-run.md`).
