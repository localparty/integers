## Lemma (ITPFI_χ factorization of KMS_1 on the χ-twisted BC algebra)

Let χ be a primitive Dirichlet character mod q, (q, N)=1. Let A_BC be the Bost–Connes C*-algebra (Bost–Connes 1995, §1; Laca–Raeburn 1996, §2), generated by a unitary representation {μ_n}_{n∈ℕ₊} of ℕ₊ by isometries together with C(Ẑ), with the standard BC relations. Define the χ-twisted modular flow on generators by

    σ_t^χ(μ_n) = χ(n) · n^{it} · μ_n,   σ_t^χ|_{C(Ẑ)} = id.

This extends uniquely to a one-parameter group of *-automorphisms of A_BC, because multiplicativity (χ mult., n↦n^{it} mult.) forces compatibility with the monoidal relations μ_{nm}=μ_n μ_m. We denote the resulting dynamical system (A_BC, σ_t^χ) by A_{BC,χ}.

Then the following hold:

**(1) Prime factorization of the algebra and flow.** For each prime p, let A_{BC,χ}^{(p)} := C*(μ_{p^k} : k≥0) ⊆ A_BC (the sub-C*-algebra generated by the powers of μ_p, together with the p-component of C(Ẑ)). Then

    A_BC ≅ ⊗_p^{min} A_BC^{(p)},      σ_t^χ = ⊗_p σ_t^{χ,(p)},

with σ_t^{χ,(p)}(μ_{p^k}) = χ(p)^k p^{kit} μ_{p^k}. (Minimal C*-tensor product; the factorization of A_BC is Laca–Raeburn 1996 Prop 2.1; the factorization of σ_t^χ is automatic because χ and n↦n^{it} are both completely multiplicative, so the twist is diagonal in the prime decomposition μ_n = ∏_p μ_{p^{v_p(n)}}.)

**(2) Local KMS_1 state, existence and uniqueness.** At each prime p, A_{BC,χ}^{(p)} is isomorphic to the Toeplitz algebra 𝒯 (Murphy 1987) generated by a single isometry μ_p satisfying μ_p^* μ_p = 1, μ_p μ_p^* = projection. The local flow σ_t^{χ,(p)} is the gauge action twisted by χ(p)∈U(1): σ_t^{χ,(p)}(μ_p) = (χ(p) p^{it}) μ_p. **Bratteli–Robinson Vol. II, Theorem 5.3.30** (on KMS states for quasi-free dynamics with bounded generator on product-type algebras; the Toeplitz/UHF case is the standard corollary) gives: the KMS_β state at inverse temperature β exists and is unique whenever the local partition function

    Z_{χ,p}(β) = Σ_{k≥0} |χ(p)^k p^{-kβ}| = Σ_{k≥0} p^{-kβ} = (1 − p^{-β})^{-1}

converges, i.e. for β > 0. At β=1 the state is

    ω_{1,χ}^{(p)}(μ_{p^k} (μ_{p^j})^*) = δ_{kj} · χ(p)^k p^{-k}   (1 − p^{-1}),

which is the Gibbs state with Hamiltonian H_p = (log p) · N_p and χ-phase in the off-diagonal. Existence + uniqueness are **exactly** Laca–Raeburn 1996 Prop. 2.1 applied with the twisted flow; the twist is a bounded cocycle (|χ(p)|=1) so it does not alter the existence/uniqueness domain.

**(3) Tensor product state is KMS_1.** Define ω_{1,χ} := ⊗_p ω_{1,χ}^{(p)} on A_BC = ⊗_p^{min} A_BC^{(p)}. The tensor product state is well-defined because the partition function factorizes:

    Z_χ(1) = ∏_p Z_{χ,p}(1) · (twist numerator) = ∏_p (1 − χ(p) p^{-1})^{-1} = L(1, χ),

which is **finite and nonzero** for χ primitive non-principal (classical, Dirichlet 1837; Davenport Ch. 4). Hence ω_{1,χ} is a bounded normalized state.

For a,b elementary tensors a = ⊗_p a_p, b = ⊗_p b_p (with a_p = b_p = 1 for all but finitely many p), the KMS_1 condition follows prime-by-prime:

    ω_{1,χ}(a σ_{t+i}^χ(b)) = ∏_p ω_{1,χ}^{(p)}(a_p σ_{t+i}^{χ,(p)}(b_p))
                             = ∏_p ω_{1,χ}^{(p)}(σ_t^{χ,(p)}(b_p) a_p)    (local KMS_1 at each p)
                             = ω_{1,χ}(σ_t^χ(b) a).

By C*-norm density of finite elementary tensors in A_BC and continuity of both sides in the norm topology, KMS_1 extends to all a,b ∈ A_BC.

**(4) Uniqueness and identification.** GRH L2 (PROVED, T9) established that (A_BC, σ_t^χ) admits a **unique** KMS_1 state (the fixed-point algebra of the gauge is trivial because χ non-principal kills invariants at any p ∤ q; the n^{it} factor handles the rest as in untwisted BC, Bost–Connes Thm 25). The tensor product state ω_{1,χ} constructed in (3) satisfies KMS_1, so by L2 it **is** the (unique) KMS_1 state. Factorization is confirmed.

## Subtleties

- **Complex χ.** Steps (1)–(4) are character-agnostic: χ(p)∈U(1) enters only as a unimodular phase, so |Z_{χ,p}(β)| = (1−p^{-β})^{-1} is the same as untwisted BC and Bratteli–Robinson 5.3.30 applies verbatim. The resulting ω_{1,χ} is complex-valued on off-diagonal elements (as required — the state is not tracial for non-real χ); this is consistent because the KMS condition at β=1 does not require reality of ω, only the analytic-continuation identity.
- **Real χ.** χ(p)∈{±1}, so ω_{1,χ} restricts to a real-valued state on the *-subalgebra generated by {μ_p + μ_p^*}; this is the self-adjoint realization used in GRH L3 (T10 §1).
- **Bratteli–Robinson citation.** Theorem 5.3.30 is stated there for quasi-free KMS states on CAR/CCR algebras, but the direct adaptation to a single isometry with gauge action is standard (Pimsner–Voiculescu; Laca–Raeburn Prop 2.1 is the explicit BC version). Any of these three references suffices; we use Laca–Raeburn as the cleanest.
