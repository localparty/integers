## Finding 1: s_0 identification requires correction

T7 wrote s_0 -> mu_2*mu_2. But mu_2*mu_2 = 1 (isometry relation), which is not an isometry with proper range. The correct even-branch isometry is s_0 = mu_2 itself: mu_2*mu_2 = 1 (isometry) and mu_2 mu_2* = P_even (range projection onto even sublattice). The even Collatz step n -> n/2 is then mu_2*, acting as the left inverse. **Status: CORRECTED, works.**

## Finding 2: the +1 shift is NOT e(1) in C*(Q/Z)

In Q/Z the element 1 is trivial (1 = 0 mod Z), so e(1) = e(0) = identity. The +1 additive shift on N* is translation by 1, which is NOT a phase operator e(r) for any r. Phase operators e(r) act as e(r)|n> = exp(2 pi i r n)|n> (diagonal in the natural basis); the translation T|n> = |n+1> is off-diagonal. The translation T does not belong to A_BC = C*(Q/Z) x_rho N* (which contains phases and Hecke operators, both preserving multiplicative structure). **The T7 identification e(1) = additive shift is incorrect.** The additive-multiplicative wall is genuine at this point.

## Finding 3: type III_1 from the (2,3)-adic sub-algebra IS sound

The sub-C*-algebra generated by {mu_2, mu_3} inside the GNS representation of omega_1 gives the ITPFI factor with Araki-Woods parameters lambda_2 = 1/2, lambda_3 = 1/3. Since log(2)/log(3) is irrational, the Connes T-invariant is T(M) = {0}, giving type III_1 (Araki-Woods 1968; Connes 1973 classification). This is the multiplicative part of the Collatz embedding and it IS type III_1. **Status: ESTABLISHED (for the multiplicative sector only).**
