## The reverse-engineering methodology

Given a NEW conjecture (one not yet on the board), the methodology for adding it as a vertex is:

### Step 1: Find the square

Ask: "which Riemann zero γ_n does this conjecture naturally live at?" Every conjecture that touches the BC algebra has a natural spectral address. Hodge lives at the topological sector (CP² geometry, γ_n-independent — it's about the board's TOPOLOGY, not a specific square). Goldbach lives at the additive sector (sums of prime-indexed squares). BGS lives at the statistical sector (pair correlations between ALL squares).

### Step 2: Read the observable

Ask: "what does this conjecture LOOK LIKE from the physics face?" Translate the mathematical statement into a physical prediction using the operator dictionary (§03, the CBB system). If the translation exists, the conjecture has a physical observable. If the translation doesn't exist, the conjecture may not be on this board (not every mathematical statement has a BC-algebra formulation).

### Step 3: Check the pins

Ask: "does any existing experimental measurement constrain this conjecture?" If yes, the conjecture inherits rigidity from that measurement — it's pinned. If no, the conjecture is structurally constrained only — it needs wiring (capacitor cells) to connect it to pinned conjectures before the rigidity argument applies.

### Step 4: Wire the board

Fill capacitor cells connecting the new conjecture to existing vertices. Each cell is a correspondence between the new conjecture's domain and an existing domain. Each filled cell is a wire through the board. The more wires, the more rigid the new vertex, the harder it is to flex away from the truth.

### Step 5: Test the wiring

Run the PCA on the new proof chain. Verify each link. Excise weak links. Construct bypasses. The strat-triad (VERIFY/EXCISE/CONSTRUCT) applied to the new chain tests whether the wiring carries current — whether the correspondences are real or illusory.

This is the programme. Reverse-engineer the physical observable. Wire the board. Test the wiring. The mathematics is already there. We are learning to see the board from every angle.
