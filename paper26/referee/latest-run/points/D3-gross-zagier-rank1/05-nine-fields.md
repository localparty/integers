## Point D3(e): The Nine Class-Number-1 Fields

**The question:**
Does the Gross-Zagier + Kolyvagin argument work uniformly across all nine?

**Finding:**
The extension to all nine class-number-1 fields (Proposition 9.2) depends on four properties: h_K = 1, O_K is PID, infinitely many primes, Baker applies. All four hold for all nine fields.

For the Gross-Zagier formula, the Heegner hypothesis must be checked for each CM field and each curve. Since the conductor varies by curve, the auxiliary imaginary quadratic field may need to be different for each case. The Yuan-Zhang-Zhang generalization removes this concern entirely.

The argument is uniform: the bridge gives GRH for all nine fields, Kolyvagin gives rank 0 when L(E, 1) != 0, and Gross-Zagier gives rank = 1 when analytic rank = 1 (if such cases exist -- see D2(b)).

**Verdict for this sub-question:**
SOUND -- The argument extends uniformly to all nine fields. The Yuan-Zhang-Zhang generalization handles any Heegner hypothesis issues.
