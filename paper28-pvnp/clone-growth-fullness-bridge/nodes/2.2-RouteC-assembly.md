## 1. Step 0: The Framework -- Crossed Product and Orbit Equivalence Relation

### 1.1 The Boolean BC system and its sectors

The Boolean Bost-Connes system is defined in preprint Section 3.2 as:

    A_BC^Bool = C*((Z/2)^infty) rtimes PCirc^+

where C*((Z/2)^infty) = C({0,1}^infty) is the abelian phase algebra and PCirc^+ is the semigroup of polynomial-time Boolean circuits under composition. KEY LEMMA 3.4.3 establishes the existence and uniqueness of a KMS_1 state omega_1^Bool. The GNS factor M_Bool = pi_1(A_BC^Bool)'' is a type III_1 factor.

For a Boolean constraint language L (a finite set of relations on {0,1}), the L-sector is defined (preprint Section 3.5) as:

    M_Bool^L = p_L M_Bool p_L

where p_L in L^infty({0,1}^infty, mu_1) is the characteristic function of the set X_L of L-compatible configurations, and mu_1 is the Bernoulli(1/2) product measure (the KMS_1 measure).

### 1.2 The crossed-product identification (CP-1)

**Theorem (CP-1; Nodes 1.3.5.7 + 1.3.5.9).** The GNS factor decomposes as:

    M_Bool = L^infty({0,1}^infty, mu_1) rtimes G_Bool

where G_Bool is the group of invertible polynomial-time circuits with polynomial-time inverses ("bi-polynomial" circuits). The L-sector decomposes as:

    M_Bool^L = L(R_L)

where R_L is the orbit equivalence relation of G_Bool restricted to X_L:

    R_L = {(x, y) in X_L x X_L : there exists g in G_Bool with g(y) = x}

**How CP-1 is established.** The sub-semigroup PCirc^+_bi of bi-polynomial invertible circuits is left-cancellative and satisfies the left Ore condition (Node 1.3.5.7, Section 4.1: for any A, B in PCirc^+_bi, set C = B circ A^{-1}, D = id; then C circ A = B = D circ B). The group of left fractions is G_Bool. The Laca-Raeburn dilation theorem (J. Funct. Anal. 135, 1996, Theorem 2.1) applied to the left-Ore semigroup PCirc^+_bi gives the group crossed product. The non-invertible circuits are absorbed into the enlarged base algebra A_tilde (standard Hecke mechanism; Node 1.3.5.7, Section 4.2). The passage to the L-sector uses the conditional expectation E_L, which is normal and faithful (Node 1.3.5.9, Sections 2-3; Takesaki, Theory of Operator Algebras II, Chapter X, Theorem 1.7).

**The equivalence-relation formulation.** For Route C the critical observation (Node 1.3.5.9, Section 4.8) is that Marrakchi's Theorem B applies to equivalence relations, not only to group crossed products. The factor L(R_L) is the von Neumann algebra of the countable Borel equivalence relation R_L on the standard probability space (X_L, mu_L). Route C does not require the stronger statement M_Bool^L = A|_L rtimes G_L; the equivalence-relation description suffices.

### 1.3 The acting group

For a constraint language L, define:

    G_L = {g in Homeo({0,1}^infty) : g and g^{-1} are polynomial-time,
           g preserves X_L setwise}

G_L is a countable discrete group (each element is specified by a finite Turing machine description; Node 1.3.5.11, Section 1.2). G_L is a subgroup of G_Bool. The action of G_L on (X_L, mu_L) is by measure-preserving homeomorphisms.

### 1.4 The orbit equivalence relation

The orbit equivalence relation of G_L on X_L is:

    R_{G_L} = {(x, y) in X_L x X_L : there exists g in G_L with g(y) = x}

For non-Taylor L, we have R_{G_L} = R_L (the full equivalence relation restricted to X_L equals the G_L-orbit relation). This is established in Node 1.3.5.9, Claim 4.9.1: the essentially unary polymorphism structure of non-Taylor languages prevents non-G_L group elements from mapping between L-compatible configurations. The underlying mechanism is that any multi-variable transformation mapping solutions to solutions would yield a non-trivial multi-ary polymorphism, contradicting the non-Taylor hypothesis (UA2, Node 1.2).

Therefore, for non-Taylor L:

    M_Bool^L = L(R_L) = L(R_{G_L}) = L^infty(X_L) rtimes G_L

and the group measure space framework applies.

## 2. Step 1: G_L Is Non-Amenable

### 2.1 External theorem

**Theorem (Connes 1976, contrapositive).** Let G be a countable discrete group acting on an amenable von Neumann algebra A by automorphisms. If the crossed product A rtimes G is injective, then G is amenable.

Equivalently: if G is non-amenable and A is amenable, then A rtimes G is non-injective.

**Source:** A. Connes, "Classification of injective factors," Ann. of Math. 104 (1976), 73-115.

### 2.2 Verification of hypotheses for our construction

**(H1) G_L is non-amenable for non-Taylor L.** This is established in Node 1.3.5.5 (Gap Beta), Lemma 2.2.2, via the following chain:

(a) **BZ-DICHOTOMY (Bulatov 2017, Zhuk 2020).** A finite Boolean constraint language L is non-Taylor if and only if CSP(L) is NP-complete. This is the Bulatov-Zhuk dichotomy theorem (independently proved):
- A. Bulatov, "A dichotomy theorem for nonuniform CSPs," FOCS 2017.
- D. Zhuk, "A proof of the CSP dichotomy conjecture," J. ACM 67 (2020).

(b) **NPC reductions generate F_2 via Toffoli gates.** For NP-complete L, the group G_L of L-preserving bi-polynomial circuits contains the free group F_2 on two generators. The construction is as follows:

The Toffoli gate T(a, b, c) = (a, b, c XOR (a AND b)) is universal for reversible classical computation (Fredkin-Toffoli, Int. J. Theoret. Phys. 21, 1982). For NPC L, any satisfying assignment can be mapped to any other by a polynomial-time reduction (by NP-completeness). These reductions can be made reversible with polynomial overhead (Bennett, IBM J. Res. Dev. 17, 1973). For input length n >= 3, the group of Toffoli-generated L-preserving circuits on {0,1}^n contains the alternating group A_{2^n} (since the Toffoli gate, combined with NOT and CNOT, generates A_{2^n} for n >= 3). For 2^n >= 5 (i.e., n >= 3), A_{2^n} contains F_2 as a subgroup.

(c) **F_2 is non-amenable** (von Neumann 1929: F_2 fails the Folner condition; equivalently, F_2 is non-amenable by the ping-pong lemma).

Since G_L contains the non-amenable subgroup F_2, G_L is non-amenable (a group containing a non-amenable subgroup is non-amenable).

### 2.3 Consequence

G_L is non-amenable. Since the base algebra A|_L = L^infty(X_L, mu_L) is abelian (hence amenable), the crossed product A|_L rtimes G_L is non-injective by Connes' theorem.

## 3. Step 2: Rad(G_L) = {e} (Trivial Amenable Radical)

### 3.1 Definition

The amenable radical Rad(G) of a discrete group G is the largest normal amenable subgroup of G. Equivalently (Furman 2003), Rad(G) = ker(G -> Homeo(partial_F(G))), the kernel of the G-action on the Furstenberg boundary.

### 3.2 Why this step is needed

Non-amenability of G_L (Step 1) does not imply Rad(G_L) = {e}. A non-amenable group can have a non-trivial normal amenable subgroup (e.g., F_2 x Z has amenable radical Z). The Jones-Schmidt theorem (Step 4) requires Rad(G_L) = {e} as an explicit hypothesis: a non-trivial amenable normal subgroup N could produce approximately invariant sequences for the quotient action G_L/N, defeating strong ergodicity.

### 3.3 Three independent arguments (Node 1.3.5.12)

**Argument 1: Furstenberg boundary (primary).**

**External theorem (Furman 2003).** For a countable discrete group G, Rad(G) = ker(G -> Homeo(partial_F(G))).

**Source:** A. Furman, "On minimal strongly proximal actions of locally compact groups," Israel J. Math. 136 (2003), 173-187.

**Verification.** Following Frisch-Schlank-Tamuz (Ergodic Theory Dynam. Systems 39, 2019, arXiv:1512.00587), construct a minimal strongly proximal G_L-space via the ultraproduct of probability measure spaces: let omega be a non-principal ultrafilter on N, set M_omega = lim_{n -> omega} P({0,1}^n), and let partial_omega be a minimal strongly proximal closed G_L-invariant subspace.

The action of G_L on partial_omega is faithful: if g in G_L acts trivially on M_omega, then for omega-almost every n, the pushforward action of g on P({0,1}^n) is trivial, which forces g to act trivially on {0,1}^n for omega-almost every n. Since a non-trivial circuit differs from the identity on at least one input, g must be the identity (Node 1.3.5.12, Claim 2.2.1).

By Furman's theorem: Rad(G_L) is contained in ker(G_L -> Homeo(partial_omega)) = {e}. Therefore Rad(G_L) = {e}. QED.

**Argument 2: C*-simplicity via Le Boudec-Matte Bon.**

**External theorem (Le Boudec-Matte Bon, Ann. Sci. ENS 51, 2018, Theorem 1.2; arXiv:1605.01651).** Let G be a group of homeomorphisms of a compact Hausdorff space X. If G acts minimally on X and every rigid stabilizer Rist_G(U) = {g in G : g|_{X \ U} = id} is non-amenable (for every non-empty open U), then G is C*-simple.

**External theorem (BKKO, Publ. Math. IHES 126, 2017, Corollary 1.4; arXiv:1509.01870).** A C*-simple group has trivial amenable radical.

**Verification of Le Boudec-Matte Bon hypotheses for G_L:**

(a) *Minimality.* For NPC L, the action of G_L on X_L is minimal: every orbit is dense. This follows from NP-completeness: for any two L-compatible finite prefixes w, w' of the same length, there exists a polynomial-time L-preserving reduction mapping w to w'. Since the topology on X_L is generated by cylinder sets [w], the action is transitive on cylinders, hence minimal (Node 1.3.5.12, Claim 3.3.1).

(b) *Non-amenable rigid stabilizers.* For any non-empty clopen U in X_L, the rigid stabilizer Rist_{G_L}(U) contains F_2. A clopen U contains a cylinder set [w] for some finite prefix w. For n large enough, the coordinates within U's support contain at least 3 "free" coordinates on which Toffoli gates can act while preserving L (because L has bounded arity, so constraints involving coordinates outside U's support are unaffected). By the Toffoli universality argument (Step 1), the circuits acting within U's support generate a group containing A_{2^m} for m >= 3, which contains F_2 (Node 1.3.5.12, Claim 3.2.1).

Therefore G_L is C*-simple (Le Boudec-Matte Bon), hence Rad(G_L) = {e} (BKKO). QED.

**Argument 3: Direct normal subgroup analysis.**

**External theorem (Jordan, 1873; see Dixon-Mortimer, "Permutation Groups," 1996, Theorem 3.3D).** A primitive permutation group of degree m containing a non-trivial element of support less than m/2 contains the alternating group A_m.

**Verification.** Let N be a non-trivial normal subgroup of G_L. Choose e != g in N. Since g is non-trivial, it acts non-trivially on {0,1}^n for some n. Since N is normal, conjugation by all h in G_L sends g to hgh^{-1} in N. For NPC L, G_L^{(n)} acts transitively on {0,1}^n (NP-completeness). The normal closure of g in G_L^{(n)} generates a group containing A_{2^n} for sufficiently large n (Jordan's theorem, with primitivity following from double transitivity of the circuit group). Since A_{2^n} contains F_2 for 2^n >= 5 (i.e., n >= 3), the normal subgroup N contains F_2 and is therefore non-amenable (Node 1.3.5.12, Section 4).

Every non-trivial normal subgroup of G_L is non-amenable. Therefore Rad(G_L) = {e}. QED.

### 3.4 Analysis of the Arimoto obstruction (2025)

Arimoto (arXiv:2510.13309, October 2025) showed that the canonical action of Thompson's group V on the Cantor set is NOT strongly ergodic and the resulting crossed product is NOT full, despite V being C*-simple with trivial amenable radical. This result does NOT threaten our argument because:

(a) V's canonical action on {0,1}^infty is NOT essentially free (elements of V that permute subtrees fix all points outside those subtrees with positive measure), while G_L's action IS essentially free (Step 3 below).

(b) V is the topological full group of the Cuntz groupoid, which is amenable; G_L is not a topological full group of an amenable groupoid.

(c) The Jones-Schmidt theorem (Step 4) requires essential freeness as an explicit hypothesis. V's action fails this hypothesis; G_L's action satisfies it. The Arimoto result demonstrates precisely why essential freeness is necessary.

See Node 1.3.5.12, Section 5 for the complete analysis.

## 4. Step 3: G_L Acts Essentially Freely on A|_L

### 4.1 Definition

The action G_L curvearrowright (X_L, mu_L) is essentially free if:

    mu_L({x in X_L : Stab_{G_L}(x) != {e}}) = 0

Since G_L is countable, this is equivalent to: for every g in G_L \ {e}, the fixed-point set Fix(g) = {x : g(x) = x} has mu_L-measure zero.

### 4.2 Three independent arguments (Node 1.3.5.11)

**Argument A: Bernoulli measure + circuit structure (primary).**

**Proposition (SE-1; Node 1.3.5.11, Proposition 2.1).** For g in G_L \ {e}, mu(Fix(g)) = 0.

**Proof.** The proof has three stages:

*Stage 1: Positive-density non-fixed set.* Since g != id, there exists x_0 in {0,1}^infty with g(x_0) != x_0. By continuity of g (homeomorphism of the Cantor set), there exists M_0 such that the entire cylinder set C_0 = [x_0|_{[1,M_0]}] consists of non-fixed-points. Hence Fix(g) subset {0,1}^infty \ C_0 and mu(Fix(g)) <= 1 - 2^{-M_0} < 1.

*Stage 2: Fix(g) contains no cylinder set (NPC rigidity).* Suppose for contradiction that Fix(g) contains a cylinder set [w]. Then g fixes every infinite extension of w, hence g acts as the identity on the sub-Cantor-set [w]. For non-Taylor L, the clone Clone(L) consists only of essentially unary operations (UA2, Node 1.2: at most 2k + 2 operations at arity k). An element g of G_L that fixes all extensions of w must act on each remaining coordinate by one of {id, neg, const_0, const_1}. Negation is ruled out (no fixed points); constants are ruled out (g is a bijection). Only the identity remains: g = id, contradicting g != e (Node 1.3.5.11, Claim 2.4).

*Stage 3: mu(Fix(g)) = 0.* Fix(g) is a closed subset of {0,1}^infty (coincidence set of continuous maps g and id). If mu(Fix(g)) > 0, then by regularity of the Bernoulli measure on the Cantor set, Fix(g) contains a cylinder set (any closed set of positive measure in an atomless product space contains a cylinder subset of positive measure). This contradicts Stage 2. Therefore mu(Fix(g)) = 0. QED.

**Argument B: UA2 rigidity (structural).**

For non-Taylor L, Clone_1(L) = {id, neg, const_0, const_1} (four unary operations). An element g in G_L, viewed as a sequence of L-preserving permutations, acts at each coordinate by one of these essentially unary operations. The only bijective operation fixing both values 0 and 1 is the identity. For a Bernoulli(1/2)-generic x, each coordinate x_j is equally likely to be 0 or 1, so the only essentially unary bijection fixing x_j for all j simultaneously is the identity. The stabilizer of a generic point is trivial: mu-a.e. point has trivial stabilizer (Node 1.3.5.11, Section 3).

**Argument C: Topological freeness + atomless measure.**

The action of G_L on {0,1}^infty is topologically free (Fix(g) has empty interior for every g != e, by the NPC rigidity argument of Stage 2). For an atomless fully-supported measure on a compact metrizable space, every closed nowhere-dense set has measure zero. By countability of G_L, the union of all Fix(g) has measure zero (Node 1.3.5.11, Section 4).

### 4.3 Why essential freeness is specific to non-Taylor L

For Taylor (P-time) languages, Clone_k(L) grows exponentially (UA1, Node 1.1). Multi-ary polymorphisms (majority, AND, XOR, etc.) can construct non-trivial group elements that stabilize generic points: med(x, x, z) = x for all z, so circuits implementing med(-, -, z) stabilize x. The essentially unary rigidity that forces essential freeness in the non-Taylor case is absent. This is consistent with the programme structure: for Taylor L, we prove non-fullness (Part (i)), not fullness.

## 5. Step 4: The Orbit Equivalence Relation R_L Is Strongly Ergodic

### 5.1 External theorem

**Theorem (Jones-Schmidt 1987, Theorem 2.1; see also Kechris, "Global Aspects of Ergodic Group Actions," Math. Surveys Monogr. 160, AMS, 2010, Chapter 3).** Let Gamma be a countable group acting on a standard probability space (X, mu) by measure-preserving transformations. Suppose:

(JS-i) Gamma is non-amenable.

(JS-ii) The action is essentially free.

(JS-iii) The action is ergodic.

(JS-iv) Rad(Gamma) = {e} (trivial amenable radical).

Then the orbit equivalence relation R_{Gamma curvearrowright X} is strongly ergodic: there is no sequence (A_n) of measurable subsets of X with 0 < mu(A_n) < 1 that is asymptotically invariant (i.e., mu(g A_n triangle A_n) -> 0 for all g in Gamma) unless mu(A_n)(1 - mu(A_n)) -> 0.

**Source:** V.F.R. Jones, K. Schmidt, "Asymptotically invariant sequences and approximate finiteness," Amer. J. Math. 109 (1987), 91-114.

**Remark on the role of (JS-iv).** The connection between the amenable radical and strong ergodicity is: if Rad(Gamma) contains a non-trivial normal amenable subgroup N, then the N-action can produce non-trivial approximately invariant sequences (by Ornstein-Weiss / Connes-Feldman-Weiss amenability). The quotient action Gamma/N could then inherit asymptotic invariance, defeating strong ergodicity. When Rad(Gamma) = {e}, no such N exists, so the condition is automatically satisfied. This connection is explicated in Schmidt, "Asymptotic properties of unitary representations and mixing," Proc. London Math. Soc. 48 (1984), and Kechris (2010, Chapter 3, Propositions 3.2-3.5).

### 5.2 Verification of hypotheses for our construction

**(JS-i) G_L is non-amenable.** Established in Step 1 (Section 2). G_L contains F_2 via Toffoli gates for NPC L.

**(JS-ii) The action is essentially free.** Established in Step 3 (Section 4). Three independent arguments converge: Bernoulli measure + NPC rigidity, UA2 structural argument, and topological freeness. For every g in G_L \ {e}, mu_L(Fix(g)) = 0.

**(JS-iii) The action is ergodic.** For NPC L, the Hecke action of G_L on {0,1}^infty is transitive on each finite level {0,1}^n restricted to L-compatible configurations (by NP-completeness: any L-satisfying assignment can be mapped to any other via a polynomial-time L-preserving reduction; Node 1.3.5.5, Section 2.4(a)). Transitivity on all finite levels implies ergodicity of the action on the projective limit ({0,1}^infty, mu_1), by the standard criterion: a measure-preserving action on a product space is ergodic if and only if the only invariant measurable sets are trivial, and transitivity on finite levels implies that no non-trivial cylinder set is invariant, which generates the full sigma-algebra (Walters, "An Introduction to Ergodic Theory," Theorem 1.19).

**(JS-iv) Rad(G_L) = {e}.** Established in Step 2 (Section 3). Three independent arguments: Furstenberg boundary, C*-simplicity via Le Boudec-Matte Bon, and direct normal subgroup analysis.

### 5.3 Conclusion

All four hypotheses (JS-i) through (JS-iv) are satisfied. By the Jones-Schmidt theorem, the orbit equivalence relation R_L = R_{G_L curvearrowright X_L} is strongly ergodic. QED.

### 5.4 Contrast with Thompson's V (the Arimoto obstruction)

The Arimoto result (arXiv:2510.13309, 2025) shows that V's canonical action on {0,1}^infty is NOT strongly ergodic despite V satisfying (JS-i) and (JS-iv). The failure occurs at (JS-ii): V's action has non-trivial point stabilizers (elements permuting subtrees fix all exterior points), so it is NOT essentially free. The Jones-Schmidt theorem correctly does not apply to V's canonical action, and correctly does apply to G_L's action.

| Hypothesis | Thompson V (canonical action) | G_L (NPC, KMS action) |
|---|---|---|
| (JS-i) Non-amenable | YES (contains F_2) | YES (Step 1) |
| (JS-ii) Essentially free | **NO** (positive-measure stabilizers) | YES (Step 3) |
| (JS-iii) Ergodic | YES | YES |
| (JS-iv) Rad = {e} | YES (BKKO) | YES (Step 2) |
| **Jones-Schmidt applies** | **NO** | **YES** |
| **Strongly ergodic** | **NO** (Arimoto 2025) | **YES** |

## 6. Step 5: M_Bool^L Is Full

### 6.1 External theorem

**Theorem (Marrakchi 2018, Theorem B; arXiv:1811.08274; published as "Stability of the automorphism group of the Hecke von Neumann algebra," Crelle).** Let Gamma curvearrowright (X, mu) be a nonsingular ergodic essentially free action of a countable group on a standard probability space. The following are equivalent:

(i) L^infty(X) rtimes Gamma is a full factor.

(ii) The orbit equivalence relation R_{Gamma curvearrowright X} is strongly ergodic.

**Source:** A. Marrakchi, arXiv:1811.08274, 2018, Theorem B.

**Remark.** Marrakchi's Theorem B is stated for orbit equivalence relations of essentially free ergodic actions on standard probability spaces. It applies to the group measure space crossed product L^infty(X) rtimes Gamma. More generally, the equivalence holds for the von Neumann algebra L(R) of any countable Borel equivalence relation R on a standard probability space (by Feldman-Moore 1977, every such R is generated by a countable group action). This generality is important because M_Bool^L = L(R_L) is most naturally described as the algebra of an equivalence relation, not always as a group crossed product.

### 6.2 Verification of hypotheses

**(M-i) The action is nonsingular.** G_L acts on ({0,1}^infty, mu_1) by homeomorphisms. The Bernoulli(1/2) measure mu_1 is G_L-quasi-invariant because each g in G_L is a measure-preserving bijection: for any measurable set A, mu_1(g(A)) = mu_1(A) (since the uniform Bernoulli measure is invariant under all permutations of binary strings). In fact, the action is measure-preserving, which is a fortiori nonsingular.

**(M-ii) The action is ergodic.** Established in Step 4, hypothesis (JS-iii).

**(M-iii) The action is essentially free.** Established in Step 3 (Section 4).

**(M-iv) The orbit equivalence relation is strongly ergodic.** Established in Step 4, Section 5.3.

### 6.3 Application

By Marrakchi's Theorem B, the implication (ii) => (i) gives:

    R_L is strongly ergodic  ==>  L^infty(X_L, mu_L) rtimes G_L is a full factor

Since M_Bool^L = L(R_L) = L^infty(X_L) rtimes G_L (by CP-1 + SECTOR-5 for non-Taylor L; see Section 1.2), we conclude:

    M_Bool^L is a full factor.

QED.

## 7. The Complete Proof (Self-Contained Summary)

**Theorem 2.2.** Let L be a non-Taylor Boolean constraint language. Then M_Bool^L is a full factor.

**Proof.** By the Bulatov-Zhuk dichotomy (FOCS 2017 / J. ACM 2020), L is NP-complete.

**Step 1.** The group G_L of L-preserving bi-polynomial circuits contains F_2 (via Toffoli gate universality for NPC reductions + Bennett reversibility). Since F_2 is non-amenable, G_L is non-amenable.

**Step 2.** Three arguments establish Rad(G_L) = {e}:

(a) *Furstenberg boundary.* The ultraproduct construction (Frisch-Schlank-Tamuz 2019) produces a minimal strongly proximal G_L-space on which G_L acts faithfully. By Furman (2003), Rad(G_L) = ker of the boundary action = {e}.

(b) *C*-simplicity.* G_L acts minimally on X_L (NPC transitivity) with non-amenable rigid stabilizers (F_2 in each rigid stabilizer via Toffoli). By Le Boudec-Matte Bon (2018), G_L is C*-simple. By BKKO (2017), Rad(G_L) = {e}.

(c) *Normal subgroup analysis.* Any non-trivial normal subgroup of G_L contains A_{2^n} for large n (Jordan + NPC transitivity), hence contains F_2, hence is non-amenable. So Rad(G_L) = {e}.

**Step 3.** Three arguments establish that G_L acts essentially freely on (X_L, mu_L):

(a) *Bernoulli + NPC rigidity.* For g != e, if Fix(g) contained a cylinder set [w], then g = id by UA2 (essentially unary polymorphisms cannot non-trivially fix a cylinder). Since Fix(g) is closed and cannot contain a cylinder, mu(Fix(g)) = 0.

(b) *UA2 structural.* The only bijective unary operation fixing a Bernoulli-generic coordinate simultaneously at both values is the identity. The stabilizer of a mu-generic point is trivial.

(c) *Topological freeness.* Fix(g) has empty interior (by NPC rigidity). A closed nowhere-dense set in {0,1}^infty has Bernoulli-measure zero.

**Step 4.** The Jones-Schmidt theorem (1987) applies: G_L is non-amenable (Step 1), the action is essentially free (Step 3), the action is ergodic (NPC transitivity on finite levels), and Rad(G_L) = {e} (Step 2). Therefore the orbit equivalence relation R_L is strongly ergodic.

**Step 5.** Marrakchi's Theorem B (2018): strong ergodicity of R_L implies L(R_L) = M_Bool^L is a full factor. QED.

## 10. Structural Remarks

### 10.1 Why Route C bypasses bi-exactness

The original approach (Node 1.3.5.5, Route 1) used Houdayer-Isono (2016): bi-exact group + strongly ergodic action => full crossed product. This required bi-exactness of G_L, which was claimed via V-embedding (G_L inside Thompson's V). The Critic (Node 1.3.5.6) identified a directional error: Q_struct establishes V inside G_Bool, not G_L inside V.

Route C replaces Houdayer-Isono with Marrakchi (2018) Theorem B: strongly ergodic equivalence relation => full factor. Marrakchi's theorem does not require bi-exactness. The strong ergodicity is established via Jones-Schmidt, which requires non-amenability + essential freeness + ergodicity + trivial amenable radical -- all of which are verified without any V-embedding claim.

The crucial insight (Node 1.3.5.6, Section 4.4): strong ergodicity was always the real engine. Bi-exactness was one sufficient condition for strong ergodicity (via Houdayer-Isono's spectral gap rigidity), but Jones-Schmidt provides an independent route that uses only group-theoretic and measure-theoretic properties.

### 10.2 The role of UA2

The UA2 result (non-Taylor => essentially unary clone; Node 1.2) is load-bearing at two points:

(a) **Essential freeness (Step 3).** The NPC rigidity argument uses UA2: a circuit fixing a cylinder set must act by essentially unary operations on remaining coordinates, forcing g = id. Without UA2, a non-trivial multi-ary polymorphism could stabilize a cylinder while acting non-trivially elsewhere.

(b) **Equivalence relation identification (Section 1.4).** The claim R_{G_L} = R_L (no extra groupoid arrows from non-G_L elements) uses UA2: a multi-variable transformation mapping solutions to solutions would yield a non-trivial multi-ary polymorphism, contradicting the non-Taylor hypothesis.

UA2 is thus the mechanism by which the computational complexity of L (NPC) translates into the operator-algebraic property (fullness) of M_Bool^L. The essential unariness is the algebraic poverty that forces essential freeness and forces the orbit equivalence relation to be generated by G_L alone.

### 10.3 The Arimoto lesson

The Arimoto result (2025) is not an obstruction but a calibration: it shows that essential freeness is genuinely necessary for the Jones-Schmidt argument, not merely a convenient hypothesis. Thompson's V satisfies all Jones-Schmidt hypotheses except essential freeness, and its canonical crossed product is indeed not full. For G_L, the essential freeness is a consequence of the non-Taylor structure (UA2), which V's action does not possess. The Arimoto result thus STRENGTHENS confidence in Route C by confirming that the essential freeness hypothesis is doing real work.
