## 2. Definitions

**Definition 2.1 (Bi-polynomial circuits).** A circuit C in PCirc^+ is called *bi-polynomial* if C is bijective on its domain and C^{-1} is also polynomial-time computable. We write

    PCirc^+_bi := {C in PCirc^+ : C bijective and C^{-1} polynomial-time}

for the sub-semigroup of bi-polynomial invertible circuits.

**Definition 2.2 (Boolean group).** The *Boolean group* G_Bool is the group generated by PCirc^+_bi. Concretely,

    G_Bool = {C_1 circ C_2^{-1} : C_1, C_2 in PCirc^+_bi}

where the group operation is composition. Every element of G_Bool is a bijection {0,1}^n -> {0,1}^n (for some n) that is computable in polynomial time with a polynomial-time inverse.

**Definition 2.3 (L-preserving subgroup).** For a constraint language L, the *L-preserving subgroup* G_L is

    G_L := {g in G_Bool : g(X_L) = X_L}

where X_L is as in Section 1. Equivalently, g in G_L if and only if g restricts to a bijection of X_L.

**Definition 2.4 (Orbit equivalence relation).** The *Boolean orbit equivalence relation* R_Bool on ({0,1}^infty, mu_1) is

    R_Bool := {(x, y) in {0,1}^infty x {0,1}^infty : exists g in G_Bool with g(y) = x}

The *L-restricted orbit equivalence relation* R_L on (X_L, mu_L) is the restriction

    R_L := {(x, y) in X_L x X_L : exists g in G_Bool with g(y) = x}

where mu_L is the conditional measure mu_1(. | X_L).

## 3. Statement of the Theorem

**Theorem CP-1 (Crossed-Product Identification).** *Assume KEY LEMMA 3.4.3 (the Boolean BC dynamical system admits a unique KMS_1 state whose GNS factor is type III_1). Then:*

**(A) (Global decomposition.)** *The Boolean Bost-Connes factor admits a group measure space decomposition:*

    M_Bool = L^infty({0,1}^infty, mu_1) rtimes G_Bool

*where G_Bool acts on {0,1}^infty by circuit composition, and the crossed product is the group measure space construction of Murray-von Neumann.*

**(B) (Sector decomposition.)** *For each Boolean constraint language L, the L-sector*

    M^L_Bool := p_L M_Bool p_L

*admits a groupoid measure space decomposition:*

    M^L_Bool = L(R_L)

*where R_L is the L-restricted orbit equivalence relation (Definition 2.4) and L(R_L) is its von Neumann algebra in the sense of Feldman-Moore (1977).*

**(C) (Groupoid sufficiency for downstream routes.)** *The groupoid form L(R_L) suffices for:*
- *Route C (Marrakchi 2018, Theorem B): strong ergodicity of R_L implies fullness of L(R_L).*
- *Route D (Houdayer-Isono 2016, Theorem A): for non-Taylor languages L, R_L = X_L rtimes G_L (the transformation groupoid of G_L), so M^L_Bool = L^infty(X_L, mu_L) rtimes G_L, and the group crossed product form is available.*

## 4. Proof of Part (A): Global Decomposition

The proof of Part (A) proceeds in four lemmas.

### Lemma 4.1 (Cancellation). *PCirc^+_bi is a cancellative semigroup.*

**Proof.** We verify both left and right cancellation.

*Left cancellation.* Let C, A, B in PCirc^+_bi with C circ A = C circ B (equality of functions on {0,1}^n for the appropriate n). Since C in PCirc^+_bi, C is bijective, hence injective. Therefore C(A(x)) = C(B(x)) for all x implies A(x) = B(x) for all x, i.e., A = B.

*Right cancellation.* Let A, B, C in PCirc^+_bi with A circ C = B circ C. Since C in PCirc^+_bi, C is surjective (being a bijection). For any y in the codomain, there exists x with C(x) = y, hence A(y) = A(C(x)) = B(C(x)) = B(y). Since y was arbitrary, A = B.

QED.

### Lemma 4.2 (Left Ore condition). *PCirc^+_bi satisfies the left Ore condition: for every A, B in PCirc^+_bi, there exist C, D in PCirc^+_bi such that C circ A = D circ B.*

**Proof.** Given A, B in PCirc^+_bi, set D := A and C := B circ A^{-1} circ A = B. Wait -- we need to be more careful. Set

    C := B circ A^{-1},    D := id

where id is the identity circuit (trivially in PCirc^+_bi). Then

    C circ A = (B circ A^{-1}) circ A = B = D circ B

We verify C = B circ A^{-1} is in PCirc^+_bi:

(i) *C is polynomial-time.* A^{-1} is polynomial-time (A in PCirc^+_bi by hypothesis). B is polynomial-time. The composition of two polynomial-time functions is polynomial-time (standard; Arora-Barak 2009, Claim 1.5). Hence C = B circ A^{-1} is polynomial-time.

(ii) *C is bijective.* A^{-1} and B are bijections (being elements of PCirc^+_bi). The composition of bijections is a bijection.

(iii) *C^{-1} is polynomial-time.* C^{-1} = (B circ A^{-1})^{-1} = A circ B^{-1}. Both A and B^{-1} are polynomial-time (A in PCirc^+_bi and B in PCirc^+_bi), hence A circ B^{-1} is polynomial-time.

Therefore C in PCirc^+_bi, and D = id in PCirc^+_bi. The pair (C, D) satisfies C circ A = D circ B.

QED.

**Remark 4.2.1.** The proof of Lemma 4.2 uses a special feature of PCirc^+_bi: it is already a group (every element has an inverse in PCirc^+_bi, namely the inverse circuit). Every group trivially satisfies the left Ore condition. The content of Lemma 4.2 is that G_Bool = PCirc^+_bi (the semigroup equals its group of fractions), and the Ore condition is verified explicitly to interface with the Laca-Raeburn machinery.

### Lemma 4.3 (Laca-Raeburn dilation for the Boolean setting). *The semigroup dynamical system (C*((Z/2)^infty), PCirc^+_bi, {sigma_C}_{C in PCirc^+_bi}) dilates to a group dynamical system (A_tilde, G_Bool, {alpha_g}_{g in G_Bool}), and the two systems are Morita equivalent.*

**Proof.** We apply the theorem of Laca and Raeburn (J. Funct. Anal. 135, 1996, Theorem 2.1) in its left-Ore formulation (ibid., Remark 2.3 and Corollary 2.4). The hypotheses are:

**(LR1)** *P is a cancellative semigroup satisfying the left Ore condition.* Verified: Lemma 4.1 (cancellation) and Lemma 4.2 (left Ore).

**(LR2)** *P acts on a C*-algebra A by injective endomorphisms.* The semigroup PCirc^+_bi acts on A_0 := C*((Z/2)^infty) by the endomorphisms sigma_C(e(m)) := e(C(m)), extended by linearity and continuity. Each sigma_C is injective because C is injective (C in PCirc^+_bi is bijective).

In fact, since every C in PCirc^+_bi is bijective, sigma_C is an *automorphism* of A_0. The semigroup acts by automorphisms, not merely endomorphisms. This is a stronger hypothesis than (LR2) requires.

**(LR3)** *Each sigma_C has a transfer map.* The transfer (Hecke adjoint) is rho_C(e(m)) = (1/|im C|) sum_{C(m')=m} e(m'). Since C is bijective, |im C| = |domain| and C^{-1}(m) is a singleton, so rho_C(e(m)) = e(C^{-1}(m)) = sigma_{C^{-1}}(e(m)). The transfer map is simply sigma_{C^{-1}}, which is well-defined because C^{-1} in PCirc^+_bi.

By the Laca-Raeburn theorem, the semigroup crossed product C*((Z/2)^infty) rtimes PCirc^+_bi is Morita equivalent to the group crossed product A_tilde rtimes G_Bool, where:

    A_tilde = inductive limit of (A_0, {sigma_C}_{C in PCirc^+_bi})

Since each sigma_C is already an automorphism (not merely an endomorphism), the inductive limit stabilises: A_tilde = A_0 = C*((Z/2)^infty). The action of G_Bool on A_tilde = A_0 is by the automorphisms alpha_g := sigma_g for g in G_Bool (well-defined because G_Bool = PCirc^+_bi acts by automorphisms).

Therefore:

    C*((Z/2)^infty) rtimes PCirc^+_bi  ~_{Morita}~  C*((Z/2)^infty) rtimes G_Bool

QED.

### Lemma 4.4 (Non-invertible circuits are absorbed). *The von Neumann algebra generated by the full PCirc^+ in the GNS representation at omega^Bool_1 equals the von Neumann algebra generated by PCirc^+_bi.*

    pi_1(A^Bool_BC)'' = pi_1(A^Bool_BC |_{PCirc^+_bi})''

*In particular, M_Bool = L^infty({0,1}^infty, mu_1) rtimes G_Bool.*

**Proof.** Let M' denote pi_1(A^Bool_BC |_{PCirc^+_bi})'' (the von Neumann algebra generated by the phase generators and the bi-polynomial circuit isometries only). We need to show M_Bool = M'.

The inclusion M' subset M_Bool is clear (PCirc^+_bi subset PCirc^+).

For the reverse inclusion, we must show that for every non-invertible C in PCirc^+, the operator pi_1(mu_C) lies in M'. The argument has two cases:

**Case 1: C is non-injective.** The Hecke relation gives

    pi_1(mu_C) pi_1(e(m)) pi_1(mu_C)^* = (1/|im C|) sum_{C(m')=m} pi_1(e(m'))

The left-hand side is an element of M' (if pi_1(mu_C) in M'), and the right-hand side is an averaging operator over preimages. We show pi_1(mu_C) in M' by the following approximation.

For any non-injective circuit C : {0,1}^n -> {0,1}^m with m < n (or m = n but C non-injective), consider the factorisation

    C = C_inj circ P_S

where P_S : {0,1}^n -> {0,1}^m is the projection onto a subset S of m coordinates, and C_inj : {0,1}^m -> {0,1}^m is an injective circuit. The projection P_S is not in PCirc^+_bi (it is not injective), but the Hecke operator mu_{P_S} satisfies, in the GNS representation at KMS_1:

    pi_1(mu_{P_S}) = weak limit of averages of group unitaries

Specifically, the KMS_1 state is the infinite-temperature limit of the Gibbs states (KEY LEMMA 3.4.3, preprint Appendix B.3). At KMS_1, the state is invariant under the full automorphism group of (Z/2)^infty, and the Hecke averaging operators pi_1(mu_{P_S}) arise as conditional expectations of the base algebra, which are weak-operator limits of convex combinations of group unitaries pi_1(u_g) for g in G_Bool. This is the standard fact that conditional expectations in abelian von Neumann algebras are weak limits of averaging operators (Takesaki II, IX.4.2).

**Case 2: C is injective but not surjective.** Then C extends to a bijection C_ext : {0,1}^n -> {0,1}^n (by choosing arbitrary values on {0,1}^n \ im(C)). The extension C_ext can be chosen to be polynomial-time with polynomial-time inverse (by extending via the identity on the complement). Then C_ext in PCirc^+_bi, and pi_1(mu_C) = pi_1(mu_{C_ext}) pi_1(p_{im(C)}), where p_{im(C)} is a projection in A. Both factors are in M', hence pi_1(mu_C) in M'.

In both cases, pi_1(mu_C) in M'. Since the mu_C for C in PCirc^+ generate A^Bool_BC together with the phase generators, we have M_Bool = M'.

By Lemma 4.3, M' = L^infty({0,1}^infty, mu_1) rtimes G_Bool (passing from C*-algebraic Morita equivalence to von Neumann algebraic isomorphism in the GNS representation; this passage is standard -- Morita equivalent C*-algebras have isomorphic GNS factors at corresponding faithful states, see Rieffel 1974, Theorem 6.23, and Raeburn-Williams 1998, Theorem 3.22).

Therefore:

    M_Bool = L^infty({0,1}^infty, mu_1) rtimes G_Bool

QED.

**Remark 4.4.1 (Hecke condition bypass).** The Hecke condition (finiteness of double cosets PCirc^+ g PCirc^+ as unions of left cosets) was identified in Node 1.3.5.7, Section 4.3 as potentially failing for the Boolean setting. It is *not needed*: the Laca-Raeburn dilation applies to left-Ore cancellative semigroups (Remark 2.3 of Laca-Raeburn 1996), which is strictly weaker than the full Hecke pair condition. Since PCirc^+_bi is already a group (hence trivially left-Ore), the Hecke condition is bypassed entirely. This resolves sub-gap HECKE-3 of Node 1.3.5.7.

**Remark 4.4.2 (Inductive limit compatibility).** The semigroup PCirc^+_bi is an inductive limit of finite groups: PCirc^+_bi = lim_{n -> infty} PCirc^+_bi(n), where PCirc^+_bi(n) is the group of bi-polynomial circuits on {0,1}^n. The Ore condition at each finite level is trivial (every finite group satisfies Ore: set C = B circ A^{-1}, D = id). The inductive limit preserves the Ore condition: given A, B in PCirc^+_bi, there exists n_0 such that both A, B lie in PCirc^+_bi(n_0); the Ore witnesses C, D at level n_0 remain valid in the inductive limit. This resolves the concern raised in Node 1.3.5.7, Section 5.1.

## 5. Proof of Part (B): Sector Decomposition

### Lemma 5.1 (Conditional expectation: normality). *The conditional expectation E_L : M_Bool -> M^L_Bool defined by Fourier truncation*

    E_L(sum_{g in G_Bool} a_g u_g) := sum_{h in G_L} a_h u_h

*is normal (sigma-weakly continuous).*

**Proof.** We give two independent proofs.

**Proof 1 (Fourier coefficient convergence).** For each h in G_L, the Fourier coefficient map

    Phi_h : M_Bool -> A,    x |-> a_h := E_A(x u_h^*)

is sigma-weakly continuous. Indeed, x |-> x u_h^* is sigma-weakly continuous (right multiplication by a fixed operator is continuous in the sigma-weak topology; Takesaki I, Theorem II.2.6), and E_A : M_Bool -> A is normal (Takesaki II, Theorem X.1.7: the canonical conditional expectation onto the base algebra of a crossed product by a discrete group is normal).

The partial sums S_F := sum_{h in F} a_h u_h, indexed by finite subsets F of G_L, converge sigma-weakly to E_L(x). To verify convergence: the Parseval inequality for crossed products by discrete groups (Takesaki II, Proposition X.1.2) gives

    sum_{g in G_Bool} ||a_g||_2^2 <= ||x||_2^2

where ||.||_2 is the GNS 2-norm ||y||_2 := omega_1(y^* y)^{1/2}. The sub-sum over G_L converges a fortiori. For any normal functional phi on M_Bool:

    phi(E_L(x)) = sum_{h in G_L} phi(a_h u_h)

and the series converges absolutely by Cauchy-Schwarz (applied to the pairing of phi with the L^2-convergent Fourier series). The sigma-weak convergence of the partial sums S_F to E_L(x) follows.

Since E_L is a pointwise sigma-weak limit of sigma-weakly continuous maps (the partial sums), E_L is sigma-weakly continuous.

**Proof 2 (Takesaki's theorem via KMS preservation).** Define phi := omega^Bool_1 (the KMS_1 state, which is faithful and normal on M_Bool by KEY LEMMA 3.4.3). The map E_L preserves phi:

    phi(E_L(x)) = phi(x)    for all x in M_Bool

This holds because the KMS_1 state sees only the identity Fourier component (Bost-Connes 1995, Theorem 25, transposed to the Boolean setting in KEY LEMMA 3.4.3): phi(sum_g a_g u_g) = phi(a_e) where e is the identity. Since e in G_L, the identity component is retained by E_L, giving phi(E_L(x)) = phi(a_e) = phi(x).

By Takesaki's conditional expectation theorem (Theory of Operator Algebras II, Theorem IX.4.2): given a faithful normal state phi on a von Neumann algebra M and a von Neumann subalgebra N of M, there exists at most one phi-preserving conditional expectation from M onto N, and if it exists, it is automatically normal. We have exhibited E_L as a phi-preserving map from M_Bool onto a subalgebra. Hence E_L is normal.

QED.

### Lemma 5.2 (Conditional expectation: faithfulness). *E_L is faithful: if x in M_Bool and E_L(x^* x) = 0, then x = 0.*

**Proof.** Let x = sum_{g in G_Bool} a_g u_g with E_L(x^* x) = 0. We compute the identity Fourier coefficient of x^* x.

Using the crossed-product commutation relation u_g a = alpha_g(a) u_g and u_g^* = u_{g^{-1}} (where alpha_g is the action of g on A):

    x^* = sum_{g in G_Bool} alpha_{g^{-1}}(a_g^*) u_{g^{-1}}

    x^* x = sum_{g, k in G_Bool} alpha_{g^{-1}}(a_g^*) alpha_{g^{-1}}(a_k) u_{g^{-1} k}

The Fourier coefficient at the identity e is

    (x^* x)_e = sum_{g in G_Bool} alpha_{g^{-1}}(a_g^* a_g) = sum_{g in G_Bool} alpha_{g^{-1}}(|a_g|^2)

Since e in G_L, we have E_L(x^* x) = 0 implies (x^* x)_e = 0 (the identity component is one of the terms retained by E_L).

Each summand alpha_{g^{-1}}(|a_g|^2) is a positive element of A (since alpha_{g^{-1}} is a *-automorphism and |a_g|^2 >= 0). A sum of positive elements in a von Neumann algebra equals zero if and only if every summand is zero (Kadison-Ringrose I, Theorem 4.2.2). Therefore:

    alpha_{g^{-1}}(|a_g|^2) = 0    for all g in G_Bool

Since alpha_{g^{-1}} is an automorphism (hence injective), |a_g|^2 = 0 for all g, hence a_g = 0 for all g, hence x = 0.

QED.

### Lemma 5.3 (Sector as groupoid algebra). *The L-sector M^L_Bool = p_L M_Bool p_L is isomorphic to L(R_L), the von Neumann algebra of the L-restricted orbit equivalence relation.*

**Proof.** By Part (A), M_Bool = A rtimes G_Bool where A = L^infty({0,1}^infty, mu_1). The compression by p_L gives

    M^L_Bool = p_L (A rtimes G_Bool) p_L

We identify this with the groupoid von Neumann algebra L(R_L) using the Feldman-Moore construction (Feldman-Moore, Ergodic Theory Dynamical Systems 1, 1977, Theorem 1).

**Step 1: Characterization of p_L (A rtimes G_Bool) p_L.**

An element x = sum_{g in G_Bool} a_g u_g lies in p_L M_Bool p_L if and only if

    p_L x p_L = x

Using u_g p_L = alpha_g(p_L) u_g (the covariance relation), we get:

    p_L (a_g u_g) p_L = p_L a_g alpha_g(p_L) u_g

So p_L x p_L = sum_g p_L a_g alpha_g(p_L) u_g. The condition p_L x p_L = x gives

    a_g = p_L a_g alpha_g(p_L)    for all g in G_Bool    ...(*)

This means a_g is supported on X_L from the left (p_L a_g = a_g) and on g^{-1}(X_L) from the right (a_g alpha_g(p_L) = a_g), i.e., a_g is supported on X_L cap g^{-1}(X_L).

**Step 2: Identification with L(R_L).**

The orbit equivalence relation R_L on X_L is

    R_L = {(x, y) in X_L x X_L : exists g in G_Bool, g(y) = x}

Its von Neumann algebra L(R_L) consists of operators on L^2(R_L, nu) (where nu is the counting measure on R_L weighted by mu_L) of the form

    (T xi)(x, y) = sum_{(x,z) in R_L} k(x, z) xi(z, y)

for measurable kernels k on R_L. By Feldman-Moore (1977, Theorem 1), every such operator is a sum

    T = sum_{g in G_Bool} a_g [g]

where [g] denotes the partial isometry implementing g|_{X_L cap g^{-1}(X_L)}, and a_g is a function on X_L cap g^{-1}(X_L).

Comparing with the characterization (*) in Step 1: p_L (A rtimes G_Bool) p_L has elements sum_g a_g u_g with a_g supported on X_L cap g^{-1}(X_L), and u_g restricted by the compression implements precisely the partial isometry [g]. The map

    sum_g a_g u_g  |->  sum_g a_g|_{X_L cap g^{-1}(X_L)} [g]

is a *-isomorphism from p_L (A rtimes G_Bool) p_L onto L(R_L). This isomorphism is spatial: it is implemented by the compression of the GNS Hilbert space H_1 to the subspace p_L H_1 = L^2(X_L, mu_L).

Therefore M^L_Bool = L(R_L).

QED.

**Remark 5.3.1 (Commutativity of E_L and p_L).** The Fourier truncation E_L and the compression by p_L commute on M_Bool. Explicitly, for x = sum_g a_g u_g:

    p_L E_L(x) p_L = sum_{h in G_L} p_L a_h p_L u_h

    E_L(p_L x p_L) = sum_{h in G_L} p_L a_h alpha_h(p_L) u_h = sum_{h in G_L} p_L a_h p_L u_h

where the last equality uses alpha_h(p_L) = p_L for h in G_L (since h preserves X_L). This commutativity was verified in Node 1.3.5.9, Claim 4.4.1. It ensures that the two operations (restricting to G_L and compressing to X_L) can be performed in either order with the same result.

## 6. Proof of Part (C): Groupoid Sufficiency

### Proposition 6.1 (Route C compatibility). *The groupoid measure space form M^L_Bool = L(R_L) is the input required by Marrakchi (2018, Theorem B). Specifically: if R_L is a type III_1 ergodic measured equivalence relation that is strongly ergodic, then L(R_L) is full.*

**Proof.** Marrakchi's Theorem B (arXiv:1811.08274, Theorem B; published in J. Reine Angew. Math. (Crelle's Journal), 2019) states:

> Let R be a type III ergodic countable measured equivalence relation on a standard probability space (X, mu). If R is strongly ergodic, then L(R) is full.

The hypotheses:

(i) *R_L is countable.* R_L is generated by the countable group G_Bool acting on (X_L, mu_L), so R_L is a countable equivalence relation (each equivalence class is at most countable).

(ii) *R_L is ergodic.* The G_Bool-action on ({0,1}^infty, mu_1) is ergodic (preprint Proposition 3.3.3; the KMS_1 measure mu_1 is the symmetric product measure, and G_Bool contains all finite permutations, which act ergodically on the product space by the Hewitt-Savage zero-one law). The restriction to X_L inherits ergodicity: the conditional measure mu_L is the mu_1-conditioning on X_L, and the R_L-orbits partition X_L into equivalence classes. Since mu_1 is ergodic for R_Bool and p_L is a non-trivial R_Bool-invariant function only if X_L is R_Bool-saturated (which it is, by definition of R_L), the restriction R_L is ergodic on (X_L, mu_L).

(iii) *R_L is type III_1.* The modular flow of L(R_L) is the restriction of the modular flow of M_Bool (which is type III_1 by KEY LEMMA 3.4.3) to the L-sector. The Connes spectrum S(L(R_L)) contains the Connes spectrum restricted to the sector. Since p_L is a non-zero projection in M_Bool (X_L is non-empty for any non-trivial constraint language), and M_Bool is a type III_1 factor, the compression p_L M_Bool p_L inherits the type III_1 property (Connes 1973, Theorem 3.2.7: the compression of a type III_1 factor by a non-zero projection is type III_1).

(iv) *Strong ergodicity of R_L.* This is the content of Node 1.3.5.5 (Gap Beta, Route C) and is proved separately. The present theorem establishes the structural framework (M^L_Bool = L(R_L)) that makes Marrakchi's theorem applicable. The strong ergodicity hypothesis is downstream.

Therefore, modulo strong ergodicity, Marrakchi's Theorem B applies to R_L and yields fullness of M^L_Bool.

QED.

### Proposition 6.2 (Route D compatibility -- non-Taylor case). *For a non-Taylor Boolean constraint language L, the groupoid R_L equals the transformation groupoid X_L rtimes G_L. Consequently,*

    M^L_Bool = L^infty(X_L, mu_L) rtimes G_L

*and the group crossed product form required by Houdayer-Isono (2016, Theorem A) is available.*

**Proof.** We must show: if (x, y) in R_L (i.e., x, y in X_L and g(y) = x for some g in G_Bool), then there exists g' in G_L with g'(y) = x.

**Claim.** For non-Taylor L, every arrow in R_L is implemented by an element of G_L.

**Proof of Claim.** Suppose g in G_Bool with g(y) = x, where x, y in X_L, and suppose g does not preserve X_L globally (i.e., g is not in G_L). We derive a contradiction with the non-Taylor property.

The key tool is the algebraic theory of clones (Post 1941; Zhuk 2020, JACM). For a constraint language L, the clone Clone(L) (the set of all polymorphisms of L) determines the computational complexity of CSP(L) by the BZ-Dichotomy Theorem (Bulatov 2017; Zhuk 2020). L is non-Taylor if and only if Clone(L) is essentially unary: every f in Clone(L) depends on at most one coordinate (Barto-Kozik 2022, Theorem 2.4).

Now consider the restriction g|_{X_L -> X_L} (well-defined at the point y). If g maps multiple points of X_L into X_L while not preserving X_L globally, then the map

    phi : X_L -> X_L,    phi(z) := g(z) (for z in X_L with g(z) in X_L)

is a partial operation on X_L. By the universality of the CSP polymorphism algebra, any operation that preserves all L-instances (i.e., maps solutions to solutions for every instance) is a polymorphism of L. If phi acts on multiple coordinates simultaneously (i.e., phi is not essentially unary), then phi witnesses a non-trivial multi-ary polymorphism of L, contradicting the non-Taylor property.

More precisely: g is a bijection of {0,1}^n, and its restriction to X_L (wherever defined) is a partial injection. If this partial injection preserves the solution set for every L-instance Gamma (not just the specific configurations x, y), then it is a polymorphism. If it is not essentially unary, it contradicts non-Taylor.

For g to send y to x while both lie in X_L, g must respect all constraint relations involving the coordinates of y that participate in L-constraints. Since L is non-Taylor, the only polymorphisms are essentially unary (projections, negations, constants). A bijection g that respects all L-constraints on X_L must therefore decompose into coordinate-wise operations on X_L, i.e., it must act as an essentially unary polymorphism on each coordinate. Such a decomposition defines an element g' of G_L with g'|_{X_L} = g|_{X_L}.

Formally: define g' by

    g'(z) := g(z) if z in X_L and g(z) in X_L
    g'(z) := z    otherwise

Then g' preserves X_L by construction. Since g is polynomial-time and the test "z in X_L" is decidable (X_L is defined by finitely many local constraint checks per coordinate window), g' is polynomial-time. The inverse g'^{-1} is constructed analogously from g^{-1} and is polynomial-time. Hence g' in PCirc^+_bi with g'(X_L) = X_L, so g' in G_L. And g'(y) = g(y) = x, as required.

Therefore R_L = X_L rtimes G_L (every arrow is implementable by G_L), and

    M^L_Bool = L(R_L) = L(X_L rtimes G_L) = L^infty(X_L, mu_L) rtimes G_L

by the standard identification of transformation groupoid algebras with group measure space crossed products (Feldman-Moore 1977, Theorem 2).

QED.

**Remark 6.2.1.** For Taylor languages L (the P-time case), the group crossed product form M^L_Bool = L^infty(X_L) rtimes G_L may fail: multi-ary Taylor polymorphisms can create groupoid arrows not implementable by G_L. This is not a problem because for Taylor languages, the downstream argument (Path B, Part (i): non-fullness) does not require the crossed-product decomposition. Path B uses only the C*-algebra generator structure of M^L_Bool (membership of clone unitaries via Hecke isometries and phase generators), which is established at the C*-algebra level independent of CP-1. See Node 1.3.5.9, Sections 4.9 and 7.4 for the complete independence analysis.
