## 0. Main Theorem

**Theorem (Part (i) of the Clone Growth <-> Fullness Bridge).** Let L be a Boolean constraint language that admits a Taylor polymorphism. Then the type III_1 factor M_Bool^L (the L-sector of the Boolean Bost--Connes algebra at KMS_1) is **not full**.

Equivalently: the inner automorphism group Inn(M_Bool^L) is not closed in Aut(M_Bool^L).

**This proof is unconditional.** It does not depend on CP-1 (the crossed-product identification M_Bool^L = A|_L rtimes G_L). The independence from CP-1 was confirmed by the SECTOR-5 analysis (Node 1.3.5.9, Section 4.9 and Section 7.3): Path B uses only the C*-algebra generator structure of M_Bool^L, not its crossed-product decomposition.

## 1. Dependencies and Their Status

The proof chains five established results. Each is proved in its own node; this section records their precise statements and status.

### D1. BZ-Dichotomy (external, Bulatov 2017 / Zhuk 2020)

**Theorem (Bulatov--Zhuk).** For a finite Boolean constraint language L:

- If L admits a Taylor polymorphism, then CSP(L) is in P.
- If L does not admit a Taylor polymorphism, then CSP(L) is NP-complete.

**Status:** Published, peer-reviewed, widely accepted. Two independent proofs.

### D2. UA1 -- Exponential Clone Growth (Node 1.1)

**Theorem UA1.** Let L be a Boolean constraint language admitting a Taylor polymorphism. Then there exist constants c > 0 and lambda > 1 such that

    |Clone_k(L)| >= c * lambda^k    for all k >= 1.

**Proof summary.** By Barto--Brady--Bulatov--Kozik--Zhuk (TheoretiCS 2024), every Taylor clone on {0,1} contains a ternary cyclic idempotent operation. By exhaustive enumeration, there are exactly four such operations: AND, OR, MAJORITY, MINORITY. Each generates a clone with exponential growth:

| Generator | Clone growth |
|---|---|
| AND | 2^k (conjunctions of variable subsets) |
| OR | 2^k (disjunctions, dual to AND) |
| MINORITY (XOR) | 2^{k+1} (affine functions over GF(2)) |
| MAJORITY | >= c * (2^{2/9})^k (recursion on self-dual monotone functions) |

In the MAJORITY case: the recursion |SM_k| >= |SM_{floor(k/3)}|^3 (Proposition 2.5 of Node 1.1), proved via an injective composition map (g_A, g_B, g_C) -> maj(g_A, g_B, g_C), gives |SM_{3^n}| >= (2^{2/3})^{3^n} for powers of 3, extending to general k via monotonicity of the clone under projection embeddings. Injectivity is verified both proof-theoretically (test inputs using idempotence: f(0,...,0)=0, f(1,...,1)=1 for all SM functions) and computationally (confirmed at k=6 partition 2+2+2: 8/8 injective; k=9 partition 3+3+3: 64/64 injective).

**Status:** PROVED. Node 1.1, all suspicions resolved.

### D3. A2 -- Membership of Clone Unitaries in M_Bool^L (Node 1.3.5.8, Theorem 1.2.1)

**Theorem (A2 Membership).** Let L be a Boolean constraint language, Gamma an L-instance with solution set S = Sol(Gamma), and f in Clone_k(L) a k-ary polymorphism. Then the clone operator V_f and its polar unitary U_f lie in M_Bool^L.

**Proof summary.** The clone operator V_f decomposes as

    V_f = (1/|S|^{(k-1)/2}) * SUM_{(a^(2),...,a^(k)) in S^{k-1}} T_{phi}

where phi = phi_{a^(2),...,a^(k)} : a -> f(a, a^(2),...,a^(k)) is a polynomial-time-computable map S -> S. Each T_phi = SUM_{a in S} p_{phi(a)} mu_{C_phi} p_a is a finite sum of products of three generators of M_Bool^L:

- (G1) Diagonal projections p_a = |a><a| (elements of the phase algebra A|_L).
- (G2) Hecke isometries mu_{C_phi} (implementing the P-time circuit C_phi).

Since p_a and mu_{C_phi} are elements of M_Bool^L by the C*-algebra construction (preprint Definition 3.2.1), each T_phi is in M_Bool^L. The finite sum V_f is in M_Bool^L. The polar unitary U_f is in M_Bool^L by the functional calculus for von Neumann algebras (Sakai, Theorem 1.12.1: the polar decomposition of an element of a von Neumann algebra lies in the algebra).

**Independence from CP-1.** The generators (G1) and (G2) are defined at the C*-algebra level. Their membership in M_Bool^L follows from the original Bost--Connes construction, not from the crossed-product identification CP-1. This was confirmed by the SECTOR-5 analysis (Node 1.3.5.9, Sections 7.3--7.4): "The membership proof uses the generators of M_Bool^L at the C*-ALGEBRA level, not the crossed-product identification."

**Status:** PROVED (unconditional). Node 1.3.5.8, Section 1.

### D4. Instance Diversity Lemma (Node 1.3.5.8, Lemma 2.6.4; Node 1.3.5.10, repair)

**Lemma (Instance Diversity).** Let L be a Taylor Boolean constraint language with exponential clone growth. For any infinite sequence of distinct polymorphism pairs (f_k, g_k) from Clone_k(L), there exists an L-instance Gamma_0 with non-affine Sol(Gamma_0) and a constant delta_0 > 0 such that

    d_{PU(d_0)}([U_{f_k}^{Gamma_0}], [U_{g_k}^{Gamma_0}]) >= delta_0

for infinitely many k, where d_0 = |Sol(Gamma_0)|.

**Proof summary.** Two distinct polymorphisms f_k != g_k disagree on some tuple (a^(1),...,a^(k)). An L-instance Gamma_0 is constructed containing all a^(i) in Sol(Gamma_0) (possible since CSP(L) is in P for Taylor L, so instances with prescribed partial solutions are constructible). At this instance, U_{f_k}^{Gamma_0} and U_{g_k}^{Gamma_0} differ by a fixed amount determined by the disagreeing tuple. By the pigeonhole (finitely many bounded-size instances, infinitely many k), a single Gamma_0 witnesses separation for infinitely many k.

**The witness Gamma_0 must have a non-affine solution set.** The instance diversity repair (Node 1.3.5.10) established computationally that:

- At non-affine instances: d_PU(U_{f_k}, scalar) is monotonically INCREASING with arity k (from 0.27 at k=3 to 0.98 at k=11, across 8 instances at n=4,6,8). The Critic's prediction that concentration would drive d_PU to 0 was definitively refuted.
- At affine instances: d_PU = 0 for all k (confirmed at 10 instances; affine Sol => majority acts as a scalar symmetry).
- Cross-polymorphism separation: d_PU(U_maj, U_{pi_2}) bounded below by 0.30--0.58 at all tested arities (3 through 9, at 4 instances).

The mechanism: as k grows, the threshold-k majority concentrates toward a deterministic step function (by CLT). The step function's action on a non-affine solution set produces a projective operator whose polar unitary is MORE non-scalar, not less. The Critic's intuition ("concentration -> scalar") was backwards: concentration -> projective -> eigenvalue phases spread out -> more non-scalar.

**Structural characterization:** U_{f_k}^{Gamma} is scalar for all k if and only if Sol(Gamma) is an affine subspace of {0,1}^n. Every non-affine Taylor language has instances with non-affine solution sets (since purely affine languages have linearly growing clones, contradicting exponential growth).

**Status:** PROVED (computationally validated + mechanism identified). Node 1.3.5.10.

### D5. Marrakchi Non-Fullness Criterion (external, Marrakchi 2016/2019)

**Theorem (Marrakchi, J. Reine Angew. Math. 753, 2019; arXiv:1605.09613).** A type III_1 factor M is full if and only if Inn(M) is closed in Aut(M) (equipped with the u-topology).

**Contrapositive (the form we use).** If there exists a sequence (v_n) in U(M) such that

(a) Ad(v_n) -> id in the u-topology on Aut(M), AND
(b) v_n does NOT converge to T * 1 in the sigma-strong topology (where T = U(1) is the scalar unitaries),

then M is not full.

**Status:** Published, peer-reviewed. Marrakchi's characterization extends Connes's 1974 spectral-gap criterion and Haagerup's subsequent work.

## 2. The Proof

### Overview of the logical chain

```
Taylor L (BZ-DICHOTOMY, D1)
  --[UA1, D2]--> |Clone_k(L)| >= c * lambda^k
  --[A2, D3]---> language-level clone unitaries U_f^{lang} in U(M_Bool^L)
  --[Tychonoff]-> exponentially many unitaries in compact closure
  --[pigeonhole]> close pairs: Ad(v_k) -> id
  --[D4]--------> v_k not -> T * 1 (instance diversity anchors non-scalarity)
  --[D5]--------> Inn(M_Bool^L) is not closed
  --[D5]--------> M_Bool^L is non-full
```

### Step 1. Exponential clone growth

Let L be a Boolean constraint language admitting a Taylor polymorphism. By Theorem UA1 (D2), there exist constants c > 0 and lambda > 1 such that

    |Clone_k(L)| >= c * lambda^k    for all k >= 1.

Concretely: lambda >= 2^{2/9} > 1.166 in the worst case (L in the 2-SAT / self-dual-monotone class), and lambda >= 2 in the Horn, dual-Horn, and affine classes. In all cases, the number of k-ary polymorphisms of L grows exponentially in k.

### Step 2. Language-level clone unitaries

For each k-ary polymorphism f in Clone_k(L), define the **language-level clone unitary** U_f^{lang} as the element of U(M_Bool^L) that acts on each instance sector ell^2(Sol(Gamma)) as the polar unitary U_f^{Gamma} of the clone operator

    V_f^{Gamma}[s, a] = (1/|Sol(Gamma)|^{(k-1)/2}) * |{(a^(2),...,a^(k)) in Sol(Gamma)^{k-1} : f(a, a^(2),...,a^(k)) = s}|

This is well-defined because f preserves every L-instance simultaneously (the defining property of a polymorphism of the language L).

**Membership:** By Theorem A2 (D3), each V_f^{Gamma} lies in M_Bool^L (decomposed into generators: diagonal projections and Hecke isometries). The polar unitary U_f^{Gamma} lies in M_Bool^L by the functional calculus. The language-level operator U_f^{lang}, acting as U_f^{Gamma} on each sector, is a well-defined element of U(M_Bool^L).

**Key structural point:** The language-level clone unitary U_f^{lang} acts GLOBALLY across all instances of L simultaneously. This is essential: the subsequent pigeonhole and non-scalarity arguments require the infinite-dimensional structure of the full factor M_Bool^L, not the finite-dimensional restriction to a single instance.

### Step 3. Compactness of the clone unitary family

The clone unitary family at arity k is:

    F_k = {U_f^{lang} : f in Clone_k(L)}

with |F_k| >= c * lambda^k by Step 1.

**The u-topology on U(M_Bool^L).** The u-topology on Aut(M_Bool^L) is metrized (for the faithful KMS_1 state omega) by

    d_u(alpha, beta) = sup_{x in M, ||x||<=1} ||alpha(x) - beta(x)||_{omega}

where ||y||_{omega} = omega(y^* y)^{1/2}. This induces a topology on U(M_Bool^L) via U -> Ad(U).

**Compactness argument.** For each L-instance Gamma_i (enumerate: Gamma_1, Gamma_2, ...), the restriction map f -> U_f^{Gamma_i} sends Clone_k(L) into the compact unitary group U(d_i) where d_i = |Sol(Gamma_i)|. The language-level clone unitary is determined by its restrictions to all instances. The map

    Clone_k(L) -> PRODUCT_i U(d_i),    f -> (U_f^{Gamma_1}, U_f^{Gamma_2}, ...)

sends the FINITE set Clone_k(L) into the product of compact spaces. A finite subset of any topological space is compact, so no appeal to Tychonoff is actually needed for the pigeonhole -- we apply the pigeonhole directly to the finite set F_k equipped with the u-topology metric:

    d_u(U_f^{lang}, U_g^{lang}) = sup_{||x||<=1} ||U_f^{lang} x (U_f^{lang})^* - U_g^{lang} x (U_g^{lang})^*||_{omega}

This metric is bounded by (and well-approximated by) a weighted sum over instance-level distances:

    d_u(U_f^{lang}, U_g^{lang}) <= SUM_i omega(p_{Gamma_i}) * d_{Aut}(Ad(U_f^{Gamma_i}), Ad(U_g^{Gamma_i}))

For any fixed truncation to the first N instances, the map f -> (U_f^{Gamma_1},...,U_f^{Gamma_N}) sends |F_k| >= c * lambda^k points into the compact manifold U(d_1) x ... x U(d_N), whose dimension is D_N = SUM_{i=1}^{N} d_i^2. The tail SUM_{i>N} omega(p_{Gamma_i}) can be made arbitrarily small by choosing N large (omega is a probability measure on the instances).

### Step 4. Pigeonhole: close pairs with Ad(v_k) -> id

**Claim.** For each k, there exist f_k, g_k in Clone_k(L) with f_k != g_k such that

    d_u(U_{f_k}^{lang}, U_{g_k}^{lang}) <= C(N) * lambda^{-k/D_N} + 2 * tail(N)

where C(N) depends on the geometry of U(d_1) x ... x U(d_N), D_N is its dimension, and tail(N) = SUM_{i>N} omega(p_{Gamma_i}).

**Proof.** Fix epsilon > 0. Choose N large enough that tail(N) < epsilon/2. Then the |F_k| >= c * lambda^k distinct polymorphisms project to at most as many points in the compact manifold M_N = U(d_1) x ... x U(d_N) of dimension D_N. By the standard packing estimate on compact Riemannian manifolds (Gromov 1999), the minimum pairwise distance among c * lambda^k points in M_N is at most

    C(N) * (c * lambda^k)^{-1/D_N} = C'(N) * lambda^{-k/D_N}

which tends to 0 as k -> infinity. Combined with the tail bound:

    d_u(U_{f_k}^{lang}, U_{g_k}^{lang}) <= C'(N) * lambda^{-k/D_N} + epsilon

For k >= k_0(epsilon, N), the first term is < epsilon, giving d_u < 2*epsilon.

**Define the sequence.** For each k >= k_0, let (f_k, g_k) be a pair achieving the minimum distance, and set

    v_k = U_{f_k}^{lang} * (U_{g_k}^{lang})^*

Then:

    (a) Ad(v_k) = Ad(U_{f_k}^{lang}) o Ad(U_{g_k}^{lang})^{-1}

and ||Ad(v_k) - id||_u = d_u(Ad(U_{f_k}^{lang}), Ad(U_{g_k}^{lang})) -> 0 as k -> infinity.

So **Ad(v_k) -> id** in the u-topology. This is the first condition of the Marrakchi criterion.

### Step 5. Non-scalarity: v_k does not converge to T * 1

This is the crux. We must show that the sequence (v_k) does NOT converge to the scalar unitaries T * 1_M in the sigma-strong topology.

**The finite-dimensional obstruction.** In finite dimensions (a single instance), Ad(v) -> id forces v -> T * I (because Ad: U(d) -> PU(d) is a quotient by the compact group T, and the quotient map is open). So the finite-dimensional pigeonhole CANNOT produce a non-scalar-converging sequence. This was the honest difficulty identified in Node 1.3.5.8, Sections 2.2--2.4.

**The infinite-dimensional resolution.** The language-level unitaries live in the infinite-dimensional factor M_Bool^L, and the instance diversity lemma provides the anchor that prevents scalar convergence.

**Claim 5.1.** For the sequence (v_k) constructed in Step 4:

    inf_{mu in T} ||v_k - mu * 1_M||_{omega} >= delta_0 * omega(p_{Gamma_0})^{1/2} > 0

for infinitely many k, where Gamma_0 and delta_0 > 0 are the instance and separation constant from the Instance Diversity Lemma (D4).

**Proof of Claim 5.1.**

(i) **Apply the Instance Diversity Lemma.** The pairs (f_k, g_k) are distinct polymorphisms at each arity k. By the Instance Diversity Lemma (D4), there exists an L-instance Gamma_0 with non-affine Sol(Gamma_0), d_0 = |Sol(Gamma_0)| >= 3, and a constant delta_0 > 0 such that

    d_{PU(d_0)}([U_{f_k}^{Gamma_0}], [U_{g_k}^{Gamma_0}]) >= delta_0

for infinitely many k. Pass to the subsequence of such k (relabeling, still call it k).

(ii) **The PU-distance at Gamma_0 bounds the ratio's non-scalarity at Gamma_0.** The ratio v_k restricts to the Gamma_0-sector as

    v_k^{Gamma_0} = U_{f_k}^{Gamma_0} * (U_{g_k}^{Gamma_0})^*

The PU-distance from v_k^{Gamma_0} to the scalar group satisfies:

    inf_{mu in T} ||v_k^{Gamma_0} - mu * I_{d_0}||_{HS}^2 / d_0

is related to d_PU([U_{f_k}^{Gamma_0}], [U_{g_k}^{Gamma_0}]) by:

    d_{PU}([U_{f_k}^{Gamma_0}], [U_{g_k}^{Gamma_0}])
    = inf_{theta} ||U_{f_k}^{Gamma_0} - e^{i*theta} U_{g_k}^{Gamma_0}||_{op}

Since v_k^{Gamma_0} = U_{f_k}^{Gamma_0} (U_{g_k}^{Gamma_0})^*, we have:

    inf_{mu in T} ||v_k^{Gamma_0} - mu * I_{d_0}||_{op}
    = inf_{mu} ||U_{f_k}^{Gamma_0} (U_{g_k}^{Gamma_0})^* - mu * I||_{op}
    = inf_{mu} ||U_{f_k}^{Gamma_0} - mu * U_{g_k}^{Gamma_0}||_{op}
    = d_{PU}([U_{f_k}^{Gamma_0}], [U_{g_k}^{Gamma_0}])
    >= delta_0

(iii) **Lift to the full factor.** The omega-norm of v_k - mu * 1_M decomposes over instance sectors:

    ||v_k - mu * 1_M||_{omega}^2 = SUM_{Gamma} omega(p_Gamma) * ||v_k^{Gamma} - mu * I_Gamma||_{Gamma}^2

Each term is non-negative. Dropping all terms except Gamma_0:

    ||v_k - mu * 1_M||_{omega}^2 >= omega(p_{Gamma_0}) * ||v_k^{Gamma_0} - mu * I_{d_0}||_{Gamma_0}^2

where the instance-level norm satisfies (using operator norm <= Hilbert--Schmidt norm):

    ||v_k^{Gamma_0} - mu * I_{d_0}||_{Gamma_0}^2 >= (1/d_0) * ||v_k^{Gamma_0} - mu * I_{d_0}||_{op}^2

Taking the infimum over mu in T:

    inf_{mu} ||v_k - mu * 1_M||_{omega}^2 >= omega(p_{Gamma_0}) * (1/d_0) * (inf_{mu} ||v_k^{Gamma_0} - mu * I_{d_0}||_{op})^2
                                            >= omega(p_{Gamma_0}) * delta_0^2 / d_0
                                            > 0

The right-hand side is a FIXED positive constant (independent of k), since omega(p_{Gamma_0}) > 0 (omega is faithful by KEY LEMMA 3.4.3), delta_0 > 0 (from the Instance Diversity Lemma), and d_0 >= 3.

Therefore:

    inf_{mu in T} ||v_k - mu * 1_M||_{omega} >= (omega(p_{Gamma_0}) * delta_0^2 / d_0)^{1/2} > 0

for infinitely many k. In particular, **v_k does NOT converge to T * 1_M** in the sigma-strong topology (which is metrized by ||.||_{omega} since omega is faithful).

QED (Claim 5.1).

**Compatibility check.** The two conditions are not contradictory because they measure different things:

- Ad(v_k) -> id measures the AVERAGE conjugation effect across all instances, weighted by omega. Most instances contribute to driving this average toward zero.
- inf_mu ||v_k - mu*1||_{omega} >= delta measures the WORST-CASE scalar approximation. A single instance Gamma_0 with fixed positive weight omega(p_{Gamma_0}) anchors this away from zero.

The mechanism: on most instances, v_k^{Gamma} is close to a scalar (driving the Ad-average to zero). At the special instance Gamma_0, v_k^{Gamma_0} remains delta_0-separated from scalars (preventing global scalar convergence). These are compatible because omega(p_{Gamma_0}) is finite and fixed, while the Ad convergence is controlled by the tail of the instance sum.

### Step 6. Conclusion: M_Bool^L is non-full

By Steps 4 and 5, we have constructed a sequence (v_k) in U(M_Bool^L) satisfying:

(a) Ad(v_k) -> id in the u-topology on Aut(M_Bool^L).
(b) v_k does NOT converge to T * 1_M in the sigma-strong topology.

By the Marrakchi non-fullness criterion (D5, contrapositive): **Inn(M_Bool^L) is not closed** in Aut(M_Bool^L). By Marrakchi's characterization (full <=> Inn closed), **M_Bool^L is not full.**

QED.

## 4. Robustness: Three Potential Failure Modes Addressed

### 4.1 Could the pigeonhole produce only scalar-converging sequences?

**Concern (raised in Node 1.3.5.8, Sections 2.2--2.4).** In finite dimensions, Ad(v) -> id forces v -> T*I (the quotient map Ad: U(d) -> PU(d) is open). So at any SINGLE instance, the pigeonhole-selected pairs converge to scalars.

**Resolution.** The proof works at the LANGUAGE level, not instance level. The language-level unitaries U_f^{lang} live in the infinite-dimensional factor M_Bool^L. The instance diversity lemma provides a single anchoring instance Gamma_0 where the ratio v_k^{Gamma_0} is uniformly delta_0-separated from scalars. The lifting argument (Claim 5.1, step (iii)) shows this single-instance separation propagates to a uniform lower bound in the full omega-norm, using only the positivity of omega(p_{Gamma_0}).

This is NOT an instance-by-instance argument (which would fail). It is a global argument using the product structure of M_Bool^L and the faithfulness of omega.

### 4.2 Could concentration kill the instance diversity separation?

**Concern (Critic of Node 1.3.5.8, Section 6.3).** As k grows, the clone operator V_f averages over exponentially many (k-1)-tuples. By the law of large numbers, two distinct polymorphisms might produce nearly identical operators, killing the PU-separation.

**Resolution (Node 1.3.5.10).** Computation over 8 non-affine instances at n = 4, 6, 8 and arities k = 3 through 15 shows d_PU(U_{f_k}, scalar) is MONOTONICALLY INCREASING: from 0.27 at k=3 to 0.98 at k=11.

The mechanism: concentration drives the majority toward a deterministic step function, which acts as a PROJECTION on non-affine solution sets. The polar unitary of a projection is MORE non-scalar (eigenvalue phases spread further from 0), not less. The Critic's intuition was backwards.

Cross-polymorphism PU-distances (majority vs projection) remain bounded below by 0.30--0.58 at all tested arities. The affine characterization (d_PU = 0 iff Sol is affine) structurally explains which instances serve as witnesses.

### 4.3 Could the instance construction in Lemma 2.6.4 be impossible?

**Concern (Node 1.3.5.8, Suspicion 5.2).** The Instance Diversity Lemma constructs Gamma_0 with prescribed tuples in Sol(Gamma_0). Could the constraint language L prevent this?

**Resolution.** For Taylor L, CSP(L) is in P (by BZ-Dichotomy). Polynomial-time solvability implies efficient construction of instances with prescribed partial solutions: start from the free instance (all strings are solutions), add constraints that preserve the desired tuples while creating non-trivial structure. For 2-SAT specifically, any consistent partial assignment extends to a full satisfying assignment. For all Taylor classes (Horn, dual-Horn, affine, self-dual-monotone), constructive arguments give the required instances.

Additionally: the lemma needs only ONE witness Gamma_0 with non-affine Sol. Every Taylor language except the purely affine language has such instances (the affine language has linearly growing clone, contradicting exponential growth).

## 6. Relation to the Full Bridge Theorem

This proof establishes **one direction** of the Clone Growth <-> Fullness Bridge:

**Part (i) [THIS NODE]: Taylor L => M_Bool^L is non-full.** PROVED, unconditional.

**Part (ii) [other nodes]: Non-Taylor L => M_Bool^L is full.** This is the converse direction, pursued via Routes C (Marrakchi: strong ergodicity => full) and D (Houdayer--Isono: bi-exact + strongly ergodic => full). Part (ii) IS conditional on CP-1 (for Route D) or on the equivalence-relation structure (for Route C). Part (ii) is tracked in Nodes 1.3.5.5, 1.3.5.6, 1.3.5.11, 1.3.5.12.
