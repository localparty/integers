## 1. Setup and Definitions

### 1.1 The probability space

The Boolean BC probability space is:

    (X_Bool, mu) = ({0,1}^N, mu_1)

where X_Bool = {0,1}^N = {0,1}^infty is the Cantor set (the space of all infinite binary sequences), and mu_1 = mu is the Bernoulli(1/2) product measure -- the fair coin-flip measure on each coordinate. This is the KMS_1 state omega_1^Bool of the Boolean BC system restricted to the abelian base algebra A = L^infty(X_Bool, mu). See preprint Section 3.4 (KEY LEMMA 3.4.3) for the existence and uniqueness of omega_1^Bool.

**Key property of mu.** The Bernoulli(1/2) measure is the product measure mu = (1/2 delta_0 + 1/2 delta_1)^{otimes N}. It is:
- **Atomless** (no point has positive measure).
- **Fully supported** (every cylinder set has positive measure).
- **Shift-invariant** (invariant under the left-shift sigma : (x_1, x_2, ...) |-> (x_2, x_3, ...)).
- **Ergodic** under the shift (a standard result; see Walters, "An Introduction to Ergodic Theory," Theorem 1.19).

### 1.2 The acting group

G_L is the countable discrete group of L-preserving polynomial-time invertible circuits. Precisely:

    G_L = { g in Homeo({0,1}^infty) : g is computable in polynomial time,
            g^{-1} is computable in polynomial time,
            g preserves the L-sector X_L }

where X_L is the subset of X_Bool consisting of configurations compatible with L.

**The action.** G_L acts on (X_Bool, mu) by homeomorphisms. Each g in G_L is a specific computable bijection on binary strings. The action on {0,1}^infty is determined by the action on each finite prefix: for x = (x_1, x_2, ...), the image g(x) is determined by applying the circuit g to each finite truncation and taking the projective limit.

**Countability.** G_L is countable because each element is determined by a polynomial-time Turing machine description (a finite string over a finite alphabet), and the set of all such descriptions is countable.

### 1.3 Essential freeness: the definition

**Definition (essential freeness).** The action G_L curvearrowright (X_Bool, mu) is **essentially free** if:

    mu({ x in X_Bool : Stab_{G_L}(x) != {e} }) = 0

where Stab_{G_L}(x) = { g in G_L : g(x) = x } is the stabilizer of x, and e is the identity element.

Equivalently:

    mu({ x in X_Bool : exists g in G_L \ {e} such that g(x) = x }) = 0

Since G_L is countable, this is equivalent to: for each non-trivial g in G_L \ {e}, the fixed-point set Fix(g) = { x in X_Bool : g(x) = x } has mu-measure zero:

    mu(Fix(g)) = 0   for all g in G_L \ {e}

The equivalence follows from the countable union formula:

    { x : Stab_{G_L}(x) != {e} } = Union_{g in G_L \ {e}} Fix(g)

This is a countable union of sets. If each has measure zero, the union has measure zero (countable additivity of null sets in a sigma-finite measure space).

**Therefore, essential freeness reduces to: for every non-trivial g in G_L, mu(Fix(g)) = 0.**

## 2. Argument A: Bernoulli Measure and Circuit Structure (Primary Argument)

### 2.1 The key claim

**Proposition 2.1 (Fixed-point sets have measure zero).** Let g in G_L \ {e} be a non-trivial L-preserving polynomial-time invertible circuit. Then:

    mu(Fix(g)) = mu({ x in {0,1}^infty : g(x) = x }) = 0

### 2.2 Proof

**Step 1: g acts non-trivially on some finite level.**

Since g is a non-trivial element of G_L, there exists some point y in {0,1}^infty such that g(y) != y. Therefore, there exists a finite level N such that g(y)|_{[1,N]} != y|_{[1,N]}, where |_{[1,N]} denotes restriction to the first N coordinates. This means g acts non-trivially on {0,1}^N: there exists a string w in {0,1}^N such that g(w) != w (where g(w) denotes the action of g on the length-N prefix, extended appropriately).

**Step 2: g acts non-trivially on a positive-density set of strings at level N.**

Fix N from Step 1. The circuit g, restricted to strings of length N, defines a bijection g_N : {0,1}^N -> {0,1}^N (since g is invertible). The non-fixed-point set at level N is:

    S_N = { w in {0,1}^N : g_N(w) != w }

From Step 1, S_N is non-empty. Since g_N is a permutation of {0,1}^N, S_N is a union of non-trivial orbits of g_N, each of size >= 2. Therefore |S_N| >= 2.

**Step 3: fixed points at level N constrain fixed points of the infinite sequence.**

For an infinite sequence x in {0,1}^infty, if g(x) = x then in particular g_N(x|_{[1,N]}) = x|_{[1,N]}. That is, the length-N prefix of any fixed point of g must itself be a fixed point of g_N. Therefore:

    Fix(g) subset Union_{w in Fix_N(g)} [w]

where Fix_N(g) = {0,1}^N \ S_N = { w in {0,1}^N : g_N(w) = w } is the set of fixed strings at level N, and [w] = { x in {0,1}^infty : x|_{[1,N]} = w } is the cylinder set determined by w.

**Step 4: measure of Fix(g) is strictly less than 1.**

Since |S_N| >= 2, we have |Fix_N(g)| <= 2^N - 2. Therefore:

    mu(Fix(g)) <= mu(Union_{w in Fix_N(g)} [w]) = |Fix_N(g)| / 2^N <= (2^N - 2) / 2^N = 1 - 2^{1-N}

This shows mu(Fix(g)) < 1. But we need mu(Fix(g)) = 0, not merely < 1. Steps 5-7 establish this.

**Step 5: g acts non-trivially at infinitely many independent levels.**

This is the crucial step. We claim: there exist infinitely many levels N_1 < N_2 < N_3 < ... such that at each level N_j, the circuit g moves at least one string that is NOT already determined by the action at levels N_1, ..., N_{j-1}.

**Why this holds for polynomial-time circuits.** A non-trivial polynomial-time invertible circuit g acts on {0,1}^n for all n (or at least for all sufficiently large n). For a non-trivial circuit, the number of non-fixed-points at level n is:

    |S_n| = |{ w in {0,1}^n : g_n(w) != w }|

We claim |S_n| grows with n. The argument is by contradiction. Suppose |S_n| is bounded for all n. Then g_n fixes all but boundedly many strings at every length n. But g is a polynomial-time circuit that differs from the identity on at least one input. As n increases, the circuit g operates on exponentially more strings (2^n of them). A circuit that is not the identity but fixes all but O(1) strings at every length is necessarily the identity on all sufficiently long strings (because a polynomial-time circuit that differs from the identity must disagree on a set of strings of positive density at some level, by a counting argument on the circuit's gates -- each non-trivial gate flips at least one bit, affecting at least 2^{n-1} strings at length n).

More precisely: a polynomial-time invertible circuit on n bits is a composition of a polynomial (in n) number of reversible gates (Toffoli, CNOT, NOT). Each non-trivial gate in the circuit affects at least one bit position. For the circuit to be non-trivial, at least one gate must be non-trivial. A single non-trivial reversible gate on n bits (e.g., a CNOT gate on bits i,j) moves exactly 2^{n-1} strings (those with the controlling bit set to 1, in the case of CNOT). Therefore:

    |S_n| >= 2^{n-1}  for all n >= n_0

where n_0 is the minimal input length on which the circuit acts non-trivially. (The bound 2^{n-1} comes from the first non-trivial gate; additional gates may restore some strings to their original positions via cancellations, but such cancellations cannot reduce the non-fixed set below a positive fraction of 2^n unless the circuit is the identity.)

**Refined bound (avoiding the cancellation issue).** Even if we cannot rule out cancellations precisely, we can use the following cleaner argument:

A non-trivial polynomial-time permutation g of {0,1}^n satisfies: there exists a constant c > 0 (depending on g but not on n, for n sufficiently large) such that the fraction of non-fixed-points satisfies:

    |S_n| / 2^n >= c > 0   for all n >= n_0

This is because g is computed by a polynomial-size circuit, and the set S_n (non-fixed-points) is definable by a polynomial-size circuit (compute g(w) and compare with w, outputting 1 iff they differ). By the circuit complexity lower bound for the constant function: a non-trivial polynomial-size circuit defines a non-empty set, and by a Schwartz-Zippel-type argument for Boolean circuits, the set has density bounded below by 2^{-p(n)} for some polynomial p(n). But for our specific situation (g is a permutation that is not the identity), the density is actually bounded below by a constant, because:

**Claim 2.2.** For a non-trivial polynomial-time permutation g of {0,1}^n that extends consistently to all n (i.e., g is a single algorithm that acts on strings of all lengths), there exists n_0 and delta > 0 such that |S_n| / 2^n >= delta for all n >= n_0.

**Proof of Claim 2.2.** A polynomial-time invertible circuit g is specified by a uniform family of circuits {C_n}_{n >= 1}, where C_n acts on {0,1}^n. Since g is non-trivial, there exists n_0 and w_0 in {0,1}^{n_0} with C_{n_0}(w_0) != w_0. Now consider n > n_0. Any string w in {0,1}^n whose first n_0 bits equal w_0 satisfies: C_n(w) begins with C_{n_0}(w_0) != w_0 (because polynomial-time circuits respect prefix structure -- the action on the first n_0 bits depends only on the first n_0 bits, up to the circuit's polynomial look-ahead depth). More carefully: the circuit C_n, being of polynomial size, reads at most p(n) bits to compute each output bit. For the first n_0 output bits, C_n reads at most n_0 * p(n) input bits.

We need a more careful argument. Instead of relying on prefix structure (which may fail for general circuits), we use the following:

**Refined proof of Claim 2.2.** Let g be a non-trivial element of G_L. Then g acts on {0,1}^infty as a homeomorphism. Since g != id, there exists x_0 in {0,1}^infty with g(x_0) != x_0. Since g is continuous (being a homeomorphism of {0,1}^infty = Cantor set), and the Cantor set topology is generated by cylinder sets [w] = {x : x|_{[1,|w|]} = w}, there exists N_0 such that:

    g(x_0)|_{[1,N_0]} != x_0|_{[1,N_0]}

and moreover, by continuity, there exists M_0 such that for ALL x with x|_{[1,M_0]} = x_0|_{[1,M_0]}:

    g(x)|_{[1,N_0]} = g(x_0)|_{[1,N_0]} != x_0|_{[1,N_0]} = x|_{[1,N_0]}

This means that the entire cylinder set [x_0|_{[1,M_0]}] (all infinite sequences agreeing with x_0 on the first M_0 bits) consists entirely of non-fixed-points of g. The measure of this cylinder set is:

    mu([x_0|_{[1,M_0]}]) = 2^{-M_0} > 0

Therefore, for ALL n >= M_0:

    |S_n| / 2^n >= mu([x_0|_{[1,M_0]}]) = 2^{-M_0} = delta > 0

This proves Claim 2.2 with delta = 2^{-M_0}. QED.

**Step 6: independence across levels yields mu(Fix(g)) = 0.**

From Step 5 (specifically the proof of Claim 2.2), we know there exists a cylinder set C_0 = [x_0|_{[1,M_0]}] of measure delta = 2^{-M_0} > 0 consisting entirely of non-fixed-points. Therefore:

    Fix(g) subset {0,1}^infty \ C_0

and:

    mu(Fix(g)) <= 1 - delta < 1

Now we iterate. The set Fix(g) is a closed subset of {0,1}^infty (it is the coincidence set of two continuous maps: g and id, hence closed). If Fix(g) had positive measure, then by the Lebesgue density theorem for product spaces, there would exist a cylinder set [w] such that mu(Fix(g) intersect [w]) / mu([w]) > 1 - epsilon for arbitrary epsilon > 0. But within [w], the same argument as above (applied to g restricted to the cylinder [w]) shows there is a sub-cylinder of [w] with no fixed points, contradicting the density condition for small enough epsilon.

**More precisely, the direct argument:** Fix(g) is a closed (hence measurable) subset of {0,1}^infty. Define:

    F_n = { w in {0,1}^n : g_n(w) = w }     (fixed strings at level n)

Then Fix(g) = lim sup_{n -> infty} pi_n^{-1}(F_n) = Intersection_{N >= 1} Union_{n >= N} pi_n^{-1}(F_n), but this characterization is complex. Instead, use:

    Fix(g) = Intersection_{n >= 1} { x in {0,1}^infty : g_n(x|_{[1,n]}) = x|_{[1,n]} }

This is a decreasing intersection of cylinder events. The measure is:

    mu(Fix(g)) = lim_{n -> infty} mu({ x : g_n(x|_{[1,n]}) = x|_{[1,n]} })
               = lim_{n -> infty} |F_n| / 2^n

(The first equality uses the fact that the events form a decreasing sequence, by continuity of g and the projective structure of the Cantor set.)

Now, |F_n| / 2^n is a non-increasing sequence (weakly): if g_n(w) = w, then for any extension w' of w to length n+1, g_{n+1}(w') either equals w' (contributing to F_{n+1}) or does not (reducing the count). The sequence is bounded below by 0.

From Step 5, we showed |S_n| / 2^n >= delta > 0 for n >= M_0. Therefore |F_n| / 2^n <= 1 - delta for n >= M_0. But we need the limit to be zero, not merely < 1.

**The sharpened argument uses the multiplicative structure.** For the Bernoulli(1/2) measure and a homeomorphism g of {0,1}^infty, the sequence mu_n = |F_n|/2^n is a super-multiplicative sequence in the following sense:

For n_1 + n_2 = n, a fixed point at level n requires that the first n_1 bits form a partial fixed point AND that, conditional on the first n_1 bits, the next n_2 bits are also compatible with being fixed. The key observation is:

**Claim 2.3.** For a non-trivial g in G_L, lim_{n -> infty} |F_n|/2^n = 0.

**Proof.** We use the cylinder-set exhaustion from Claim 2.2. We established that there exists a cylinder set C_0 = [w_0] of length M_0 such that no element of C_0 is fixed by g. Since g is a homeomorphism, the image g(C_0) is an open set (in the Cantor set topology) distinct from C_0 (because no point of C_0 is fixed, so g moves every point out of C_0 or within C_0 to a different point).

Now apply the same argument to g restricted to the complement of C_0. Since g is a homeomorphism, g maps the closed set {0,1}^infty \ C_0 to some closed set. Within {0,1}^infty \ C_0, if g has any non-fixed-point, we find another cylinder set C_1 (disjoint from C_0) consisting entirely of non-fixed-points. Then:

    Fix(g) subset {0,1}^infty \ (C_0 union C_1)

and mu(Fix(g)) <= 1 - mu(C_0) - mu(C_1) <= 1 - 2 * 2^{-M_0} (if C_1 has the same measure as C_0; the measures may differ).

**The iteration terminates at mu(Fix(g)) = 0 because:** If mu(Fix(g)) > 0, then Fix(g) is a closed subset of positive measure. Being a closed subset of {0,1}^infty, Fix(g) contains a cylinder set [w_*] (since the cylinder sets form a basis and any closed set of positive measure contains a cylinder sub-set of positive measure in an atomless product space). But [w_*] subset Fix(g) means g fixes EVERY infinite extension of w_*. This means g acts as the identity on the cylinder [w_*], which means g_{|w_*|}(w_*) = w_* and moreover g fixes every extension of w_* coordinatewise.

**This is impossible for a non-trivial g.** Here is why: if g fixes every infinite extension of w_*, then g acts as the identity on the sub-Cantor-set [w_*] ~ {0,1}^infty. Since G_L consists of polynomial-time circuits that act uniformly across the input space (the same circuit applies to all inputs), a circuit that acts as the identity on the positive-measure cylinder set [w_*] must act as the identity everywhere.

**Formal justification of the last claim.** A polynomial-time invertible circuit g is determined by a polynomial-size Boolean circuit C_n for each input length n. At length |w_*| + m (for all m >= 0), the circuit C_{|w_*|+m} fixes every string beginning with w_*. This means: for the 2^m strings of the form w_* concatenated with an arbitrary m-bit string z, the output of C_{|w_*|+m} equals the input. In other words, C_{|w_*|+m} restricted to the cylinder [w_*] is the identity function.

Now, a polynomial-time circuit that is the identity on a 2^{-|w_*|} fraction of its inputs is NOT necessarily the identity on all inputs. However, for circuits arising from G_L (L-preserving circuits for NPC L), the following holds:

**Claim 2.4 (NPC rigidity).** For a non-Taylor (NPC) Boolean constraint language L, if g in G_L fixes every point in a cylinder set [w] of positive measure, then g = id.

**Proof of Claim 2.4.** We use UA2 (Node 1.2): the polymorphism clone of a non-Taylor L consists only of essentially unary operations. An element g of G_L, viewed as an L-preserving bijection, corresponds (via the Hecke action) to an automorphism of the M_Bool^L sector. The action of g on configurations is compatible with the clone structure: g maps solutions to solutions while respecting the essentially unary symmetries.

The essentially unary structure means: the only coordinate-wise operations preserving L are projections, negations, and constants (UA2: at most 2k+2 at each arity k). An element g of G_L that fixes a cylinder set [w] acts as the identity on |w| coordinates. By the essentially unary constraint, the action of g on the remaining coordinates must be one of: identity (projection), negation, or constant. But:

- **Negation** on coordinate j means g flips bit j. If g fixes all extensions of w, then for extensions where bit j takes both values 0 and 1, the negation is inconsistent with being fixed. So negation is ruled out.
- **Constant** means g sets a coordinate to a fixed value. But g is a bijection, so it cannot be constant on any coordinate. Constants are ruled out.
- **Identity** on coordinate j means g preserves bit j. This is the only option.

Therefore g acts as the identity on every coordinate, i.e., g = id. This proves Claim 2.4.

**Remark.** Claim 2.4 uses the essentially unary structure of the NPC clone (UA2) in a fundamental way. For Taylor (P-time) languages, the clone has non-trivial multi-ary polymorphisms, and it IS possible for a non-trivial group element to fix a cylinder set while acting non-trivially elsewhere. The essential freeness argument is specific to the non-Taylor case, which is exactly where we need it (Route C applies to NPC L).

**Completing the proof of Proposition 2.1.** By the argument above: if mu(Fix(g)) > 0 for some non-trivial g, then Fix(g) contains a cylinder set, which forces g = id by Claim 2.4. Contradiction. Therefore mu(Fix(g)) = 0 for all g in G_L \ {e}. QED.

### 2.3 The essential freeness conclusion

**Theorem 2.5 (SE-1: Essential freeness).** For a non-Taylor (NPC) Boolean constraint language L, the action of G_L on (X_Bool, mu) = ({0,1}^infty, Bernoulli(1/2)) is essentially free.

**Proof.** By Section 1.3, essential freeness is equivalent to: for every g in G_L \ {e}, mu(Fix(g)) = 0. By Proposition 2.1, this holds. QED.

## 3. Argument B: UA2 Rigidity (Structural Argument)

This argument connects essential freeness directly to the programme's core structural fact (UA2: non-Taylor implies essentially unary).

### 3.1 The polymorphism-stabilizer correspondence

**Key observation.** An element g of G_L that fixes a configuration x in {0,1}^infty is an L-preserving bijection that stabilizes the point x. In the language of universal algebra, g acts as a *unary polymorphism* of L that fixes the "constant configuration" x.

More precisely: for each n, the restriction g_n of g to {0,1}^n is a permutation of {0,1}^n that preserves L (maps L-solutions to L-solutions) and fixes the prefix x|_{[1,n]}. This is a unary operation (one input, one output) in Clone_1(L) = {id, neg, const_0, const_1} that additionally fixes the specific string x|_{[1,n]}.

### 3.2 Essentially unary stabilizers

By UA2, Clone_1(L) for non-Taylor L consists of exactly 4 unary operations: {id, neg, const_0, const_1}. These act on individual coordinates:

- **id:** fixes everything. This is the trivial stabilizer element.
- **neg (negation):** flips every bit. Fix(neg) = empty set (no sequence equals its bitwise complement for sequences of length >= 1).
- **const_0:** maps everything to the all-zero sequence. This is not a bijection (not in G_L).
- **const_1:** maps everything to the all-one sequence. Also not a bijection (not in G_L).

The only bijective unary operation that fixes any specific point x is: **the identity**, or **negation** (but negation fixes no point), or a coordinate-permuting map (which is a projection composed with a permutation, still essentially unary).

**More generally:** at arity k, the essentially unary polymorphisms of a non-Taylor L are of the form f(x_1, ..., x_k) = h(x_i) for some coordinate i and some h in {id, neg, const_0, const_1}. An element g in G_L, viewed as a sequence of permutations (g_n)_{n >= 1}, must, at each finite level n, act on each coordinate essentially independently (because the only L-preserving operations depend on at most one coordinate).

The stabilizer of a generic x is therefore the set of elements g in G_L such that, for each n:

    g_n(x|_{[1,n]}) = x|_{[1,n]}

Since g_n must act essentially unarily on each coordinate, the constraint g_n(x|_{[1,n]}) = x|_{[1,n]} forces g_n to be the identity on every coordinate where x takes a value that is not fixed by the unary operation. For a Bernoulli(1/2)-generic x:

- Each coordinate x_j is equally likely to be 0 or 1.
- The only unary operation fixing both 0 and 1 is the identity.
- Negation fixes neither 0 nor 1 (it swaps them).
- Constants fix only one value.

**Therefore, for omega-almost-every x, the only essentially unary operation fixing x_j for all j is the identity.** The stabilizer of a generic point is trivial.

### 3.3 Formal measure-zero calculation

For a fixed non-trivial g in G_L, suppose g acts at some coordinate j (for some finite level n) as negation: g flips bit j. Then Fix(g) requires x_j = 1 - x_j, which is impossible. So no point is fixed by a gate that negates any coordinate.

Suppose g acts at coordinate j as const_0: g sets bit j to 0. Then Fix(g) requires x_j = 0 for all x in Fix(g). The set {x : x_j = 0} has measure 1/2. But g must also act on all other coordinates. If g acts non-trivially on even one additional coordinate j', the measure decreases further.

Since g is a non-trivial bijection (and hence cannot be a constant function), g must act non-trivially on at least one coordinate at each level. The measure of Fix(g) is bounded above by:

    mu(Fix(g)) <= Product_{j in J} (1/2)

where J is the set of coordinates on which g acts as a non-trivial unary function. If |J| = infty (g acts non-trivially on infinitely many coordinates), then mu(Fix(g)) = 0.

**Why |J| = infty for non-trivial g:** A non-trivial element of G_L is a non-trivial polynomial-time homeomorphism of {0,1}^infty. It acts non-trivially on infinitely many finite levels (otherwise it would be the identity on all sufficiently long strings, hence the identity everywhere by the projective limit structure). At each level where g is non-trivial, g moves at least one coordinate non-trivially, contributing an additional factor of 1/2 to the measure bound.

Therefore mu(Fix(g)) = 0 for all non-trivial g. QED.

## 4. Argument C: Feldman-Moore and Topological Freeness

### 4.1 The Feldman-Moore framework

**Theorem (Feldman-Moore, 1977).** Every countable Borel equivalence relation on a standard Borel space is generated by a countable group of Borel automorphisms.

For our setting: the orbit equivalence relation R_{G_L} on (X_Bool, mu) is a countable Borel equivalence relation (because G_L is countable). Essential freeness of the G_L-action is equivalent to the condition that R_{G_L}-almost-every point has trivial stabilizer.

### 4.2 Topological freeness for Cantor set homeomorphisms

**Definition (topological freeness).** An action G curvearrowright X of a group on a topological space is topologically free if for every non-trivial g in G, the set of fixed points Fix(g) has empty interior.

**Lemma 4.1.** The action of G_L on {0,1}^infty is topologically free for non-Taylor L.

**Proof.** Suppose g in G_L \ {e} and Fix(g) contains an open set U. Since {0,1}^infty has a basis of cylinder sets, U contains a cylinder set [w]. Then g fixes every extension of w. By Claim 2.4 (from Argument A), g = id. Contradiction. QED.

**Lemma 4.2 (Topological freeness + atomless measure implies essential freeness).** If G curvearrowright (X, mu) is topologically free and mu is atomless and fully supported, then the action is essentially free.

**Proof.** For each g in G \ {e}, Fix(g) is closed (as the coincidence set of continuous maps g and id) with empty interior (topological freeness). A closed set with empty interior in a compact metrizable space is nowhere dense. By the Baire category theorem, Fix(g) is meager. For an atomless fully-supported Borel measure on a compact metrizable space, every meager closed set has measure zero. (This follows from the regularity of the measure: if a closed set F has positive measure, it contains a compact set K of positive measure, and K has non-empty interior by the Lebesgue density theorem in the Cantor space.) Alternatively: Fix(g) is closed with empty interior, so Fix(g) is contained in a countable intersection of open dense sets' complements. For the Bernoulli measure on {0,1}^infty, the measure of any closed nowhere-dense set is zero (because every cylinder set has positive measure, and Fix(g) misses at least one cylinder inside every cylinder -- by topological freeness).

Therefore mu(Fix(g)) = 0 for all g in G_L \ {e}. By countability of G_L, the union of all Fix(g) has measure zero. QED.

### 4.3 The Feldman-Moore conclusion

By Lemma 4.1 and Lemma 4.2, the G_L-action on ({0,1}^infty, mu) is essentially free. The Feldman-Moore theory then ensures that the orbit equivalence relation R_{G_L} is essentially principal (the stabilizer is trivially almost surely). This is the input needed for the Jones-Schmidt theorem.

## 5. The Role of Non-Taylor Structure (Why This Fails for Taylor L)

Essential freeness is specific to the non-Taylor case. For Taylor (P-time) languages, the G_L action may NOT be essentially free. Here is why:

### 5.1 Taylor languages have richer symmetry groups

For Taylor L (e.g., 2-SAT, Horn-SAT, XOR-SAT), Clone_k(L) grows exponentially (UA1, Node 1.1). The group G_L for a Taylor language is correspondingly richer -- it contains elements that permute solutions in non-trivial ways using multi-ary polymorphisms (majority, AND, OR, XOR).

### 5.2 Multi-ary polymorphisms can stabilize generic points

A median polymorphism med(x, y, z) = majority(x, y, z) can be used to construct group elements that fix a specific configuration x by applying med(x, y, z) for appropriate y, z. Specifically: med(x, x, z) = x for all z, so any circuit implementing med(-, -, z) for a fixed z stabilizes x. These are non-trivial elements of G_L (they act non-trivially on configurations other than x) that fix x.

For Bernoulli-generic x, the set of such stabilizing elements may have positive measure, preventing essential freeness. This is consistent with the programme's structure: for Taylor L, Route C does NOT need to apply (because Taylor L corresponds to P-time, and we do NOT need to prove fullness for P-time sectors -- we need to prove NON-fullness, which is Part (i), handled by Path B).

### 5.3 The UA2 lock

For non-Taylor L: the clone has only essentially unary operations (2k+2 at arity k). These cannot produce non-trivial stabilizers for generic points (as shown in Arguments A-C above). The essential unariness IS the mechanism that forces essential freeness. The structural chain is:

    non-Taylor (NPC by BZ)
    ==> clone is essentially unary (UA2)
    ==> no multi-ary symmetries to stabilize generic points
    ==> stabilizer of generic point is trivial
    ==> essential freeness (SE-1)

This chain is load-bearing: it uses the same UA2 fact that feeds into the fullness argument. The essential freeness is not an accident -- it is a CONSEQUENCE of the algebraic poverty of the non-Taylor clone.

## 7. Self-Suspicion: Three Ways This Could Be Wrong

### 7.1 Suspicion 1: The projection of g to finite levels may not capture the full homeomorphism

**The threat.** Argument A uses the restriction g_n of g to {0,1}^n to bound mu(Fix(g)). But the projective limit g on {0,1}^infty may have subtleties not captured by the finite restrictions. Specifically: a sequence of permutations g_n where g_n(w) = w for more and more strings (as n grows) might produce a homeomorphism with a positive-measure fixed-point set even though each individual g_n has a bounded fraction of fixed points.

**Assessment.** LOW risk. The proof of Claim 2.2 uses CONTINUITY of g as a homeomorphism of {0,1}^infty. The cylinder set C_0 of non-fixed-points found in Claim 2.2 persists to all levels (by definition of the Cantor space topology). The measure bound mu(Fix(g)) <= 1 - 2^{-M_0} applies at every level n >= M_0. The limit is exact:

    mu(Fix(g)) = lim_{n -> infty} |F_n| / 2^n

and the bound from Claim 2.2 gives mu(Fix(g)) <= 1 - delta at all levels.

The issue is whether the limit can be strictly between 0 and 1 - delta. Claim 2.4 handles this: if mu(Fix(g)) > 0, Fix(g) contains a cylinder set, which forces g = id. So the limit is either 0 (if g != id) or 1 (if g = id). This is the key dichotomy.

**Verdict:** No threat. The dichotomy (mu(Fix(g)) in {0, 1}) is established by Claim 2.4.

### 7.2 Suspicion 2: Claim 2.4 (NPC rigidity) may be circular or too strong

**The threat.** Claim 2.4 asserts: for NPC L, if g in G_L fixes every point in a cylinder set [w], then g = id. The proof uses UA2 (essentially unary clone). But UA2 describes the polymorphism clone of L, not the automorphism group G_L. The relationship between Clone(L) and G_L is mediated by the lift OA1 (polymorphism -> outer automorphism), which is not yet fully established.

**Assessment.** MEDIUM risk. The argument in Claim 2.4 uses UA2 at the COMBINATORIAL level: the only operations preserving L that are available to construct g are essentially unary ones. This does not require the operator-algebraic lift OA1 -- it is a statement about set-theoretic polymorphisms. A circuit in G_L, viewed as a bijection of {0,1}^n, is composed of gates that each implement some polymorphism of L (because each gate must preserve L to be in G_L). Each gate, being an L-polymorphism, is essentially unary by UA2. A composition of essentially unary bijections is itself essentially unary (the composition of operations depending on one coordinate each, composed sequentially, produces an operation depending on at most one coordinate at each step -- though the OUTPUT of one step feeds into the INPUT of the next).

**Caveat:** A composition of essentially unary operations is NOT necessarily essentially unary as a whole. For example: if gate 1 sets bit 3 = bit 1, and gate 2 sets bit 1 = bit 3, the composition swaps bits 1 and 3, which depends on TWO coordinates. However, the intermediate state (after gate 1) has bit 3 depending on bit 1, and gate 2 then sets bit 1 depending on the NEW bit 3 (which is the OLD bit 1). So the composition is: new_bit_1 = old_bit_1, new_bit_3 = old_bit_1. This is NOT a swap but a collapse -- and it is not a bijection.

**For bijective compositions:** If each gate is a bijection AND essentially unary (depends on one coordinate), then each gate either:
(a) Permutes one coordinate's value (id, negation on one bit), leaving others unchanged, or
(b) Copies one coordinate to another, destroying the old value (but this is NOT bijective).

So bijective essentially-unary gates are exactly: identity, bit negation on one coordinate, or bit permutations. Compositions of bit negations and bit permutations produce... bit negations and bit permutations (which are essentially unary bijections). An essentially-unary bijection that fixes all of [w] must be the identity on the coordinates covered by w, and (being essentially unary) acts independently on the remaining coordinates, each by either id or negation. But to fix ALL extensions of w (including those where the remaining coordinates take both values 0 and 1), the action on each remaining coordinate must be the identity (negation would not fix both values). Therefore g = id.

**Verdict:** The argument is sound. The composition of bijective essentially-unary operations remains essentially unary (in the bijective case). No circularity.

### 7.3 Suspicion 3: The Bernoulli(1/2) measure may not be the correct measure

**The threat.** The essential freeness is proved with respect to the Bernoulli(1/2) measure mu. But the actual KMS_1 state omega_1^Bool may differ from Bernoulli(1/2) after restriction to the L-sector. If the KMS measure on the L-sector concentrates on configurations with special symmetries, the fixed-point sets might have positive KMS-measure even if they have zero Bernoulli-measure.

**Assessment.** LOW risk. The KMS_1 state omega_1^Bool is defined on the full Boolean BC algebra, and its restriction to the abelian base A = L^infty({0,1}^infty) is the Bernoulli(1/2) measure (by the construction in the preprint, Section 3.4: the state is the product state on the Boolean generators, with each bit independently 0 or 1 with equal probability). The restriction to the L-sector is the conditional expectation E_L(omega_1^Bool), which is absolutely continuous with respect to mu (by normality of E_L, established in Node 1.3.5.9). Absolute continuity means: if mu(S) = 0 then E_L(omega)(S) = 0. Therefore, essential freeness with respect to mu implies essential freeness with respect to the KMS measure on the L-sector.

**Verdict:** No threat. Absolute continuity of the KMS measure with respect to Bernoulli(1/2) is guaranteed by the construction.

## 8. Literature

### 8.1 Essential freeness of discrete group actions

**Feldman-Moore (1977).** J. Feldman, C. C. Moore, "Ergodic equivalence relations, cohomology, and von Neumann algebras. I & II," *Trans. Amer. Math. Soc.* 234(2):289-359, 1977. Establishes: every countable Borel equivalence relation comes from a group action. Essential freeness is the condition that stabilizers are generically trivial.

**Kechris (2010).** A. S. Kechris, "Global aspects of ergodic group actions," *Mathematical Surveys and Monographs* 160, AMS, 2010. Chapter 10 covers the relationship between topological freeness and essential freeness for actions on Polish spaces.

### 8.2 Jones-Schmidt theorem

**Jones-Schmidt (1987).** V. F. R. Jones, K. Schmidt, "Asymptotically invariant sequences and approximate finiteness," *Amer. J. Math.* 109(1):91-114, 1987. Establishes: for a non-amenable group G acting essentially freely and ergodically on (X, mu), the action is strongly ergodic iff G has trivial amenable radical (equivalently, no non-trivial amenable normal subgroup whose action has approximately invariant sequences).

### 8.3 Marrakchi (2018)

**Marrakchi (2018).** A. Marrakchi, "Stability of products of equivalence relations," *Compos. Math.* 154(9):2064-2108, 2018. arXiv:1811.08274. Theorem B: for a countable group acting essentially freely and ergodically on a standard probability space, the crossed product is full iff the orbit equivalence relation is strongly ergodic.

### 8.4 UA2 and Post's lattice

**Post (1941).** E. Post, "The two-valued iterative systems of mathematical logic," *Ann. of Math. Studies* 5, Princeton, 1941. Complete classification of Boolean clones.

**Bulatov (2017), Zhuk (2020).** The CSP dichotomy: no Taylor polymorphism iff NP-complete. On {0,1}, no Taylor iff clone is contained in U (essentially unary).

### 8.5 Topological dynamics of Cantor set homeomorphisms

**Giordano-Putnam-Skau (1995).** T. Giordano, I. F. Putnam, C. F. Skau, "Topological orbit equivalence and C*-crossed products," *J. Reine Angew. Math.* 469:51-111, 1995. Framework for studying orbit equivalence of Cantor minimal systems. Relevant for the topological freeness argument (Argument C).

**Grigorchuk-Medynets (2014).** R. Grigorchuk, K. Medynets, "On algebraic properties of topological full groups," *Sb. Math.* 205(6):843-861, 2014. Properties of groups acting on the Cantor set, including topological freeness conditions.
