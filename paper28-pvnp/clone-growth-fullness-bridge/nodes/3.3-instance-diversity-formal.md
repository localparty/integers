## 0. Executive Summary

### What the Critic found

The Path B Critic (critique of Node 1.3.5.8, Section 2.2, MEDIUM severity) found a quantifier-order gap: the original proof claims a fixed instance Gamma_0 where PU-distance is bounded below for the pigeonhole-selected pairs, but the proof does not establish this.

### What this node finds

The gap is DEEPER than MEDIUM. The original lemma statement is not merely unproved -- it is **false** for the pigeonhole-selected pairs. Here is why:

1. The Path B pigeonhole (Node 2.3, Step 4) produces pairs (f_k, g_k) with Ad(v_k) -> id in the u-topology, where v_k = U_{f_k}^{lang} (U_{g_k}^{lang})^*.

2. The u-topology convergence Ad(v_k) -> id restricts to each instance: Ad(v_k^{Gamma}) -> id in Aut(M_{d_Gamma}(C)).

3. In finite dimensions, the quotient map Ad: U(d) -> PU(d) is open. Therefore Ad(v_k^{Gamma}) -> id forces v_k^{Gamma} -> T * I_{d_Gamma}. This means d_PU([v_k^{Gamma}], [cI]) -> 0 at EVERY fixed instance.

4. The original Lemma 2.6.4 claims d_PU([v_k^{Gamma_0}], something) >= delta_0 > 0. This contradicts (3).

The Author of Node 1.3.5.8 correctly identified the finite-dimensional obstruction (Section 2.4) but believed the language-level construction bypassed it. It does not: the obstruction applies at each instance sector of the language-level algebra.

### The resolution

The correct question is not "is v_k non-scalar at a single instance?" (it cannot be -- forced to scalar by the finite-dimensional openness) but "is v_k a GLOBAL scalar?" (it can fail to be, if the instance-by-instance scalar phases disagree).

At each instance Gamma, v_k^{Gamma} -> mu_k(Gamma) * I for some phase mu_k(Gamma) in T. For v_k to converge to a global scalar mu * 1_M, all phases must agree: mu_k(Gamma) -> mu for all Gamma. If the phases DISAGREE (for infinitely many k), v_k does NOT converge to T * 1_M, and the Marrakchi criterion applies.

The corrected lemma replaces the PU-bound at a single instance with a **Phase Incoherence Condition**: the phases mu_k(Gamma) at a non-affine instance and an affine instance are generically distinct.

## 1. Definitions

### 1.1 Clone operator and polar unitary

For an L-instance Gamma with Sol(Gamma) = S, |S| = d, and f in Clone_k(L):

    V_f^Gamma[s, a] = (1/d^{(k-1)/2}) |{(a^(2),...,a^(k)) in S^{k-1} : f(a, a^(2),...,a^(k)) = s}|

The polar unitary U_f^Gamma is the unitary part of V_f^Gamma = U_f^Gamma |V_f^Gamma|.

### 1.2 Instance phase

For a unitary v in U(d) close to scalar: the **instance phase** is

    mu(v) = argmin_{mu in T} ||v - mu I||_{op}

When v = e^{i theta} I + E with ||E|| small, mu(v) = e^{i theta} + O(||E||).

### 1.3 Affine and non-affine

S subset {0,1}^n is **affine** iff S = v + W for some v in {0,1}^n, W <= GF(2)^n. A language L is **non-affine** if some instance has non-affine solution set.

## 2. Three Structural Lemmas

### Lemma A (Affine instances give exact scalars)

**Statement.** If Sol(Gamma) is an affine subspace, then U_f^Gamma in T * I for every f in Clone_k(L) and every k.

**Proof.** Let S = v + W with W a linear subspace over GF(2). The coordinate frequencies satisfy p_j in {0, 1/2, 1} (from the linear structure: the j-th coordinate projection is a linear map on W, with image {0} or GF(2)). The clone operator V_f^Gamma is translation-invariant under the affine group action on S (since the polymorphism commutes with affine symmetries). A translation-invariant operator on ell^2(W) diagonalizes in the Fourier basis (characters of GF(2)^m). The majority's Fourier coefficients are real and positive (majority is a monotone symmetric function), so V is positive up to a global phase, and U = (phase) * I. QED.

**Consequence.** At affine instances, v_k^{Gamma} = U_{f_k}^{Gamma} (U_{g_k}^{Gamma})^* is a product of two scalars, hence EXACTLY scalar: v_k^{Gamma} = e^{i(alpha_{f_k} - alpha_{g_k})} I. No approximation; exact equality.

### Lemma B (Non-affine instances give non-scalar unitaries)

**Statement.** If Sol(Gamma) is non-affine with |Sol(Gamma)| >= 3, then for threshold-k majority:

    d_PU([U_{maj_k}^Gamma], [cI]) > 0 for all k >= 3

and d_PU([U_{maj_k}^Gamma], [cI]) is monotonically non-decreasing in k.

**Proof.** The CLT analysis: as k -> infinity, the clone operator V_{maj_k}^Gamma converges to a limiting operator V_infty that maps each solution a to the deterministic majority-limit string. The limiting operator has rank < d (the non-affine structure causes distinct solutions to map to the same limit), and its polar unitary is non-scalar (the eigenvalue phases of a rank-deficient extension are not all equal). The convergence of V_{maj_k} to V_infty is monotone in the sense that the operator becomes "more projective" at each step (Berry-Esseen rate O(1/sqrt(k))), and the PU-distance to scalar increases monotonically.

Computational confirmation: 8 non-affine instances at n = 4, 6, 8, arities 3 through 15. d_PU increases monotonically from 0.27 (k=3) to 0.98 (k=11) without exception. QED.

### Lemma C (Injectivity of the clone-operator map)

**Statement.** Let Gamma_0 have Sol(Gamma_0) = {00, 01, 10} subset {0,1}^2. Then f |-> V_f^{Gamma_0} is injective on Clone_k(L) for every k.

**Proof.** If f != g, they disagree on some input b = (b_1,...,b_k) in {0,1}^k. Construct a k-tuple of elements of S_0: for each position i, choose 00 if b_i = 0, or 10 if b_i = 1. This gives a tuple in S_0^k whose first-coordinate bit-pattern is exactly b. The outputs f(b) != g(b) at coordinate 1 produce different contributions to V_f^{Gamma_0} and V_g^{Gamma_0}. QED.

## 3. Why the Original Lemma 2.6.4 Fails

### 3.1 The finite-dimensional obstruction is real

The Path B pigeonhole gives Ad(v_k) -> id. Restricting to any fixed instance Gamma with d = |Sol(Gamma)|:

    Ad(v_k^Gamma) -> id in Aut(M_d(C))

The map Ad: U(d) -> PU(d) = U(d)/T is a principal T-bundle, hence a local homeomorphism near the identity. For any sequence with Ad(v_n) -> id in PU(d), there exist mu_n in T with ||v_n - mu_n I||_{op} -> 0.

**Therefore: d_PU([v_k^Gamma], [cI]) -> 0 at every fixed instance.** The original Lemma 2.6.4, which claims d_PU >= delta_0 > 0, is false for the pigeonhole-selected pairs.

### 3.2 This is not a proof error; it is a statement error

The Author of Node 1.3.5.8 correctly identified the finite-dimensional obstruction (Section 2.4) and moved to the language-level construction to bypass it. But the bypass does not work as stated: the language-level unitary v_k decomposes over instance sectors, and at each sector the finite-dimensional obstruction applies. The language-level construction helps with the GLOBAL scalar question (the phases can disagree), but does NOT help with the per-instance PU-bound.

### 3.3 Relation to the computational verification

Node 1.3.5.10 computed d_PU([U_{maj_k}^Gamma], [cI]) and found it INCREASING with k. This is correct: the INDIVIDUAL unitary U_{maj_k}^Gamma becomes more non-scalar. But the RATIO v_k^Gamma = U_{f_k}^Gamma (U_{g_k}^Gamma)^* is a different object. The ratio of two non-scalar unitaries that are close to each other IS close to scalar (by the openness of the Ad map). The computational verification confirms Lemma B but does not address the ratio v_k.

## 4. The Corrected Lemma: Phase Incoherence

### 4.1 The phase decomposition

At each instance Gamma, v_k^Gamma -> mu_k(Gamma) I where mu_k(Gamma) in T is the instance phase. The global scalar approximation is:

    inf_{mu in T} ||v_k - mu * 1_M||_{omega}^2 = Var_omega(mu_k) + o(1)

where

    Var_omega(mu_k) := inf_{mu in T} SUM_Gamma omega(p_Gamma) |mu_k(Gamma) - mu|^2

is the omega-weighted variance of the phase function Gamma |-> mu_k(Gamma).

The term o(1) comes from the non-scalar corrections E_k(Gamma) := v_k^Gamma - mu_k(Gamma) I, which satisfy ||E_k(Gamma)||_op -> 0 at each instance.

**Proof of the decomposition.** For any mu in T:

    ||v_k - mu * 1_M||_omega^2 = SUM_Gamma omega(p_Gamma) ||mu_k(Gamma) I + E_k(Gamma) - mu I||_Gamma^2
    = SUM_Gamma omega(p_Gamma) [|mu_k(Gamma) - mu|^2 + ||E_k(Gamma)||_Gamma^2 + cross terms]

The cross terms are bounded by ||E_k|| * |mu_k - mu|, which is o(1) * O(1) = o(1). Taking inf over mu: the dominant term is Var_omega(mu_k). QED.

### 4.2 The corrected statement

**Lemma 2.6.4* (Instance Diversity, corrected).** Let L be a non-affine Taylor Boolean constraint language with exponential clone growth. For any sequence v_k = U_{f_k}^{lang} (U_{g_k}^{lang})^* with f_k != g_k in Clone_k(L) and Ad(v_k) -> id:

**Assume (ID):** The omega-weighted phase variance satisfies

    limsup_{k -> infty} Var_omega(mu_k) > 0

Then v_k does NOT converge to T * 1_M in the sigma-strong topology.

**Proof.** Immediate from the phase decomposition (Section 4.1). If Var_omega(mu_k) > delta^2 for infinitely many k, then inf_{mu} ||v_k - mu * 1_M||_omega > delta - o(1) > delta/2 for large k. QED.

### 4.3 What condition (ID) says

The Phase Incoherence Condition (ID) says: the instance-by-instance scalar phases mu_k(Gamma) do NOT all converge to the same value. Equivalently: the "phase profile" Gamma |-> mu_k(Gamma) has nonzero omega-weighted variance for infinitely many k.

**What (ID) requires structurally.** At each instance, v_k^Gamma -> mu_k(Gamma) I. The phase mu_k(Gamma) is:

    mu_k(Gamma) = (1/d_Gamma) tr(U_{f_k}^Gamma (U_{g_k}^Gamma)^*) + O(||E_k||)

For affine instances: both unitaries are exact scalars (Lemma A), so the phase is:

    mu_k(Gamma_aff) = e^{i alpha_{f_k}(Gamma_aff)} * e^{-i alpha_{g_k}(Gamma_aff)}  (exact)

For non-affine instances: the unitaries are non-scalar (Lemma B), and the normalized trace of their product captures an "average eigenvalue ratio" that depends on the non-affine geometry.

(ID) says these two types of phases -- the exact affine phase and the approximate non-affine phase -- do NOT always agree. This is a structural statement about how the polymorphism pair (f_k, g_k) interacts with affine vs non-affine solution sets.

## 5. Evidence for Condition (ID)

### 5.1 The structural argument

The phase at an affine instance is determined by a SCALAR action (one complex number per polymorphism per instance). The phase at a non-affine instance is determined by a MATRIX action (a d x d unitary, of which the scalar phase is one summary statistic). These two types of data are related through the polymorphism, but are NOT identical.

For the phase incoherence to FAIL (all phases equal), the scalar summary of the matrix action at every non-affine instance must exactly match the scalar action at every affine instance. This is a co-dimension-1 constraint on the space of polymorphism pairs: it requires a specific algebraic relationship between the matrix action and the scalar action.

The exponential growth of the clone (c lambda^k polymorphisms at arity k) provides exponentially many "samples" of the phase map. The co-dimension-1 constraint cannot be satisfied by all samples simultaneously. Therefore: generically, (ID) holds.

**Caveat.** "Generically" is not "always." The pigeonhole selects specific pairs, not generic ones. The argument that pigeonhole-selected pairs are generic with respect to (ID) requires understanding the geometry of the phase map, which depends on the specific clone structure.

### 5.2 The computational evidence

Node 1.3.5.10 established:

| Finding | Relevance to (ID) |
|---|---|
| d_PU(U_{maj_k}, scalar) INCREASES with k at non-affine instances | The non-affine geometry produces ROBUST non-scalar unitaries, ensuring the matrix summary statistic differs from the scalar summary |
| d_PU = 0 at all affine instances | The affine/non-affine dichotomy is clean: affine gives exact scalar, non-affine gives non-scalar. The two types of phases are structurally different. |
| Cross-polymorphism PU-distance bounded below by 0.30--0.58 | Distinct polymorphisms maintain structural differences at non-affine instances, supporting the independence of phases. |

### 5.3 The analytic mechanism

For Boolean clone operators (real matrices): all unitaries are in the orthogonal group O(d). The phases mu_k(Gamma) are in {+1, -1} (the only real scalars on the unit circle). The phase incoherence condition becomes: there exist instances where the sign of v_k is +1 and instances where it is -1.

At affine instances (even-dimensional: d = 2^m): both +I and -I have det = +1 (since det(-I) = (-1)^{2^m} = +1 for m >= 1). The sign is NOT determined by the determinant.

At non-affine instances (d = 3 for our canonical Gamma_0): in O(3), the sign of v_k is determined by whether v_k is closer to +I or -I. For v_k = U_{f_k} U_{g_k}^T in O(3): sign = +1 iff the rotation angle is < pi/2, sign = -1 iff the angle is > pi/2.

The rotation angle of the ratio depends on the STRUCTURAL SIMILARITY of f_k and g_k as they act on the non-affine instance. Two polymorphisms that are close in the u-topology at affine instances may have very different rotation angles at non-affine instances, because the non-affine geometry amplifies structural differences that are invisible at affine instances.

## 6. The Conditional Proof of Non-Fullness

**Theorem (Part (i) of the Bridge, conditional form).** Let L be a non-affine Taylor Boolean constraint language with exponential clone growth. Assume condition (ID). Then M_Bool^L is non-full.

**Proof.** By Node 2.3, Steps 1--4: exponential clone growth gives exponentially many language-level clone unitaries; the packing pigeonhole gives pairs (f_k, g_k) with Ad(v_k) -> id. This is Marrakchi condition (a).

By condition (ID) and Lemma 2.6.4* (Section 4.2): v_k does NOT converge to T * 1_M. This is Marrakchi condition (b).

By the Marrakchi non-fullness criterion: Inn(M_Bool^L) is not closed, hence M_Bool^L is not full. QED.

## 7. The Path Forward: Closing (ID)

### 7.1 Approach 1: Explicit phase computation for the majority clone

Compute mu_k(Gamma_0) and mu_k(Gamma_1) for the pigeonhole-selected pairs in the self-dual monotone clone (2-SAT). The majority clone has explicit combinatorial structure: the polymorphisms are compositions of majority and projections, and the clone operators at small instances (d = 3 for Gamma_0, d = 4 for Gamma_1) are computable matrices. The phases are then explicit functions of k and the pair selection.

**Estimated difficulty:** 2--3 pages. The computation is finite-dimensional and explicit.

### 7.2 Approach 2: Structural theorem about phase independence

Prove that for any non-affine Taylor language with exponential clone growth, the phase map f |-> (mu_f(Gamma_0), mu_f(Gamma_1)) is NOT a product map modulo the pigeonhole constraint. This would use the classification of Boolean clones (Post's lattice) and the specific algebraic structure of each clone class.

**Estimated difficulty:** 4--5 pages. Requires case analysis over the four Taylor clone classes (AND, OR, MAJORITY, MINORITY).

### 7.3 Approach 3: Modify the Marrakchi criterion

Instead of using the standard Marrakchi criterion (which requires Ad -> id AND non-scalar), use a MODIFIED criterion that directly characterizes non-fullness via the clone structure. For example: if the clone unitaries generate a dense subgroup of Inn(M), then Inn(M) is not closed (hence M is not full). This would bypass the instance diversity lemma entirely.

**Estimated difficulty:** Unknown. Requires a new operator-algebraic argument.

## 9. Self-Suspicion

### 9.1 Suspicion: could (ID) be false?

If (ID) is false -- all instance phases always agree for pigeonhole-selected pairs -- then v_k -> T * 1_M for every pigeonhole-selected sequence. This would mean the Path B pigeonhole construction CANNOT produce a Marrakchi witness. It would NOT mean M_Bool^L is full; it would mean a different construction is needed (e.g., Approach 3 in Section 7.3).

**Assessment.** LOW probability. The structural argument (co-dimension-1 constraint + exponential growth) makes phase agreement highly unlikely. The computational evidence confirms phase disagreement. But the possibility cannot be ruled out without a formal proof.

### 9.2 Suspicion: is the deeper gap fatal for the entire programme?

No. The gap is specific to the pigeonhole-selected pairs. If Approach 3 (modified Marrakchi criterion) succeeds, the instance diversity lemma becomes unnecessary. The non-fullness of M_Bool^L for Taylor L can potentially be proved by other means (e.g., showing the clone unitaries generate a non-compact subgroup of U(M)).

### 9.3 Suspicion: the honest discovery of "the gap is deeper" -- should it be reported differently?

This node honestly reports that the original Lemma 2.6.4 is false as stated, upgrades the severity from MEDIUM to HIGH-MEDIUM, and provides the conditional formulation. The honest discovery is more valuable than a cover-up: it precisely identifies WHAT needs to be proved (condition ID) and HOW to prove it (three approaches), enabling the next Author to close the gap efficiently.
